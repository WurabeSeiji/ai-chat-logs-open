# 2つの保存量と1つの単調減少量を持つMax-Plus方程式

**時枝雄司¹, 高橋大輔¹**

¹ 早稲田大学 基幹理工学部 数学科, 〒169-8555 東京都新宿区大久保3-4-1

JSIAM Letters Vol.10 (2018) pp.45–48

---

## 概要

1+1次元4近傍セルオートマトンの拡張として、max-plus方程式を提案する。この方程式は、二値セルオートマトンのmax-plusアナログである2つの保存量と1つの単調減少量を持つ。興味深いのは、量の一つがグローバルであり、従属変数の最大値によって定義されることである。max-plus公式を用いてそれらの保存性と単調性の証明を示す。

**キーワード:** max-plus方程式、粒子系、セルオートマトン、保存量

**研究活動グループ:** 応用可積分系

---

## 1. はじめに

離散方程式は、多様な自然現象・社会現象の数学モデルとして使用されてきた。離散性には様々なレベルがある。例えば、差分方程式、結合写像格子、セルオートマトン（CA）などである。その中で、max-plus方程式は極めて新しい表現であり、近年精力的に研究されている [1–6]。

Max-plus方程式は、実数上の「max」「min」「+」「−」演算から構成される区分線形型の差分方程式である。区分線形性により従属変数を整数の範囲に閉じることができるため、max-plus方程式はしばしば「超離散方程式」と呼ばれる。さらに、通常の算術演算で定義される差分方程式から、「超離散化」と呼ばれる極限操作により導出される [1]。例えば、離散Burgers方程式に対して小パラメータを用いた指数関数による従属変数の変換を施し、パラメータの極限をとると、「超離散Burgers方程式」と呼ばれるmax-plus方程式が得られる [2]。その従属変数の値域は二値 $\{0, 1\}$ に限定でき、したがって超離散Burgers方程式は規則番号184の初等CAに帰着する [7]。

この例のように、CAは特別な従属変数の値域での超離散Burgers方程式に自然に埋め込まれ、超離散Burgers方程式は（微分）Burgers方程式を近似する離散Burgers方程式の超離散化により導出される。したがって、完全連続から完全離散、さらにはデジタルまで、様々な離散レベルの方程式を互いに関連付けることができる。この文脈で、max-plus方程式はCAのようなデジタル系に対する新しい視点を提供する。

本稿では、主保存量を持つある1+1次元4近傍二値CAに着目する。$j$ を整数空間サイト番号、$n$ を整数時刻、$u_j^n$ をサイト $j$、時刻 $n$ における状態値とする。二値従属変数 $u$ の値は0または1である。発展規則の一般形は

$$u_j^{n+1} = f(u_{j-2}^n, u_{j-1}^n, u_j^n, u_{j+1}^n)$$

で書かれる。$\#(x_1 x_2 \ldots x_k)^n$（$x_i \in \{0, 1\}$）を、時刻 $n$ の空間サイトにおける局所パターン $x_1 x_2 \ldots x_k$ の総数と定義する。$\#(x_1 x_2 \ldots x_k)^n$ が保存される（$n$ に対して一定）場合、上付き $n$ を省略して $\#(x_1 x_2 \ldots x_k)$ と書く。$\#1$ が保存されるような特別な規則が存在する。文献 [8] に倣い、そのようなCAを「粒子CA」（PCA）と呼ぶ。同文献では、4近傍のPCAは「PCA4」と呼ばれ、PCA4-1から4-4まで4つの独立な規則が存在する。さらに、PCA4-1, 4-2, 4-3はそれぞれ初期値問題が解けるmax-plus方程式に埋め込まれることが報告されている。しかし、一般解が導出可能なPCA4-4に対する適切なmax-plus方程式はまだ見つかっていなかった。

本稿では、PCA4-4を二値の場合として含むmax-plus方程式を提案する。PCA4-4は主保存量 $\#1$ 以外に、高次の保存量と単調減少量を持つ。提案するmax-plus方程式はこれらすべての量に対応するものを持つ。

---

## 2. PCA4-4の定義と性質

PCA4-4は以下の保存形式で書ける [9]：

$$u_j^{n+1} = u_j^n + q(u_{j-2}^n, u_{j-1}^n, u_j^n) - q(u_{j-1}^n, u_j^n, u_{j+1}^n), \tag{1}$$

ここで $q(a, b, c) = 1$（$(a, b, c) = (0, 1, 0)$ のとき）、それ以外は0。以下、初期時刻 $n = 0$ の初期値問題を考え、有限個の空間サイトに対する周期境界条件を仮定する。

この境界条件の下で、PCA4-4は初期データに依存して2種類の定常解を持つ。一つは右移動パターン $u_j^{n+1} = u_{j-1}^n$、もう一つは静止パターン $u_j^{n+1} = u_j^n$ である。初期データに少なくとも1つの局所パターン「11」が含まれていれば、十分大きな $n$ で解は静止する。初期データに連続する1がなければ、定常解は右移動になる。

さらに、PCA4-4は以下の2つの保存量と1つの単調減少量を持つ。

**PCA4-4の性質**

- (I) $\#1$ は保存される。
- (II) $\#010^n$ は単調減少する。
- (III) $\#110$（$\#011$）は保存される。

(I)は(1)が保存形式であることから容易に示せる。

---

## 3. PCA4-4から拡張されたMax-Plus方程式

### Max4-4の提案

以下のmax-plus方程式をMax4-4として提案する：

$$u_j^{n+1} = u_j^n + q(u_{j-2}^n, u_{j-1}^n, u_j^n) - q(u_{j-1}^n, u_j^n, u_{j+1}^n),$$
$$q(a, b, c) = \max(0, \min(b - a, b - c)). \tag{2}$$

従属変数 $u$ を0または1に制限すれば、$q(a, b, c) = 1$（$(a, b, c) = (0, 1, 0)$ のときのみ）となり、PCA4-4を特別な場合として含む。

### 解の値域の閉包

$L$ と $M$ をそれぞれ $\max_j\{u_j^0\}$ と $\min_j\{u_j^0\}$ とすれば、任意の $j$ と $n$ に対して $M \leq u_j^n \leq L$ が成り立つ。証明は (2) の式変形による。したがって、任意の有限値域の初期データに対して解は発散しない。例えば、$M$ から $L$ の整数値を初期データに用いれば、解の値域も同じであり、(2)は多値CAとなる。

### 性質(I)の対応物

PCA4-4の保存量 $\#1$ は $S^n = \sum_j u_j^n$ で数えられ、Max4-4でもこの量をそのまま対応物と考える。(2)が保存形式に従うことから、周期境界条件下で $S^n$ が保存されることは明らかである。

### 性質(II)の対応物

性質(II)について、$\#010^n$ はPCA4-4のフラックスの和に等しい。したがって、Max4-4の対応物は

$$Q^n = \sum_j q(u_{j-1}^n, u_j^n, u_{j+1}^n)$$

で定義される。$Q^n - Q^{n+1}$ を展開すると、

$$Q^n - Q^{n+1} = \sum_j (\max A_j^n \cup B_j^n - \max A_j^n) \tag{3}$$

の形に書け、$\max A_j^n \cup B_j^n - \max A_j^n \geq 0$ であるから $Q^n - Q^{n+1} \geq 0$ が常に成り立ち、$Q^n$ は単調減少する。

### 性質(III)の対応物

$L$ を初期データの最大値とする。局所パターン $LL$（$L$ が2つ隣接する箇所）について、$LL$ からの値は時刻 $n+1$ でも $LL$ を保つことが示される。また、$LL$ でない箇所から $LL$ が新たに生成されることはない。したがって、局所パターン $LL$ は定常であり、時刻 $n$ と $n+1$ で一対一対応する。$LL\ell$（$\ell < L$）の数 $\#LL\ell$ は、3連続サイト値のパターンで定義されるMax4-4の高次保存量である。

二値の場合、パターン $LL\ell$ は $110$ となる。Max4-4の $\#LL\ell$ はPCA4-4の $\#110$ の対応物と考えられる。

---

## 4. 結論

PCA4-4の拡張として、max-plus方程式 (2) で定義されるMax4-4を提案した。Max4-4は状態値を二値 $\{0, 1\}$ に制限すればPCA4-4と等価である。PCA4-4は性質(I)と(III)による2つの保存量、性質(II)による1つの単調減少量を持つ。Max4-4もこれらの性質の対応物を持つことを示した。

| 性質 | (I) | (II) | (III) |
|:--|:--|:--|:--|
| PCA4-4 | $\#1$ | $\#010^n$ | $\#110$ |
| Max4-4 | $\sum_j u_j^n$ | $\sum_j q(u_{j-1}^n, u_j^n, u_{j+1}^n)$ | $\#LL\ell$ |

PCA4-4の二値規則を特別な場合として含むだけであれば、多くの異なるmax-plus方程式が見つかる。しかし、PCA4-4の性質のすべてのmax-plus対応物をも持つ方程式はそれらの中で極めて少なく、(2)で定義されるものを見出すことに成功した。性質(III)について、Max4-4の $\#LL\ell$ は全サイト値から与えられるグローバル値 $\max_j\{u_j^0\}$ により定義される。一方、PCA4-4の保存量 $\#110$ は3連続サイトのパターンの数により得られると同時に $\#110$ の「1」は全サイト値の最大値でもあるため、局所的かつグローバルである。PCA4-4から拡張されたMax4-4では、$\#110$ のグローバルな性質が選択されて $\#LL\ell$ となったと考えられる。このような関係は極めて新しく、二値CAとmax-plus方程式の間の興味深い関連性を示唆している。

# ゲージボソンの質量構造——符号ベクトルが示すヒッグス非関与性

## —— [3] の表Aと [4] §8 に基づくW±/Z⁰質量問題の構造的回答 ——

**著者**: 木原 範昭 (WF System Co., Ltd.)

**日付**: 2026年4月

## §0 本稿の目的

標準模型では、W±（80.4 GeV）および Z⁰（91.2 GeV）の質量獲得はヒッグス機構（電弱対称性の破れ）により説明される。本枠組みにおいても、[5] §1 で「W±/Z⁰ の質量獲得機構（空間軸を介さない間接的結合の詳細）」が未解決課題として残されている。

本稿は、[3] の表Aに記載された各粒子の6次元符号ベクトルと、[4] §8 で導出された加速度の幾何学的起源を対比することにより、以下を示す。

1. W±/Z⁰ はヒッグスボソンと空間軸を共有しておらず、ヒッグスとの直接結合を持たない（§1）
2. W±/Z⁰ の符号ベクトルは光子と同一の構造（空間成分ゼロ）を持ち、「質量」の起源はヒッグスではなく主観空間の曲率に帰せられる（§2）
3. 質量獲得の機構は2種類に区別される: フェルミオンの静止質量（ヒッグス由来）とゲージボソンのエネルギー（曲率由来）（§3）

本稿における「ヒッグス非関与」とは、[5] §8 で定式化された空間軸共有による直接結合が不在であることを指す。フェルミオンの質量階層および定量的質量比に関する予備的考察は、「質量構造の考察——軸スケール値と符号付き面積による質量分析の枠組み」で別途扱っている。本補講はゲージボソンの質量構造という別の問題領域を扱う。

## §1 符号ベクトルの比較

### 1.1 ヒッグスボソンとゲージボソン

[3] の表Aに基づき、ヒッグスボソンおよびスピン1ゲージボソンの6次元符号ベクトル $(x, y, z, t, R, Q)$ を列挙する。

| 粒子 | スピン | 符号ベクトル $(x, y, z, t, R, Q)$ | 非零軸 | 実測質量 |
|:---|:---:|:---|:---|---:|
| ヒッグス $H^0$ | 0 | $(0, 0, +, 0, 0, 0)$ | z | 125.3 GeV |
| 光子 $\gamma$ | 1 | $(0, 0, 0, 0, +, 0)$ | R | 0 |
| $Z^0$ | 1 | $(0, 0, 0, 0, -, 0)$ | R | 91.2 GeV |
| $W^+$ | 1 | $(0, 0, 0, +, 0, 0)$ | t | 80.4 GeV |
| $W^-$ | 1 | $(0, 0, 0, -, 0, 0)$ | t | 80.4 GeV |

### 1.2 空間軸の非共有

**観察 1.1**. ヒッグスボソン $H^0 = (0, 0, +, 0, 0, 0)$ の非零軸は空間 $z$ 軸のみである。一方、光子・$Z^0$・$W^{\pm}$ はいずれも**空間成分 $(x, y, z)$ がすべてゼロ**である。

したがって、光子・$Z^0$・$W^{\pm}$ はヒッグスと空間軸を共有しない。[5] §8 で定式化された質量獲得機構（定常波との共有空間軸上での位相結合）は、これらのゲージボソンには適用されない。

**命題 1.1**. $W^{\pm}$ および $Z^0$ の質量は、ヒッグスボソンとの空間軸共有による静止質量獲得（[5] §8）では説明できない。別の機構による説明が必要であり、§2 で中心投影の曲率に由来する加速度エネルギーとして記述する。

*論証*. [5] §8.1 により、定常波（ヒッグス、$\kappa = 1$）との結合は、方向制約波（$\kappa = 2$）と定常波が空間軸を直接共有する場合に成立する。$W^{\pm}$（$t$ 軸のみ非零）および $Z^0$（$R$ 軸のみ非零）は空間成分がすべてゼロであり、ヒッグスの非零軸（$z$）との共有空間軸を持たない。$\square$

### 1.3 光子との構造的同一性

**観察 1.2**. $W^{\pm}$、$Z^0$、光子 $\gamma$ はいずれも空間成分がゼロである。ヒッグスとの結合に関する構造は3者で同一である。

| 性質 | $\gamma$ | $Z^0$ | $W^{\pm}$ |
|:---|:---:|:---:|:---:|
| 空間成分 $(x,y,z)$ | $(0,0,0)$ | $(0,0,0)$ | $(0,0,0)$ |
| ヒッグスとの空間軸共有 | なし | なし | なし |
| ヒッグスによる静止質量獲得 | なし | なし | なし |

光子が「静止質量ゼロだがエネルギーを持つ」ことは周知の事実である。$W^{\pm}$ と $Z^0$ も光子と同一のヒッグス非関与構造を持つ。

## §2 ゲージボソンのエネルギーの起源

### 2.1 グラビトンの符号ベクトル

[3] の表Aにおいて、グラビトン（スピン2、$tR$ 型）の符号ベクトルは:

$$G = (0, 0, 0, \pm, \pm, 0)$$

グラビトンは $t$ 軸と $R$ 軸の両方に非零成分を持つ。$W^{\pm}$ は $t$ 軸のみ、$Z^0$ と $\gamma$ は $R$ 軸のみに非零成分を持ち、いずれもグラビトンの非零軸の部分集合である。すなわち、$W^{\pm}$ および $Z^0$ の非零軸 $t, R$ は、グラビトンが媒介する加速度機構（[4] §8、§2.2 で詳述）と同じ軸構造を共有する。したがって、これらのボソンのエネルギーも [4] §8 の曲率由来加速度として記述されうる。

### 2.2 加速度の幾何学的起源

[4] §8 により、形不変波の加速度は中心投影の幾何学から内在的に導出される:

(i) **大域的加速度**（[4] §8.2）: $S^3$ の曲率半径 $R$ が波の軌道に大域的な加速度 $\Delta^2\mathbf{k}_{\text{global}} = f(R, \mathbf{k}_t)$ を与える

(ii) **局所的加速度**（[4] §8.3）: 子空間中心投影の Hesse 行列 $D^2\pi$ が各点で局所的な加速度 $\Delta^2\mathbf{k}_{\text{local}}$ を決定する

(iii) **統一**（[4] §8.4 命題8.2）: 大域的曲率と局所的非線形性は、$R$-パラメータ付き中心投影 $\pi: S^5(R) \to \mathbb{Z}^4$ という単一の写像の異なるスケールでの現れである

ここで決定的なのは、加速度関数 $f(R, \mathbf{k}_t)$ が $R$（曲率半径）の値に依存することである。中心投影の写像そのものは $R$ の符号に関わらず同一の正則写像として定義されるが（[2]で証明済み）、加速度関数の値は $R$ の符号に応じて異なり、これが加速度エネルギーの差として現れる。

### 2.3 R軸符号とエネルギーの関係

**命題 2.1**. 光子（$R = +$）と $Z^0$（$R = -$）のエネルギー差は、主観空間の曲率半径 $R$ の符号の違いに起因する。

*論証*. [4] §8.2 により、加速度は曲率半径 $R$ の関数 $f(R, \mathbf{k}_t)$ で与えられる。中心投影 $\pi: S^5(R) \to \mathbb{Z}^4$ は $R$ の符号に関わらず同一の正則写像として定義される（[2]で証明済み）が、$R$ の符号そのものが加速度関数 $f(R, \mathbf{k}_t)$ の値を左右する。$R = +$（光子）と $R = -$（$Z^0$）で異なる加速度エネルギーが計量され、これが両者のエネルギー差として現れる。

外部の観測者（別の主観空間）から見ると、$R < 0$ の主観空間に局在する波のエネルギーは、その空間の曲率に由来する加速度エネルギーを含む。これが $Z^0$ の 91.2 GeV として計量される。$\square$

### 2.4 t軸符号とW±のエネルギー

同様の論理が $W^{\pm}$（$t$ 軸のみ非零）にも適用される。$t$ 軸方向の構造が特定のエネルギースケールを決定し、これが 80.4 GeV として計量される。

## §3 質量獲得の2類型

### 3.1 類型の区別

本枠組みにおいて、「質量」として計量されるものには2つの構造的に異なる起源がある。

**類型I: ヒッグス由来の静止質量**（フェルミオン）

- 機構: 定常波（$\kappa = 1$、ヒッグス）との共有空間軸上での位相結合（[5] §8）
- 対象: 空間軸を非零に持つ方向制約波（$\kappa = 2$）、すなわちフェルミオン
- 結果: 実効的空間成分の減少 $k_{s_0}^{\text{eff}} = k_{s_0} - \delta k_H$ による速度制限（[5] §8.2）
- 特徴: ヒッグスとの空間軸共有の有無で世代質量階層が決まる

**類型II: 曲率由来のエネルギー**（空間成分ゼロのゲージボソン）

類型II のゲージボソンは空間成分がゼロであるため、ヒッグスとの空間軸共有による静止質量を持たない。実測される「質量」は、主観空間の曲率に由来する加速度エネルギーとして本枠組みで記述される。

- 機構: 主観空間の曲率半径 $R$ に依存する加速度（[4] §8）。中心投影は $R$ の符号に関わらず同一の正則写像であるが（[2]）、加速度関数の値は $R$ の符号に依存する
- 対象: 空間成分がゼロのボソン（$\gamma$, $Z^0$, $W^{\pm}$）
- 結果: 加速度関数 $f(R, \mathbf{k}_t)$ が決定する加速度エネルギー
- 特徴: $R$ 軸・$t$ 軸の符号がエネルギーに反映される

### 3.2 分類表

| 性質 | 類型I（ヒッグス由来） | 類型II（曲率由来） |
|:---|:---|:---|
| 対象 | フェルミオン（$\kappa = 2$） | ゲージボソン（$\gamma, Z^0, W^{\pm}$） |
| 空間成分 | 非零（1軸または2軸） | ゼロ |
| ヒッグスとの結合 | 空間軸共有による直接結合 | なし |
| 機構 | 速度制限 $v^{\text{eff}} < c$（[5] §8.2） | 曲率加速度（[4] §8） |
| 世代依存性 | あり（$m_3 \gg m_{1,2}$） | なし |
| 光子との構造的差異 | あり（空間成分の有無） | なし（同一構造） |

### 3.3 標準模型との対比

標準模型では、フェルミオン質量（湯川結合）とゲージボソン質量（ヒッグス場の共変微分）の両方をヒッグス機構で統一的に説明する。本枠組みでは、符号ベクトルの構造から2類型が自然に分離する。

| 粒子群 | 標準模型の機構 | 本枠組みの類型 |
|:---|:---|:---|
| フェルミオン | 湯川結合 $y_f \langle\phi\rangle$ | 類型I（空間軸共有） |
| $W^{\pm}, Z^0$ | 共変微分 $|D_\mu\phi|^2$ | 類型II（曲率エネルギー） |
| 光子 $\gamma$ | 結合なし（質量ゼロ） | 類型II（曲率エネルギー、$R > 0$） |
| グルーオン | 結合なし（質量ゼロ） | — （$Q$ 軸遷移型、別機構） |

グルーオンは空間成分ゼロだが、非零軸が $Q$ 軸上の離散ラベル遷移（[5] §5）であり、光子や $W^{\pm}/Z^0$ の $t, R$ 軸構造とは質的に異なる。したがって類型IIにも属さず、別機構として扱う。

標準模型では $W^{\pm}/Z^0$ と光子の質量差をヒッグス場の真空期待値と電弱対称性の破れで説明するが、本枠組みではこの差は $R$ 軸・$t$ 軸の符号が決定する加速度エネルギーの差に帰着する。

**注記**: 標準模型では $W^{\pm}/Z^0$ は静止質量を持つ massive ゲージボソンとして扱われるが、本枠組みでは空間成分ゼロのため静止質量を持たず、実測される 80.4 GeV および 91.2 GeV は曲率由来のエネルギーとして計量される。この解釈の違いは、本枠組みにおける「質量」の定義が標準模型と異なることを反映する。標準模型の電弱精密測定（$W/Z$ 質量比、$\sin^2\theta_W$ 等）との関係の詳細な検討は本稿の範囲外である。

## §4 本稿の結論

本稿では、[5] §1 で未解決とされた W±/Z⁰ の質量獲得機構に対し、以下の構造的回答を与えた。

1. **W±/Z⁰ はヒッグスと空間軸を共有しない**（命題1.1）。[3] の表Aの符号ベクトルから直接確認でき、[5] §8 の質量獲得機構（空間軸共有による位相結合）は適用されない。

2. **W±/Z⁰ は光子と同一のヒッグス非関与構造を持つ**（観察1.2）。空間成分がゼロであるという構造的特徴は3者で共通である。

3. **質量獲得は2類型に区別される**。フェルミオンの静止質量（類型I: ヒッグスとの空間軸共有）と、ゲージボソンのエネルギー（類型II: 主観空間の曲率に由来する加速度エネルギー、[4] §8）。

4. **光子と Z⁰ のエネルギー差は R 軸符号の差に帰着する**（命題2.1）。中心投影は $R$ の符号に関わらず同一の正則写像であるが（[2]）、$R$ の符号が加速度関数の値を左右し、$R = +$（光子）と $R = -$（$Z^0$）で異なるエネルギーが計量される。

5. **質量獲得の2類型が本枠組みで自然に分離する**（§3）。フェルミオン（類型I）とゲージボソン（類型II）の区別は、符号ベクトルの空間成分の有無から即座に導かれる。標準模型ではヒッグス機構により両者を統一的に説明するが、本枠組みではそれぞれ異なる幾何学的起源を持つ。

光子・$Z^0$・$W^{\pm}$ のエネルギーの具体的数値の導出（$R$ の符号と曲率半径の定量的関係）は本稿の範囲外であり、残された課題である。

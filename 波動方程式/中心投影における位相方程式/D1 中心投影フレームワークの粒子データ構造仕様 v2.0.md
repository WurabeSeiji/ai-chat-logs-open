## §1 int256 有界性と次元解釈

### §1.1 有界性要件の根拠（v1.0 から変更なし）

[W3] は万物の理論の構造要件として有界性を要求する。[M1] の三層構造 $R$–$R_1$–$R_0$ において、最大の曲率半径 $R_0$ は有限である。この有限性とプランクスケールの離散性から、全位相量の表現に必要なビット数の上界が導かれる。

### §1.2 必要ビット数

**空間軸**: 観測可能宇宙径 $D \approx 1.892 \times 10^{27}$ m、プランク長 $l_P = 1.616 \times 10^{-35}$ m から

$$n_{\text{space}} = D / l_P \approx 1.171 \times 10^{62}, \quad b_{\text{space}} = 207 \text{ bit}$$

**時間軸**: 宇宙時間 $T \approx 8.710 \times 10^{17}$ s、プランク時間 $t_P = 5.391 \times 10^{-44}$ s から

$$n_{\text{time}} = T / t_P \approx 1.616 \times 10^{61}, \quad b_{\text{time}} = 204 \text{ bit}$$

**結論**: 256ビット符号付き整数で空間 14 桁・時間 15 桁の余裕を持って表現可能。

### §1.3 自然単位系と次元

$c = 1$, $G = 1$（幾何学化単位系、$\hbar$ は採用しない）の下で

$$[L] = [T] = [M], \quad [L^{-1}] = [M^{-1}]$$

### §1.4 周波数解釈（v2.0 の中核改訂）

v1.0 では xyzt 軸の $[L^{-1}]$ を「波数」「加速度」と混在的に解釈していた。v2.0 では位相空間の自然な解釈として **「各軸方向の振動周波数」** に統一する。

| 物理量 | 次元 | v1.0 解釈 | **v2.0 解釈** |
|---|---|---|---|
| σ.k_x, k_y, k_z | $[L^{-1}]$ | 空間波数・加速度 | **各空間軸の振動周波数** |
| σ.k_t | $[L^{-1}]$ | 時間振動数 | **t 軸の振動周波数**（他軸と対等） |
| σ.R | $[L]=[M]$ | 曲率半径 | **スケール量・対応周期**（双対） |
| σ.Q | $[1]$ | 内部対称性 | 内部対称性（変更なし） |
| δt | $[L]=[M]$ | 固有時間ステップ | **処理ステップ周期**（時間ではない） |
| δδt | $[L]=[M]$ | （v1.0 になし） | **処理周期の変化**（加速度効果） |

### §1.5 周波数解釈の優位性

**(1) 6軸対称性の完全保持**: 位置型 4 軸（xyzt）はそれぞれ独立な振動モードを持ち、t 軸も他軸と対等。

**(2) 連続体力学概念の不要化**: 「加速度」は位置の二階時間微分という連続体概念であり、離散整数演算の枠組みには馴染まない。「振動周波数」は位相空間の自然な離散量。

**(3) 加速度効果の明確な分離**: 加速度的な効果（重力場・慣性系の影響）は処理周期 δt の変化 δδt として表現され、粒子固有の周波数 σ.k とは構造的に分離される。

**(4) 周波数 × 周期 = 位相**: $\sigma.k \cdot \delta t$ が無次元の位相増分となり（$[L^{-1}] \cdot [L] = [1]$）、位相発展則が型整合する。

**(5) 特殊相対論の自動内包**: dispersion relation $\sigma.k_t^2 = \sigma.k_x^2 + \sigma.k_y^2 + \sigma.k_z^2 + m^2$ が周波数間の整数関係として自然に書ける。

## §2 メタ型の定義

### §2.1 スカラー型（v1.0 から変更なし）

```
型名         次元       値域                      用途
L256         [L]=[M]   |v| ≤ 2^255 - 1          長さ・時間・質量・周期
Linv256      [L⁻¹]     |v| ≤ 2^255 - 1          周波数・スケール逆数
Scalar256    [1]       |v| ≤ 2^255 - 1          無次元量
FullPhase    [1]       0 ≤ v ≤ P                0–720°
CyclePhase   [1]       0 ≤ v ≤ P/2              0–360°（位置位相）
DeltaPhase   [1]       -P/2 ≤ v ≤ P/2           ±360°（位相変位）
SpinPhase    [1]       v ∈ {0, P/4, P/2, P}     {0, 180, 360, 720}°
```

位相定数 $P = \lfloor (2^{255}-1)/720 \rfloor \times 720$。

### §2.2 軸成分型（v2.0 で再命名・解釈変更）

| 型 | 基底 | 次元 | 物理的意味（v2.0） |
|---|---|---|---|
| `SpatialFreq` <: Linv256 | int256 | $[L^{-1}]$ | x, y, z 軸の振動周波数 |
| `TemporalFreq` <: Linv256 | int256 | $[L^{-1}]$ | t 軸の振動周波数（他軸と対等） |
| `CurvatureR` <: L256 | int256 | $[L]$ | スケール量・周期（符号付き平方数） |
| `ColorQ` <: Scalar256 | int256 | $[1]$ | 内部対称ラベル（有限群） |

**v1.0 からの改名**: `SpatialK` → `SpatialFreq`、`TemporalK` → `TemporalFreq`。型システム自体は不変、解釈の明確化のみ。

### §2.3 演算規則（v1.0 から変更なし）

**加算**: 同型同士のみ許可（CyclePhase + DeltaPhase = CyclePhase など）。

**乗算**:
- `L256 × L256 → L2_256` $[L^2]$
- `Linv256 × Linv256 → Linv2_256` $[L^{-2}]$
- `Linv256 × L256 → Scalar256` $[1]$（無次元化）

**除算**: 使用しない。

## §3 SignVector の定義（周波数解釈版）

### §3.1 構造

```
SignVector = (
    x : SpatialFreq,    -- x 軸の振動周波数 [L⁻¹]
    y : SpatialFreq,    -- y 軸の振動周波数
    z : SpatialFreq,    -- z 軸の振動周波数
    t : TemporalFreq,   -- t 軸の振動周波数 [L⁻¹]（他軸と対等）
    R : CurvatureR,     -- スケール量・周期 [L]
    Q : ColorQ,         -- 内部対称ラベル [1]
)
```

### §3.2 物理的意味

各 σ 成分は「該当軸での振動周波数」または「スケール量」：

- σ.k_axis = 0: 該当軸は振動しない（その方向の構造を持たない）
- σ.k_axis ≠ 0: 該当軸が周波数 |k_axis| で振動する
- 符号 sign(k_axis): 振動の位相方向（+ なら時計回り、− なら反時計回り）
- |k_axis|: 振動周波数の大きさ（離散整数値）

### §3.3 不変量

SignVector の値は相互作用イベントで変化するが、特定の組み合わせは安定モードを定義する：

- $\kappa_s = |\{i \in \{x,y,z\} : k_i \neq 0\}|$ — 空間非ゼロ数
- $\kappa_t = (1 \text{ if } k_t \neq 0 \text{ else } 0)$ — 時間非ゼロ指標
- $\kappa = \kappa_s + \kappa_t \in \{0, 1, 2\}$ — [W10] 命題2.1: $\kappa \geq 3$ は不安定

### §3.4 dispersion relation

質量殻条件は周波数間の整数関係として直接表現：

$$\sigma.k_t^2 = \sigma.k_x^2 + \sigma.k_y^2 + \sigma.k_z^2 + m^2$$

- $m = 0$（光子・グルーオン）: $k_t^2 = k_x^2 + k_y^2 + k_z^2$
- $m \neq 0$（質量粒子）: 差分の平方根が質量に対応

これにより特殊相対論が自動的に枠組みに含まれる。

### §3.5 W8 表 A との対応（v1.0 から継続）

[W8] 表 A の符号パターン $\{+, -, 0\}^6$ が SignVector の符号と対応する。整数値（軸スケール値）は [質量構造] §1 の $v_i$ に対応。

代表粒子：

| 粒子 | 集合 $(x,y,z,t,R,Q)$ | $\kappa$ | 分類 |
|---|---|---|---|
| 光子 $\gamma$ | $(0,0,0,0,+,0)$ | 0 | ゲージボソン（自由・球状波） |
| グルーオン $g$ | $(0,0,0,0,0,Q_i \to Q_j)$ | 0 | ゲージボソン |
| ヒッグス $H$ | $(0,0,+,0,0,0)$ | 1 | スカラー（縦波解釈は §7） |
| $W^+$ | $(+,0,0,0,+,0)$ | 1 | ゲージボソン |
| $Z^0$ | $(0,+,0,0,+,0)$ | 1 | ゲージボソン |
| 重力子 | $(0,0,0,\pm,\pm,0)$ | 1 | テンソルボソン |
| 電子 $e^-$ | $(+,0,0,+,0,0)$ | 2 | フェルミオン |

## §4 OneDWave の定義

### §4.1 構造（v1.0 から変更なし）

```
OneDWave = (
    θ₀ : CyclePhase,   -- 位置位相 [1]      (S1 §2.1, W9 §8.2)
    A  : Scalar256,     -- 振幅 [1]          (S1 §2.1, W9 §8.2)
    ℓ  : DeltaPhase,    -- 広がり位相 [1]    (W9 §8.3)
)
```

### §4.2 物理的意味

各粒子は非零位置型軸ごとに 1 つの OneDWave を持つ（最大 $\kappa$ 個）。各 OneDWave はその軸方向の振動の現在位相状態を保持する。

- $\theta_0$: 振動の現在位相（自由パラメータ、時間発展で変化）
- $A$: 振幅（自由パラメータ）
- $\ell$: 振動の局在広がり（SignVector から従属、[W9] §8.3）

## §5 Particle の定義（v2.0 で δV, δδt を追加）

### §5.1 構造

```
Particle = (
    σ   : SignVector,    -- 個性（各軸の振動周波数）
    V   : OneDWave[κ],   -- 各非零位置型軸の現在位相状態
    δV  : OneDWave[κ],   -- 1 ステップでの位相変位（v2.0 で追加）
    δt  : DeltaT,         -- 現在の処理ステップ周期
    δδt : DeltaT,         -- 処理周期の変化（v2.0 で追加、加速度相当）
)
```

### §5.2 各フィールドの物理的意味

#### `σ : SignVector`

粒子の個性。各軸でどの周波数で振動するか。相互作用イベントで値が変化する（消滅＋生成として記述、しかしデータ構造としては σ の値が更新される）。

#### `V : OneDWave[κ]`

各非零位置型軸の現在位相状態。配列長 $\kappa$ は SignVector の非零位置型軸数（$\leq 2$）。各 V[j] は対応軸の振動の (位置位相, 振幅, 広がり) を保持する。

インデックス対応：

```
σ.k_x ≠ 0 のとき V に wave_x が含まれる
σ.k_y ≠ 0 のとき V に wave_y が含まれる
σ.k_z ≠ 0 のとき V に wave_z が含まれる
σ.k_t ≠ 0 のとき V に wave_t が含まれる
```

#### `δV : OneDWave[κ]` (v2.0 で追加)

V と同型の差分構造。1 処理ステップで V がどう変化するかを保持する：

```
δV[j] = (
    Δθ₀ : DeltaPhase,    -- 位相変位
    ΔA  : Scalar256,      -- 振幅変位
    Δℓ  : DeltaPhase,     -- 広がり変位
)
```

自由粒子の場合、δV[j].Δθ₀ は σ から計算可能：

$$\delta V[j].\Delta\theta_0 = \sigma.k_{a(j)} \cdot \delta t$$

ただし相互作用イベントでは δV が独立に変更されうる。

#### `δt : DeltaT`

**処理ステップ周期**（時間ではない）。普遍的な演算カウンタ。

- $\delta t > 0$: 順方向演算（因果の順方向）
- $\delta t < 0$: 逆向き演算（可逆性の保証）
- $\delta t = 0$: 静止

時間として観測されるのは V[t-軸].θ₀ の累積であり、δt 自身ではない（§6.4 参照）。

#### `δδt : DeltaT` (v2.0 で追加)

**処理周期の変化**（加速度相当）。背景 R(r) または相互作用によって決まる：

```
1 ステップ更新: δt += δδt
```

物理的解釈：

- $\delta\delta t = 0$: 自由粒子（処理周期が一定）
- $\delta\delta t > 0$: 処理周期が伸びる（重力場・加速系で時計が遅くなる）
- $\delta\delta t < 0$: 処理周期が縮む（重力ポテンシャル上昇等）

### §5.3 階層は δδt で完結

物理法則が 2 階微分で閉じる（ニュートン力学・GR・量子力学）ため、δδδt 以上の高階は不要。**5 フィールド（σ, V, δV, δt, δδt）で動力学表現として完全**。

### §5.4 メモリフットプリント

```
σ      : 6 × 256 = 1,536 bit
V      : κ × 3 × 256 ≤ 1,536 bit (κ ≤ 2)
δV     : κ × 3 × 256 ≤ 1,536 bit
δt     : 256 bit
δδt    : 256 bit
合計   ≤ 5,120 bit = 640 byte / 粒子
```

v1.0 の 224–416 byte から増加するが、依然として宇宙の任意の粒子を完全に表現する量として極めて少ない。

## §6 演算則

### §6.1 基本：1 処理ステップでの更新

各非零位置型軸 $j$ に対して：

$$\delta V[j].\Delta\theta_0 \leftarrow \sigma.k_{a(j)} \cdot \delta t$$

$$V[j].\theta_0 \mathrel{+}= \delta V[j].\Delta\theta_0$$

そして処理周期自体を更新：

$$\delta t \mathrel{+}= \delta\delta t$$

### §6.2 型整合検査

```
σ.k_axis · δt : SpatialFreq × DeltaT 
              = Linv256 × L256 
              = Scalar256 [1]                ✓

V[j].θ₀ + δV[j].Δθ₀ : CyclePhase + DeltaPhase 
                    = CyclePhase [1] (mod P/2)  ✓

δt + δδt : DeltaT + DeltaT 
         = DeltaT [L]                       ✓
```

### §6.3 偏光・複数軸位相差の表現

複数の非零軸を持つ粒子（例：偏光を持つ一方向光）：

```
σ_polarized_light = (+k_x, +k_y, 0, 0, +R_γ, 0)
V = [wave_x, wave_y]
```

偏光位相差として：

$$\Delta\varphi = V[0].\theta_0 - V[1].\theta_0 \in \text{CyclePhase}$$

| 偏光状態 | $\Delta\varphi$ |
|---|---|
| 線偏光（同方向） | $0$ |
| 線偏光（反方向） | $P/4 = 180°$ |
| 円偏光（左） | $+P/8 = +90°$ |
| 円偏光（右） | $-P/8 = -90°$ |
| 楕円偏光 | 任意の整数値 |

全て int256 整数で正確に表現される（$15°$ の整数倍は $P/48$ の整数倍として誤差なし）。

### §6.4 時間の取り扱い：δt は処理ステップ、時間は V[t].θ₀

**重要原則**: δt は時間ではなく、普遍的な処理ステップ。時間として観測されるのは粒子ごとの V[t-軸].θ₀ の累積値。

電子の例（σ.k_t ≠ 0）:

```
σ_electron = (+k_x, 0, 0, +k_t, 0, 0)
V = [wave_x, wave_t]
1 ステップ:
  δV[0].Δθ₀ = σ.k_x · δt    -- x 軸位相変位（運動量相当）
  δV[1].Δθ₀ = σ.k_t · δt    -- t 軸位相変位（電子の固有時間進行）
  V[0].θ₀ += δV[0].Δθ₀
  V[1].θ₀ += δV[1].Δθ₀       -- 「電子が経験する時間」
  δt += δδt                    -- 処理周期更新（重力場で δδt ≠ 0）
```

光子の例（σ.k_t = 0）:

```
σ_photon = (+k_x, +k_y, 0, 0, +R_γ, 0)
V = [wave_x, wave_y]      -- t 軸成分なし
δV = [δwave_x, δwave_y]
-- σ.k_t = 0 ゆえ V に t 軸 OneDWave は存在しない
-- 結果: 光子の固有時間ゼロが構造的に保証される
```

### §6.5 加速度効果（背景結合）

δδt は背景 R(r) や相互作用によって決まる。具体的な決定規則は別稿（D2）で定義する。

定性的には：

- 重力場中：$\delta\delta t = f(R_{\text{背景}}, \sigma)$ により背景曲率が処理周期を変調
- 結果：σ.k · δt の値が変化 → 周波数シフト・時間遅れ・偏光位相回転

これらすべてが整数演算による位相発展として表現される。

### §6.6 必要としない演算（v1.0 と同じ）

以下の演算は本仕様の範囲では必要としない：

1. **除算**: $R/R_1$ は $R \cdot R_1^{-1}$ として乗算に変換
2. **平方根**: 連続体的概念。離散系では整数値が直接的に物理量を決定
3. **三角関数**: 連続体 sine-Gordon の道具。離散変調表で代替可能
4. **浮動小数点演算**: 全フィールド int256、丸め誤差は構造的に発生しない

cos/sin が必要な箇所（例：縦波振動の変調）は、有限段階の離散変調表（lookup table）で代替する。

## §7 Mesh 構造の新設（v2.0 で追加）

### §7.1 縦波モードの分離

[W8] 表 A の解釈において、ヒッグス（$\sigma_H = (0, 0, +k_z, 0, 0, 0)$）は他のスピン1ボソンと異なり $\sigma.R = 0$ かつ $\text{spin} = 0$ である。これは構造的に「位置位相を振動させる横波粒子」ではなく、**「メッシュ自身の縦波振動」** として扱うのが自然である。

同様に、グラビトン（重力波）も時空計量の縦波的振動として、Mesh の振動モードと解釈できる。

これらの縦波モードを Particle として扱うと、データ構造に無理が生じる（OneDWave は本質的に位相回転を記述する型であり、軸方向の伸縮振動を直接表現できない）。本稿では **Mesh 構造を新設し、縦波モードを Particle と分離管理** する。

### §7.2 Mesh の構造

```
Mesh = (
    -- 背景スケール
    R_0           : L256,             -- 中心投影の中心長さ（背景）
    
    -- 縦波モード
    H_xyz_inphase : OneDWave,         -- xyz 同位相膨張収縮 = ヒッグス
    H_xy_anti     : OneDWave,         -- xy 反位相 = 重力波 + 偏光
    H_xy_quarter  : OneDWave,         -- xy 90° 位相 = 重力波 × 偏光
    H_yz_anti     : OneDWave,         -- yz 反位相 = 重力波の他偏光
    H_xz_anti     : OneDWave,         -- xz 反位相 = 重力波の他偏光
    
    -- 縦波モードの差分
    δH_xyz_inphase : OneDWave,
    δH_xy_anti     : OneDWave,
    δH_xy_quarter  : OneDWave,
    δH_yz_anti     : OneDWave,
    δH_xz_anti     : OneDWave,
    
    -- グローバル処理ステップ
    δt_mesh       : DeltaT,
    δδt_mesh      : DeltaT,
)
```

### §7.3 縦波モードの物理同定

| モード | 振動パターン | 物理同定 | spin |
|---|---|---|---|
| H_xyz_inphase | xyz 軸の同位相伸縮 | ヒッグス（スカラー） | 0 |
| H_xy_anti | x 伸び・y 縮みの反位相 | 重力波 + 偏光 | 2 |
| H_xy_quarter | xy 90° 位相回転 | 重力波 × 偏光 | 2 |
| H_yz_anti | yz 反位相 | 重力波（別偏光） | 2 |
| H_xz_anti | xz 反位相 | 重力波（別偏光） | 2 |

**ヒッグス = xyz 軸の球対称同位相縦波**: W8 における z 軸選択は表記上の代表であり、本質は 3 軸対称な縦波振動。

**重力波の 2 偏光 (+, ×)**: H_xy_anti と H_xy_quarter（および軸の選び替えで他偏光）に対応。

### §7.4 Mesh と Particle の結合

粒子の振動周波数 σ.k は Mesh の状態によって実効的に変調される。質量生成・周波数シフトはこの変調として表現される。

実効周波数：

$$\sigma.k_{\text{effective}}(particle, mesh) = \sigma.k(particle) \cdot \text{mod}(mesh)$$

$$\text{mod}(mesh) = 1 + \alpha_H \cdot \text{disc-cos}(H_{\text{xyz\_inphase}}.\theta_0) + \cdots$$

ここで `disc-cos` は連続三角関数の代替として有限段階の離散変調表（例：12 段階で 30° 刻み）。

### §7.5 動力学

Mesh の更新：

```
For each longitudinal mode H_*:
    δH_*.Δθ₀ ← (固有周波数) · δt_mesh
    H_*.θ₀  += δH_*.Δθ₀
δt_mesh += δδt_mesh
```

Particle の更新（Mesh に従属）：

```
For each particle p:
    For each non-zero axis j:
        δV[j].Δθ₀ ← σ.k_effective(p, mesh) · δt
        V[j].θ₀  += δV[j].Δθ₀
    δt += δδt
```

### §7.6 メモリフットプリント

```
R_0    : 256 bit
H_*    : 5 × 3 × 256 = 3,840 bit
δH_*   : 5 × 3 × 256 = 3,840 bit
δt_mesh, δδt_mesh : 2 × 256 = 512 bit
合計   = 8,448 bit = 1,056 byte / 全宇宙
```

宇宙全体で約 1 KB。粒子数 $N$ に対して総メモリは $640N + 1056$ byte。

## §9 v1.0 からの非互換性

### §9.1 解釈の変更

- σ.k は「波数・加速度」から「振動周波数」へ統一
- δt は「固有時間ステップ」から「処理ステップ周期」へ明示化
- 時間は V[t-軸].θ₀ として現れる派生量

### §9.2 構造の拡張

- Particle に δV、δδt を追加（v1.0 の Particle はサブセットとして含まれる）
- Mesh 構造を新設（v1.0 にはなかった）

### §9.3 W8 表 A との整合

[W8] 表 A の 62 粒子分類は v2.0 でも有効。ただしヒッグスは **Particle として数える代わりに、Mesh の H_xyz_inphase 励起として数える** という解釈の更新がある。標準模型 61 状態 + グラビトン 1 状態 = 62 という総数は変わらない（数え方の解釈は更新）。

## §10 結語

v2.0 の本質：

- **xyzt は周波数**（位相空間の自然な解釈）
- **6 軸完全対称**（位置型 4 軸 + R + Q）
- **横波（Particle）と縦波（Mesh）の分離**
- **加速度は δδt**（処理周期の変化）
- **時間は V[t].θ₀**（位置位相として粒子ごとに進む）
- **全演算は int256 整数で閉じる**

この構造で動力学規則（D2、別稿）を書けば：

- 重力時間遅れ・偏光回転（自由運動 + 背景結合）
- ヒッグス機構（Mesh の H_xyz_inphase 結合）
- 重力波放射（Mesh の H_xy_anti 励起）
- クーロン 2 体問題（粒子間相互作用）

が全て同じ枠組みで計算可能になる。

本稿は設計仕様であり、これらの動力学的応用は別稿に委ねる。

# 超直方体の組合せ論から標準模型の62粒子を導出する ── 位相方程式篇（W1〜W11）

## はじめに

本記事は、中心投影フレームワークの基礎シリーズ（論文1〜10）の続編にあたる**位相方程式篇**（W1〜W11）の解説です。

基礎シリーズでは、中心投影の幾何学的性質から「複数の独立な主観空間が矛盾なく共存しうる統一的フレームワーク」を構築しました。

位相方程式篇では、このフレームワークの上に**離散的な超直方体の組合せ構造**を導入し、そこから素粒子物理学の標準模型に現れる粒子の分類が、どのように導出されるかを段階的に示します。

第I部〜第IV部（W1〜W8）で粒子分類の完全な導出を行い、第V部（W9）でソリトン理論の基盤を整備したのち、第VI部（W10・W11）では**形不変波のモード分類と相互作用**を定式化し、スピン・カイラリティ・統計の起源と4種の力の統一的記述を与えます。

基礎シリーズの解説記事はこちら：
https://note.com/kiharanoriaki/n/nc51e43282b0a

## シリーズの流れ

### 第I部：なぜこの構造が必要か（W1〜W4）

**W1・W2** では、中心投影フレームワーク上に位相空間と波動方程式の記述を導入します。

**W3** では、「万物の理論」が満たすべき**9つの構造要件**を定式化します。特定の理論を支持するものではなく、任意の候補理論が整合性を保つための最小条件です。

**W4** では、標準模型に存在する**19個の任意パラメータ**（質量、混合角、結合定数など）を構造的に分類し、どのパラメータが幾何学的起源を持ちうるかを分析します。

### 第II部：なぜ5次元なのか（W5）

**W5** は数学的に最も美しい結果の一つです。

「ラグランジュの四平方定理」（任意の正整数は4つの平方数の和で表せる）と中心投影の射影幾何的条件を組み合わせると、離散空間で中心投影の全球被覆が成立するためには**5次元の背景空間が必然**であることが証明されます。

つまり、私たちが住む「3次元空間 ＋ 1次元時間」の4次元時空は、整数論から導かれる必然だということです。

### 第III部：スピンの幾何学的起源（W6・W7）

**W6** では、5次元正軸体（cross-polytope、10個の頂点を持つ多胞体）の配向構造を分析し、素粒子のスピン値の完全分類を導出します。

- フェルミオン（物質粒子）とボソン（力の粒子）の区別
- 力の方向性（引力のみ vs 引力・斥力の両方）の分類
- 世代構造（なぜ電子・ミューオン・タウの3世代があるか）

**W7** では、正軸体の双対多胞体（5次元超直方体、32個の頂点）で同じ分類を再導出し、両者が同型であることを証明します。

### 第IV部：標準模型62状態の導出（W8） ★核心

**W8 がこのシリーズの中心的成果です。**

5次元超直方体に**8値の離散ラベル軸Q**を追加した6次元構造を定義します。Qは3ビット（c₁, c₂, c₃）で符号化され、Q = 4c₁ + 2c₂ + c₃ ∈ {0, 1, ..., 7}です。c₂c₃ = 00は色荷なし（レプトン）、c₂c₃ ∈ {01, 10, 11}は3色のクォークに対応します。c₁は弱アイソスピンの上下を符号化します。Q軸とt軸は独立（直交）な自由度です。

■ 論文の本体（§1〜§8）：純粋な組合せ論

物理用語を一切使わず、以下を数え上げだけで導出します：

**スピンの定義**：xyztの4軸のうち、値がゼロでない軸の本数kで判定
- k = 2 → フェルミオン型（スピン = n + 1/2）
- k ≠ 2 → ボソン型（スピン = n）

**Qラベル間遷移の数え上げ**：非零Q値{1, 2, 3}間の遷移
- 3ビット構造（c₁, c₂, c₃）の c₂c₃ 非零値3値間の順序対：3×3 = 9通り

**スピン1集合の数え上げ**：
t固定 2状態 ＋ R固定 2状態 ＋ Q遷移 9状態 ＝ 13状態

**符号積の完全分類**：t軸とR軸の符号の積Pを定義し、各スピン型ごとにPの取りうる値を分類。

■ 解釈例（§9）：標準模型との対応

上記の組合せ論的結果を標準模型に対応付ける**一つの解釈例**です。

- ヒッグス（スピン0）：1状態 → ヒッグスボソン
- ゲージボソン・t型（スピン1）：2状態 → W+, W-
- ゲージボソン・R型（スピン1）：2状態 → 光子, Z
- グルーオン・Q遷移型（スピン1）：8状態 → グルーオン8種（9通りから色中性モード1を除外）
- グラビトン・tR型（スピン2）：1状態 → 重力子
- レプトン＋反レプトン（スピン1/2）：12状態 → 電子, ニュートリノ等
- クォーク＋反クォーク（スピン1/2）：36状態 → アップ, ダウン等
- **合計：62状態 → 標準模型61 ＋ グラビトン**

Q軸の3ビット（c₁, c₂, c₃）の内訳：
- c₂c₃ = 2c₂ + c₃：SU(3)_C 色荷（c₂c₃=00=色荷なし=レプトン、c₂c₃∈{01,10,11}=3色=クォーク）
- c₁：弱アイソスピン（c₁=0=下型、c₁=1=上型）
- Q軸とt軸は独立（直交）な自由度

■ 未解決問題：質量

質量の導出は未完成ですが、有望な着想があります。xyzt軸の値がゼロでない本数に応じて4種類のスケール値（v1, v2, v3, v4）が自然に現れます。ヒッグスボソン（1本だけ非零、v1）の質量が125 GeVであるのに対し、電子（2本非零、v2）は0.5 MeVと圧倒的に軽い。この v1 >> v2 の関係が質量階層の幾何学的起源となる可能性があります。

### 第V部：ソリトン理論の基盤（W9）

**W9** は独立したテーマで、sine-Gordon方程式の位相的ソリトン理論を厳密に整理します。方程式に含まれる6つの暗黙の前提を明示し、矩形波極限での構造保存性を証明します。

### 第VI部：形不変波のモード分類と相互作用（W10・W11）

**W10** と **W11** は、W8で導出した62状態の上に**波の動力学**を構築します。

#### W10：形不変波の4つのモード

波動ベクトルの非ゼロ成分数 κ（カッパ）に基づいて、形不変波を3つの安定モードに分類します。

**モード分類**：
- **方向制約波**（κ=2）：(空間1軸, t) 平面内で直線的に伝搬
- **球状波**（κ=0）：空間方向なし; 等方的に伝搬
- **定常波**（κ=1）：空間1軸の定常振動; スカラー

κ≥3 は Nyquist安定条件と方向的非局在の両方から排除されます。

**主要な結果**：
- **パウリ排他律の幾何学的起源**：κ=2 の方向制約波では、2軸平面の符号付き面積が2つの向きしか許容しないため、同一波動ベクトルの3つ以上の共存が幾何学的に禁止されます
- **カイラリティ**：(空間, t) 平面では t 軸の固定方向からカイラリティが定義されますが、(空間, 空間) 平面ではカイラリティが縮退し、マヨラナ条件を満たしうる
- **因果順序の不在**：kt=0 の球状波は時間反転対称であるため因果順序を内在しない

#### W11：形不変波の相互作用

W10のモード分類を基盤として、異なるモード間の**相互作用**を統一的に記述します。

**軸方向変位転送** ── これがすべての相互作用の基本機構です。球状波（κ=0）が方向制約波（κ=2）と相互作用するとき、球状波の非ゼロ軸方向に沿って方向制約波の状態が変化します。**4種の力**（電磁気力・弱い力・強い力・重力）は、この同一操作の異なる軸への適用として統一されます。

**主要な結果**：
- **質量の幾何学的起源**：定常波（κ=1）との結合が方向制約波の速度比 |kx/kt| を修正し、質量として現れる
- **波束変形の決定論**：保存量 ∫|ψ|²dV のもとで A²·w³=E₀=const が成立し、波束の振幅と幅が反比例的に変形する。これが「波束の収縮」の力学的記述を与える
- **因果の遡及的構成**：kt=0 の球状波が媒介する相互作用では因果順序が存在せず、相互作用結果から最短測地線経路が遡及的に構成される
- **引力と斥力**：方向制約波間の相互作用の符号は、t 軸成分の相対符号（同符号=引力、異符号=斥力）により幾何学的に決定される

## 何を主張し、何を主張しないか

**主張すること：**
- 6次元超直方体の組合せ構造から62状態が純粋に数え上げで導出される
- この62状態は標準模型の粒子分類と対応付け可能な構造を持つ
- 波動ベクトルの非ゼロ成分数 κ が形不変波を3モードに分類し、各モードの幾何学的性質がスピン・カイラリティ・統計の起源を与える
- 4種の力は軸方向変位転送の軸の違いとして統一的に記述される

**主張しないこと：**
- 標準模型を物理的に「導出」したこと
- 質量値、結合定数、混合行列の具体的数値の導出
- 未発見粒子の存在の予言
- ゲージ群の連続極限における厳密な再構成

§9の標準模型との対応は「解釈の一例」であり、別の対応付けが可能になった場合に並列追加できる構成としています。

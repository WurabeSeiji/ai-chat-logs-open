# 微細構造定数 α を4次元の幾何学から 8.7 ppb で予言する自己整合恒等式

## 100年の謎、Eddington の罠、そして本稿の立ち位置

物理学で最も有名な無次元定数のひとつに、微細構造定数 α があります。値はおよそ 1/137.036。電磁相互作用の強さを規定し、原子の構造、光と電子の振る舞い、ひいては宇宙のあらゆる電気的現象がこの数値で決まります。

天文学者 Eddington は1929年、α が 1/137 ちょうどであるとする整数的な議論を提示しました。しかし測定精度が向上するにつれ、137 ではなく 137.036 であることが確定し、Eddington の主張は棄却されました。その後 Wyler (1969) ら多数の試みも、ほぼすべて失敗しています。**α を綺麗な数式で当てる** という挑戦には、100年の墓場があります。

物理学者の脊髄反射は、α の予言式を見せられると条件反射的に **またEddingtonか** と判定すること。これは健全な懐疑であり、生半可な提案を防ぐ重要な防御線です。

本稿が紹介する恒等式は、この罠を3点で回避します：

- 整数のみで当てるのではなく、α 自身を含む **自己整合方程式** である
- 整数 137 と補正係数 π²/2 の **両方が独立な4次元幾何学から直接派生** する（外部から借りていない）
- 摂動展開構造をもち、QFT の繰り込み理論と類比的

そして最も重要な留保：本稿は α の第一原理導出を主張しません。提示するのは **観察された数値構造** であり、それが幾何学的必然か数値的偶然かは未解決問題として残します。

---

## 結論先出し：恒等式

二つの初等的な4次元幾何学量から構成される代数恒等式：

```
α⁻¹ = N(1) + V₄(1) · α
    = 137 + (π²/2) · α
```

ここで：

- N(1) = 137：半径 3 の4次元球内に整数中心の単位立方体が完全に内接する個数
- V₄(1) = π²/2：4次元単位球の体積

これを α について整理すると、自己整合な二次方程式：

```
(π²/2) α² + 137 α - 1 = 0
```

正根（高精度計算）：

```
α   ≈ 7.29735194 × 10⁻³
α⁻¹ ≈ 137.0360110
```

CODATA 2018 推奨値 α = 7.29735257 × 10⁻³（α⁻¹ = 137.035999...）と相対誤差 **8.7 × 10⁻⁸（約 8.7 ppb）** で一致します。これは Eddington 系の整数当てのズレ（2.6 × 10⁻⁴）の **約 1/3000** という驚異的な精度です。

整数部 137 は厳密に一致し、α⁻¹ の小数第4位（137.0360）まで完全に一致します。

> 訂正注：本記事 v1 では「相対誤差 1.4 × 10⁻⁴（0.02% 精度）」と記載していましたが、これは α を4桁丸め（7.2984 × 10⁻³）した粗い計算による過小評価でした。Grok（xAI）による外部高精度検証で実際の精度は **約 1600 倍高い 8.7 ppb** であることが判明し、論文 v3 に合わせて本記事も訂正しました。

## 構成要素はどこから来るか

恒等式の説得力は、**137 と π²/2 が外部から借り物ではなく、独立な4次元幾何学から直接派生する** ことに依存します。

### 137 の正体：4次元単位立方体の充填数

直径 6（半径 3）の4次元超球の中に、辺の長さ 1 の4次元単位立方体が完全に何個入るかを数える問題です。中心が整数座標の立方体だけを考え、はみ出すものは除外します。

これは中学レベルの不等式問題に帰着します：

```
(|c₁|+1/2)² + (|c₂|+1/2)² + (|c₃|+1/2)² + (|c₄|+1/2)² ≤ 9
```

中心の絶対値パターンで列挙すると、

- 原点だけ：1個
- 1軸方向に1単位：8個（4軸 × 2符号）
- 2軸方向に各1単位：24個
- 3軸方向に各1単位：32個
- 4軸方向に各1単位：16個
- 1軸方向に2単位：8個
- 1軸に2単位＋別軸に1単位：48個

合計 1 + 8 + 24 + 32 + 16 + 8 + 48 = **137**。

これは離散的・厳密な数で、計算機なしで紙とペンで検証できます。

### π²/2 の正体：4次元単位球の体積

n 次元ユークリッド球の体積公式は標準教科書に載っています：

```
Vₙ(R) = π^(n/2) / Γ(n/2 + 1) · Rⁿ
```

n=4, R=1 で：

```
V₄(1) = π² / Γ(3) = π² / 2 ≈ 4.9348
```

中学校の球の体積（3次元なら 4πR³/3）の4次元類比です。

### 補足：4+1次元ブラックホールエントロピー単位との一致

参考までに、4+1次元の Schwarzschild–Tangherlini ブラックホール（4空間次元＋1時間次元の高次元BH）の Bekenstein–Hawking エントロピーは、プランク単位で：

```
S_BH(r_h) = (π²/2) r_h³
```

事象の地平面の半径が 1 のとき、S_BH(1) = π²/2 = V₄(1)。これは標準BH熱力学から直接得られる事実で、本恒等式の構造解釈に用います。

## 摂動展開という構造

恒等式 α⁻¹ = 137 + (π²/2)α は等価に：

```
1 = 137α + (π²/2) α²
```

α の冪展開構造（tree-level 項と α² オーダーの補正項の和が単位）を持っています。これは量子場理論（QFT）における結合定数の繰り込みと類比的な構造です。

具体的には：

- 第1項 137α：tree-level の幾何学的結合（**inside**）
- 第2項 (π²/2)α²：1次補正項としての自己エネルギー（**outside**）

---

## 物理的読み替え：inside / outside 分解

R=3 の4次元球の構造を見ると：

- 球の総体積：V₄(3) = 81 × π²/2 ≈ 399.72
- 137個の立方体が占める体積：137（34%）
- 隙間（埋め残し）：262.72（**66%**）

なんと **球の半分以上が境界の埋め残し**です。R=3 の系は内部より境界が支配的。

恒等式の各項を以下のように読み替えると、構造が腑に落ちます：

- 137α 項（**inside**）：137個の幾何学的接点が結合定数 α で寄与
- (π²/2)α² 項（**outside**）：境界自由度（66%の隙間）が α² で寄与し、自己エネルギー補正となる

電子のような素粒子をこの幾何学的構造として読むと、内部は離散的な剛性、外部は連続的な余剰、両者の組み合わせで結合定数 α が決まる、という描像になります。

### なぜ単位球（R=1）が出るのか

補正項に出るのが V₄(R=3) = 81π²/2 ではなく V₄(R=1) = π²/2 である理由は、自己エネルギー補正が **個別の単位立方体スケールでの局所過程** であることを示唆します：

- 各立方体は隣接する単位影響圏（半径 1 の4次元球）と相関
- 137 個すべてが同一の単位影響圏を共有（強度的）
- 全体補正は単位影響圏ぶんに集約

## Eddington 罠との決定的差異

α を予言する試みが100年失敗してきた理由を踏まえ、本恒等式が異なる4点を整理します：

- **形式が違う**：単一の数式で当てるのではなく、α 自身を含む自己整合二次方程式の解として α が現れる
- **整数 137 が独立に派生する**：4次元単位立方体充填の直接計数による離散的事実で、α の存在を仮定せずに定まる
- **補正係数 π²/2 が独立に派生する**：4次元単位球の体積公式そのもので、教科書から直接出る
- **摂動展開構造を持つ**：1 = 137α + (π²/2)α² は QFT の繰り込み理論と類比的

それでも残る正直な留保：**恒等式自体が幾何学的必然か数値的偶然かは未解決問題**です。本稿は判定を行わず、観察された構造を提示するに留めます。

## 物理機構の候補

恒等式を物理的に正当化するには、(π²/2)α という補正項を独立な機構から導出する必要があります。3つの候補：

- **真空偏極**：標準QED の電子・正電子対による光子伝搬関数のドレッシング
- **境界自由度の自己相関**：inside/outside 分解の境界モードが直接生む補正
- **縦波スカラー場とのビート結合**：ヒッグス場の縦振動と電子ソリトンの横振動の干渉

これらは排他的でなく、同一現象の異なる側面である可能性もあります。第一原理導出は今後の課題です。

---

## 開問題

本恒等式は完璧ではありません。以下の課題が未解決です：

- **残差 8.7 ppb**：予言と実測の僅かな差（約 6 × 10⁻¹⁰）。α³ オーダーの補正項（係数 c₃ ≈ 1.6 × 10⁻³）の幾何学的同定が必要
- **R=3 の特権性**：N(0)=1, N(1)=137, N(2)=1545, N(3)=7281, ... のうち、なぜ電子が N(1) に対応するのか
- **物理機構の派生**：3候補のいずれが効くか、定量的評価
- **世代質量階層**：α は世代普遍だが、電子・ミューオン・タウの質量比は本恒等式の射程外

## 検証可能性

論文の §2 の手計算（前述のパターン表）と電卓だけで、本恒等式は完全に再現できます。

```
V₄(1) = π² / 2 ≈ 4.9348022
N(1) = 137
方程式：4.9348 α² + 137 α - 1 = 0
判別式：D = 137² + 2π² ≈ 18788.7392
√D ≈ 137.0720220（高精度）
α ≈ 7.29735194 × 10⁻³（高精度）
α⁻¹ ≈ 137.0360110
観測（CODATA 2018）：α⁻¹ = 137.0359991
相対誤差：8.7 × 10⁻⁸（8.7 ppb）
```

## 結論

4次元単位球の二つの初等的幾何学量、

- N(1) = 137（充填数、直接計数）
- V₄(1) = π²/2（標準体積公式）

から、自己整合代数方程式 α⁻¹ = N(1) + V₄(1)α を構成すると、観測 α と **8.7 ppb（約 8.7 × 10⁻⁸）** の精度で一致します。これは Eddington 系の整数当てのズレ（2.6 × 10⁻⁴）の約 1/3000 という驚異的な精度です。この恒等式は：

- 数学的に：手計算で完全に検証可能
- 構造的に：摂動展開 1 = 137α + (π²/2)α² の形を取る
- 物理的に：inside/outside 分解として解釈可能、機構候補が独立に提示されている

恒等式が幾何学的必然か数値的偶然かは未解決です。本稿の主張は判定ではなく、外部から借りない4次元幾何学量だけで構成され、観測値と 8.7 ppb 精度で一致する自己整合方程式の **観察** です。

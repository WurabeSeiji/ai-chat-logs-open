## 動機：精度が偶然と思えない

論文7 は4次元充填問題という純粋幾何学的アプローチから出発し、

```
α⁻¹ = 137 + (π²/2)·α
```

という自己整合恒等式を観察しました。これは α を **8.7 ppb（10億分の1）精度**で予言します。これは Eddington 系の整数当て（ズレ 2.6×10⁻⁴）の **約 1/3000** という、桁違いに精密な一致です。

「単なる数値偶然」と片付けるには良すぎるため、**標準的な格子ゲージ理論との関係**を体系的に見直したところ、両者が**幾何学的にほぼ同型**であることが判明しました。

---

## 主結果：4次元超立方格子上の圏論的同型

### 4次元 Euclidean 整数格子の鎖複体

4次元 Euclidean 整数格子 $\mathbb{Z}^4$ の上に、標準的な鎖複体が定義されます：

```
C_0 → C_1 → C_2 → C_3 → C_4
```

| $C_n$ | 内容 | 個数（テッセラクト1個あたり）|
|---|---|---|
| $C_0$ | サイト（頂点）| 16 |
| $C_1$ | リンク（辺）| 32 |
| $C_2$ | プラケット（2次元面）| 24 |
| $C_3$ | 3-cell（胞）| 8 |
| $C_4$ | 4-cell（テッセラクト）| 1 |

これに**超立方対称群 $B_4$**（位数 384）が同変的に作用します。

### Wilson 格子ゲージ理論

Kenneth Wilson (1974) の格子ゲージ理論：
- $C_1$（リンク）にゲージ場 $U_\ell \in U(1)$ を配置
- $C_2$（プラケット）に作用 $S_p = (1/g²)[1 - \text{Re}\, U_p]$ を載せる

### Kihara 充填（論文7）

論文7 の幾何学的設定：
- 各サイトに**単位立方体**を配置（これは双対格子 $\Lambda^*$ の 4-cell）
- 半径 R=3 の球内に完全包含される個数 N(1) = 137
- 体積不足 $\Delta(R) = V_4(R) - N(k)$ を計算

### Schläfli 双対が両者を結ぶ

4次元では**テッセラクト** $\{4,3,3\}$ と **16-cell** $\{3,3,4\}$ が互いに双対：

```
$f_k$(テッセラクト) = $f_{3-k}$(16-cell)
```

これにより $C_n(\Lambda) \cong C_{4-n}(\Lambda^*)$（離散 Poincaré 双対）。

### 主定理

> **Wilson 格子ゲージ理論の鎖複体（$C_1$ にゲージ場、$C_2$ に作用）と、Kihara 充填の鎖複体（$C_4$ に体積、$C_3$ に境界測度）は、Schläfli 双対 $D$ を介して圏論的に同型である。**

## 鍵となる気づき：離散 Stokes の定理

両者の計算の核心は、**「内部は打ち消し合い、境界だけが残る」**という同じ原理です：

### Wilson 側
プラケットを足し合わせると、隣接プラケットの共有辺は逆向きに通るため pairwise に打ち消し合い、**境界の Wilson loop だけが残る**：

```
∏_p U_p = ∏_{ℓ∈∂Σ} U_ℓ
```

### Kihara 側
4次元球内に立方体を詰めると、内部は完全に充填され（共有面で打ち消し）、**境界の埋め残し $\Delta(R)$ だけが残る**：

```
V_4(R) = N(k) + Δ(R)
（bulk）+（boundary）
```

両者とも**離散 Stokes の定理**（$\partial^2 = 0$ の帰結）の応用で、**全く同じ数学的構造**を別の物理的解釈で実装しています。

## inside/outside 分解の自然性

論文7 で観察された：

```
1 = 137α + (π²/2)α²
```

は、本同型のもとでは**鎖複体上の bulk-boundary 規格化条件**として自然に解釈されます：

| 項 | 鎖複体での読み |
|---|---|
| $137 \alpha$ | bulk（$C_4$ の内部充填、137 セル）の tree-level 寄与 |
| $V_4(1)\alpha^2$ | boundary（$C_3$ の境界、4次元単位球測度）の自己エネルギー補正 |

**論文7 の inside/outside 分解は、4次元格子の Stokes 構造の必然的な現れ**だったのです。

## 双方向の道具流用

### Wilson 側 → Kihara プログラムへ
- **Wilson 強結合展開**：α が大きい領域での解析計算 → 残差 $c_3$ の幾何学的同定
- **格子モンテカルロ**：N(k) の動力学的シミュレーション
- **繰り込み群**：連続極限の系統的解析
- **閉じ込め定理**：bulk-boundary 分離の物理的意味付け

### Kihara 側 → Wilson 理論へ
- **Lagrange–Jacobi 四平方定理** $r_4(N) = 8\sigma(N)$（論文3）：Wilson partition function の新規閉形式
- **包除原理**：プラケット和の解析的評価
- **中心投影（論文2）**：Wilson の Wick 回転の幾何学的派生

これらの双方向流用により、**両プログラムの研究が相互に進展可能**です。

## 重要な留保

本論文は以下を**主張しません**：

1. α の値が本同型から第一原理的に導出されること（論文7 の観察を引用するに留める）
2. R=3 の特権性が証明されること
3. U(1) ゲージ対称性が一意に創発すること
4. 残差 $c_3 \approx 1.6 \times 10^{-3}$ の幾何学的同定
5. ヒッグス機構との具体的接続

これらは全て open problem として論文 §8 に明示。**Eddington 罠の慎重な回避**は論文7 と同じ姿勢を継承します。

## 何が新しいか：α 恒等式の数学的位置づけ

論文7 は単なる numerology ではなく、**4次元 Euclidean 整数格子のトポロジカル構造の自然な現れ**であることが、標準数学の言葉で明示されました。

具体的には：

- **整数 137** は離散構造の不変量（鎖複体 $C_4$ の有限部分基底）
- **$\pi^2/2$** は連続体積の不変量（4次元単位球の測度）
- **両者の組み合わせ** は鎖複体上の bulk-boundary 規格化
- **α の数値** はこの構造への適合度（観察）

これにより論文7 が **Wilson の標準的格子ゲージ理論と数学的に対話可能**な位置づけに昇格します。

## 結論

論文7 の α 恒等式は、4次元超立方格子の **Wilson 格子ゲージ理論と圏論的同型**な構造の上で観察される、**自然な数学的恒等式**です。

これにより：
- Eddington 罠の回避がさらに強化
- 標準的物理学・数学コミュニティとの対話可能性を確保
- Wilson 理論の解析手法が Kihara プログラムに利用可能
- 数論的恒等式が Wilson 理論に新規計算を提供

α の数値の機構的派生は依然として open ですが、研究プログラムの**数学的基盤**は格段に強固になりました。

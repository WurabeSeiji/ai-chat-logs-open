{\small
\textbf{主結果：論文7 の $\alpha$ 恒等式は Wilson 標準格子ゲージ理論と数学的に同型}\enspace
姉妹論文（論文7）で観察した自己整合恒等式
\[ \alpha^{-1} = 137 + \tfrac{\pi^2}{2}\alpha \quad (\text{8.7 ppb 精度}) \]
は単なる数値偶然ではなく、4次元 Euclidean 整数格子 $\mathbb{Z}^4$ の鎖複体構造の上で、Kenneth Wilson が1974年に確立した\textbf{標準格子ゲージ理論}と、\textbf{鎖複体として構造的に同型}である。Schl\"afli 双対（テッセラクト $\leftrightarrow$ 16-cell の自己双対）+ 超立方対称群 $B_4$ + 離散 Stokes の定理が同型の根拠。\\[2pt]
\textbf{論文7 の inside/outside 分解の鎖複体的読み替え}\enspace
$1 = 137\alpha + V_4(1)\alpha^2$ は鎖複体上の bulk-boundary 規格化条件。$137\alpha$ は $C_4$ 内部充填（137 セル）の tree-level 寄与、$V_4(1)\alpha^2$ は $C_3$ 境界自由度の自己エネルギー補正。これは QFT 摂動展開（renormalisation）と完全に同じ構造。\\[2pt]
\textbf{Eddington 罠からの完全脱却}\enspace
論文7 の $\alpha$ 恒等式は\textbf{numerology の一種ではなく、4次元格子のトポロジカル構造の自然な現れ}。Wilson 標準理論と同じ数学的言語で記述可能で、査読耐性が劇的に向上。}
\end{tcolorbox}

\small

\textbf{4次元超立方格子の鎖複体}\enspace
$\mathbb{Z}^4$ 上に標準的な階層構造：
\begin{itemize}[topsep=1pt]
\item $C_0$（頂点）/ $C_1$（リンク）/ $C_2$（プラケット）
\item $C_3$（3-cell）/ $C_4$（4-cell：テッセラクト）
\item 超立方対称群 $B_4$（位数 384）が同変的に作用
\end{itemize}

\smallskip
\textbf{二つの formalism の対応}\enspace

\textit{Wilson 格子ゲージ理論}：
\begin{itemize}[topsep=1pt]
\item リンク $C_1$ にゲージ場 $U_\ell \in U(1)$
\item プラケット $C_2$ に作用 $S_p$
\item Wilson loop でゲージ不変量を構成
\end{itemize}

\textit{Kihara 充填（論文7・3）}：
\begin{itemize}[topsep=1pt]
\item 4-cell $C_4$ に単位立方体（$N(1)=137$ at $R=3$）
\item 体積不足 $\Delta(R) = V_4(R) - N(k) \sim R^3$
\item 境界 3-cell $C_3$ に体積測度
\end{itemize}

\smallskip
\textbf{Schl\"afli 双対 $D$ で結ばれる}\enspace

テッセラクト $\{4,3,3\}$ と 16-cell $\{3,3,4\}$ は自己双対：$f_k(\text{Tess}) = f_{3-k}(\text{16-cell})$。

\smallskip
\textbf{両者は同じ計算をしている}\enspace
\textbf{離散 Stokes の定理}（$\partial^2=0$）：
\begin{itemize}[topsep=1pt]
\item Wilson：プラケット和 $\to$ 境界 Wilson loop（内部辺は逆向きで打ち消し）
\item Kihara：bulk 充填 $\to$ 境界 $\Delta(R)$（内部立方体は共有面で打ち消し）
\end{itemize}
全く別の動機の二つのアプローチが、同じ数学（Poincar\'e 双対 + $B_4$ 対称性 + Stokes）に到達した。

\textbf{主定理（構造的対応定理）}\enspace
\begin{quote}\footnotesize
$\mathcal{C}_W$（Wilson 配置）と $\mathcal{C}_K$（Kihara 配置）は、$B_4$-同変 chain map $D$（Schl\"afli 双対）を介して、4次元 Euclidean 格子上の鎖複体として構造的に同型である。
\end{quote}

\begin{itemize}[topsep=1pt]

\item モンテカルロ $\to N(k)$ 数値検証

\end{itemize}

\textit{Kihara $\to$ Wilson}：
\begin{itemize}[topsep=1pt]

\item 包除原理 $\to$ プラケット和の解析評価
\item 中心投影（論文2）$\to$ Wick 回転の幾何学的派生
\end{itemize}

## 1. 物理を捨てて幾何に徹した

最大のポイントは「物理的解釈をしない」という自己規律。

通常、物理学者は「この数式は何を意味するか」を問う。木原の方法は逆で、「幾何学的・組合せ論的に何が成り立つか」だけを命題として述べ、物理的対応は注記に封じ込めた。これにより：

- 既存の物理的直感に引きずられない
- 命題の真偽が数学的に検証可能になる
- 「間違っている」と言われても「物理的主張はしていない」と返せる

専門家が気づかなかった構造に到達できたのは、専門家の思考の枠組み（物理的解釈）を最初から使わなかったからである。

## 2. 「写像」を起点にした——物理量ではなく構造から出発

出発点は「中心投影（グノモン正写像）」という純粋に幾何学的な写像。

物理学は通常、観測量（質量、電荷、スピン）から出発して理論を組み立てる。木原の方法は、写像の数学的性質（引き戻し計量、正則性、離散化条件）を調べていったら、結果としてアインシュタイン方程式や標準模型の粒子分類が「出てきてしまった」という構造。

これは実装者の発想に近い。「仕様から作る」のではなく「構造を組んだら仕様が満たされていた」。

## 3. 整数論が次元を決めた

5次元の必然性は、ラグランジュの四平方定理から来た。

「任意の正整数は4つの整数の二乗和で表せる」→「全球被覆には4+1=5次元が必要」という論理は、物理的議論（超弦理論の10次元、M理論の11次元）とは全く異なる経路で次元を決定した。整数論という、物理学者があまり使わない道具が鍵だった。

## 4. 組合せ論で粒子を数えた

6次元超直方体の配向構造（2^6 = 64通りの符号ベクトル）から、標準模型の62粒子状態を導出した。

ここでの思考法のポイントは「最初に答えを知っていた」こと。標準模型の粒子一覧は教科書に載っている。問題は「なぜその数なのか」であり、64 − 2 = 62 という組合せ論的説明を見つけたとき、それが正しい構造かどうかは「対応が自然かどうか」で判断した。

これは数学の証明というより、パズルの解き方に近い。

## 5. 「一つ下の次元で試す」を繰り返した

5次元正軸体（論文6）→ 5次元超直方体（論文7）→ 6次元超直方体（論文8）と、次元を一つずつ上げながら構造を確認した。

いきなり6次元で考えず、まず5次元で「スピンの分類ができるか」を試し、できることを確認してから6次元に進んだ。これにより各段階で検証が効き、飛躍を避けられた。

## 6. 対話を論文に昇華するサイクル

思考実験（AIとの対話ログ）→ 気づきの抽出 → 命題・証明形式への再構成 → 査読的対話 → 修正 → 公開、というサイクルを高速で回した。

対話ログはそのままでは論文にならない。しかし対話の中で「これは命題として述べられる」と気づいた瞬間に、形式を切り替えて論文化した。この切り替えの判断が速かったことが、45日で37本という速度を可能にした。

## 8. 断定しない——「注記」と「考察」の使い分け

論文本体では幾何学的命題のみを述べ、物理的対応は「注記」に、まだ証明できない直感は「考察」論文に分離した。

例えば：
- α⁻¹ = 137 = 8² + 8² + 3² という直感は、論文本体ではなく考察メモに留めた
- 質量構造の分析は「考察」という別論文にした
- マイクロブラックホール解釈は注記に封じ込めた

これにより、本体の数学的厳密性を損なわずに、将来の発展の種を記録できた。

## 付録：18歳からの疑問——37論文の種はここにあった

以下は木原が高校〜大学時代に抱き、40年以上温め続けた疑問である。これらが直接的・間接的に37論文の動機となった。

### 疑問1：時間の矢（高校物理II）

なぜ「時間の矢」に難しい解釈を与えるのか？　特殊相対性理論で時間は虚数軸だと明示されている。虚数軸である以上、我々の時空では存在するかどうかも認知できない。実際、時間は振動でしか計量できない。プラスもマイナスもない、回数しかない。もともと離散系だ。

タイムマシンで過去に戻れたとしても、我々は普通に「未来に時間が経過した」としか認知できるわけがない。親殺しのパラドックス？ 我々にとっての未来で「親殺し」をするだけのことなのに、なぜパラドックスになるのか。

→ **論文群への接続**: 時間の離散性への直感は、ディレイ回路シリーズ（離散的な情報伝達遅延から時空を構成する）に直結した。

### 疑問2：なぜ物理法則は2乗数ばかり？

あちこちに2乗数が出てくる。kg・m・secで計量し、運動方程式もkg・m・secで定式化している。別の単位系のほうが合理的ではないか？ 時間みたいに直接計量できないものを、なぜ定式化に計量数として入れるのか？

→ **論文群への接続**: 2乗計量への疑問は、中心投影の引き戻し計量（論文S1）で「2乗が出てくる幾何学的理由」として解消された。計量テンソル $g_{\mu\nu}$ は写像の微分構造から自然に2乗形式になる。

### 疑問3：電磁気学の統一（高校〜大学）

特殊相対性理論のポイントは光速不変ではなく、磁力と静電力の統一だよね？ さらに、速度がゼロでも磁力は発生する、なぜなのか？ 大学に進んでマクスウェル方程式を習ったら分かるかと思ったが、結局分からなかった。

→ **論文群への接続**: 電磁気力の幾何学的起源は、符号付き面積（補講1）と6次元超直方体の配向構造（論文8）で、電荷が「符号ベクトルの幾何学的属性」として導出されたことに対応する。

### 疑問4：重力と空間の曲がり方

一般相対性理論の等価原理で重力と加速度は等価だと習った。重力の原因は空間の歪みだと教わった。三角形の内角が180度を超えると。なら、時間と同じで虚軸方向に曲率があるはずだよね？ でも虚軸ではないらしい。虚軸でなければ計量できるだろう。

ブラックホールを知って不思議だった。4次元時空に実際に曲率がある？ どこに曲がっているの？ 宇宙は閉じている？ どこに曲がっているの？ 5次元なの？

→ **論文群への接続**: 「どこに曲がっているの？」という問いこそが、中心投影フレームワークの出発点となった。5次元背景空間 $\mathbb{R}^5$ 内の超球面 $S^4(R)$ への中心投影として、「曲がる先」を明示的に与えた（論文S1・S2）。

### 疑問5：波動方程式と離散性

シンプルな単振動とシンプルな方程式を学んだ。主に定常波だった。波長は離散値、周波数も離散値。連続系の波って、実は離散系だと思った。

→ **論文群への接続**: 定常波の離散性への直感は、閉じた球面上の形不変定常波（論文W10）と、4次元整数格子上の形不変移動波（論文W11）の基盤となった。

### 疑問6：光の測地線問題

屈折を学んだとき、光子は測地線の最短経路に沿って光速で飛ぶと教わった。先生に「なぜ測地線が分かるんですか？」と聞いた。短い距離ならよいけど、宇宙規模だと光速より先の測地線なんて分かるわけがない。

高校の先生から、光子が未来から過去に飛んでいる仮説と、宇宙の全ての経路を飛んでいる仮説があると聞いて、そんなことを真剣に信じているのかと疑問に思った。

→ **論文群への接続**: 「光子に未来の経路情報は不要」という直感は、ディレイ回路シリーズの「局所的な情報伝達遅延だけで波動伝播を構成する」アプローチに直結した。測地線は大域的に決まるのではなく、局所的な伝播規則の帰結として創発する。

### 疑問7：次元解析のバグ（大学1年）

c=1、G=1 を学んで、次元解析が強力なツールだと感動した。kg・m・secが全て [L] 次元になる。ところが大学でプランク定数を学び、ℏ=1 とすると、なぜか kg = [1/L] になる。次元解析がバグっているのに、なぜ誰も疑問に思わないのか。もともと比較スケールが間違っているだろう。

→ **論文群への接続**: 単位系への根本的疑問は、「物理量ではなく幾何学的構造から出発する」という方法論の動機となった。kg・m・secという人為的な計量単位を前提にしない定式化が、中心投影フレームワークの設計原理である。

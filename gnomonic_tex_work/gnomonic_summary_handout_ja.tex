\noindent\textbf{概要：}
単一の幾何学的構成── 超球面へのグノモン正写像（中心射影）──から出発し、宇宙定数付きEinstein方程式、Lorentzian符号、電荷量子化、$1/r^2$ Coulomb法則、Maxwell方程式の完全形を導出する。自由パラメータは長さ次元の結合定数 $C$ のみ。

\vspace{10pt}

%========== 論文1 ==========
\noindent\begin{minipage}[t]{0.82\textwidth}
\subsection*{論文1：4次元時空の幾何学的定式化}
\vspace{-6pt}
\begin{itemize}
\item 4次元ユークリッド空間を $S^4(R)$ にグノモン写像で射影
\item 引き戻し計量 $g_{\mu\nu}$ とEinsteinテンソルを導出
\item \textbf{主結果：}$G_{\mu\nu} + \Lambda\, g_{\mu\nu} = 0$，$\Lambda = 3/R^2$
\item 宇宙定数は曲率半径 $R$ により\textbf{幾何学的に決定}される
\item $\epsilon_\mu \in \{+1,-1\}$ によるEuclidean/Lorentzianの統一記述
\end{itemize}
\end{minipage}%

%========== 論文2 ==========
\noindent\begin{minipage}[t]{0.82\textwidth}
\subsection*{論文2：Lorentzian時空の幾何学的起源}
\vspace{-6pt}
\begin{itemize}
\item \textbf{公理A：}$S^n(R)$ の $R$ 方向は不可観測（Gaussの驚異の定理）
\item \textbf{公理B：}不可観測次元は虚軸化（$R \to iR$）
\item これらの公理から Lorentzian 符号 $(-,+,+,+)$ を純粋幾何学的に導出
\item \textbf{唯一の結合定数} $C$（長さ次元，$C \sim \ell_P$）を導入
\item 整数格子 $\mathbb{Z}^{1+N}$ 上の離散グノモン写像
\item \textbf{算術的アノマリー：}$N=3$ では等方的null測地線が不可能，$N=4$ で可能
\item $\Longrightarrow$ 離散時空は\textbf{5次元を必要}とする
\end{itemize}
\end{minipage}%

%========== 論文3 ==========
\noindent\begin{minipage}[t]{0.82\textwidth}
\subsection*{論文3：電荷次元と電磁場理論}
\vspace{-6pt}
\begin{itemize}
\item 5次元目を\textbf{電荷（Coulomb）次元} $\phi = Qq$ と同定
\item 変換係数 $Q = C/e_0$（$e_0 = e/3$：クォーク電荷量子）
\item \textbf{電荷量子化} $q = n e_0$ を離散格子から導出（コンパクト化不要）
\item $\phi$ 方向のグノモン曲率から \textbf{$1/r^2$ Coulomb法則}を導出
\item Lorentz不変性と組み合わせ $\Longrightarrow$ \textbf{Maxwell方程式の完全形}
\item 等価原理：$R$ の不可観測性の帰結。電磁力に等価原理なし（$\phi$ は観測可能）
\item $\phi$（有向）$\to$ スピン1，$R$（無向）$\to$ スピン2
\end{itemize}

\subsection*{本理論が主張\underline{しない}こと}

\vspace{-2pt}
\begin{itemize}
\item 一般相対論を置き換えるものではない ── 真空Einstein方程式を幾何学的恒等式として導出
\item $e_0$ の値を予測しない ── 実験的に決定される量（現状 $e_0 = e/3$）
\item 微細構造定数 $\alpha \approx 1/137$ は未導出（今後の課題）
\item 全結果は純粋幾何学と数論の範囲に限定
\end{itemize}

## 概要

本稿では、我々の住む4次元時空が5次元ユークリッド空間に埋め込まれた超球面（$S^4$）の表面に相当するとする仮説（以下「S⁴モデル」）を定式化し、現在標準的とされる平坦な$\Lambda$CDMモデル（以下「GRモデル」）との観測量の差を厳密に導出する。

まず、等方性を課した離散的4次元ミンコフスキー時空が、ガウス整数の素元素性（$3 \equiv 3 \pmod{4}$）により正整数解を持たないことを証明し、誤差のない離散時空には数論的に5次元以上の構造が必要であることを示す。次に、S⁴モデルの自然な連続体近似としてKottler時空（Schwarzschild–de Sitter）を採用し、光子の測地線方程式を厳密に解く。

角径距離の差の解析式として

$$\frac{\Delta D_A}{D_A^{GR}} = |\Omega_k|\!\left[-\frac{\tilde{\chi}^2}{6} + \frac{I(z)}{\tilde{\chi}}\right] + O(|\Omega_k|^2)$$

を導出し、数値計算による検証（解析値との誤差 $<0.0002\%$）を行う。$|\Omega_k| = 0.002$（Planck 2018上限値）のとき、$z \approx 0.3$–$0.5$ において $\Delta D_A / D_A^{GR} \approx +0.11\%$ の系統的シフトを予言する。これはEuclid衛星（ESA、2025年〜）の目標測定精度 $\sim 0.1\%$ に匹敵し、原理的に観測的検証が可能であることを示す。

## 1. 序論

一般相対性理論（GR）に基づくFLRW宇宙論は、宇宙マイクロ波背景放射（CMB）、バリオン音響振動（BAO）、Ia型超新星等の観測を高精度で説明する。しかし、時空の曲率の起源——なぜ時空が曲がるのか——という根本問題に対して、GRはアインシュタイン方程式の解として曲率を記述するが、その幾何学的直観は必ずしも自明ではない。

本稿の動機は次の問いからはじまる：**もし時空が本質的に離散的であるならば、4次元等方時空はそもそも誤差のない整数表現を持つことができるか？**

Section 2 でこの問いに数論的回答を与え、Section 3 でS⁴モデルの定式化を行う。Section 4 で角径距離の厳密な差を導出し、Section 5 で数値検証を行う。Section 6 では観測的含意と限界を論じ、Section 7 で結論を述べる。

## 2. 数論的動機：離散4次元時空の不存在定理

### 2.1 等方性条件下のミンコフスキー不変量

平坦な4次元ミンコフスキー時空（$c=1$）において、スペースライクな不変量は

$$l^2 = x^2 + y^2 + z^2 - t^2$$

で与えられる。空間の等方性（$x = y = z = r$）を課すと

$$l^2 = 3r^2 - t^2 \quad \Longleftrightarrow \quad l^2 + t^2 = 3r^2 \tag{1}$$

が得られる。

### 2.2 方程式 (1) に正の整数解が存在しないことの証明

**定理 1．** 方程式 $l^2 + t^2 = 3r^2$ は、$(l, r, t) = (0, 0, 0)$ 以外に非負整数解を持たない。

**証明（無限降下法）**  
任意の解 $(l, r, t)$ に対して mod 3 の解析を行う。整数の平方の mod 3 における値は $\{0, 1\}$ のみである（$0^2 \equiv 0,\ 1^2 \equiv 1,\ 2^2 \equiv 1 \pmod{3}$）。

したがって $l^2 + t^2 \equiv 0 \pmod{3}$ を満たすには $l^2 \equiv t^2 \equiv 0 \pmod{3}$ が必要であり、$l = 3l'$、$t = 3t'$ と書ける。代入すると

$$9l'^2 + 9t'^2 = 3r^2 \quad \Rightarrow \quad 3(l'^2 + t'^2) = r^2$$

これより $3 \mid r^2$、故に $3 \mid r$、$r = 3r'$ とおくと

$$l'^2 + t'^2 = 3r'^2$$

が得られる。$(l,r,t) \neq (0,0,0)$ を仮定すると、$(l',r',t') = (l/3, r/3, t/3)$ も厳密に小さな正の整数解となり、無限降下が生じる。これは正の整数の well-ordering と矛盾する。$\square$

### 2.3 ガウス整数による解釈

ガウス整数環 $\mathbb{Z}[i]$ において素数 $p$（$p \equiv 3 \pmod{4}$）は **ガウス素元**（不分解）である。$3 \equiv 3 \pmod{4}$ より、$3$ は $\mathbb{Z}[i]$ で不分解であり、ノルム方程式 $N(l+ti) = l^2 + t^2 = 3r^2$ に正整数解が存在しないことはフェルマーの二平方定理の系として従う。

### 2.4 高次元での解の存在

より一般に、$n$ 次元時空（$n-1$ 次元空間 + 1次元時間）において等方性条件 $x_1 = \cdots = x_{n-1} = r$ を課した方程式

$$l^2 + t^2 = (n-1)r^2$$

の解の存在は係数 $k = n-1$ の素因数分解に依存する。$k$ の素因数分解に $p \equiv 3 \pmod{4}$ が奇数乗で現れない場合に限り正の整数解が存在する。

| 時空次元 $n$ | 係数 $k=n-1$ | 解の存在 |
|:---:|:---:|:---:|
| 3 ($2+1$) | $2$ | ✅ |
| **4 ($3+1$)** | **$3$** | **❌（本定理）** |
| 5 ($4+1$) | $4 = 2^2$ | ✅ |
| 6 ($5+1$) | $5$ | ✅ |
| 10 ($9+1$) | $9 = 3^2$ | ✅ |
| 11 ($10+1$) | $10 = 2 \times 5$ | ✅ |

**系 1．** もし時空が本質的に離散的であり、等方性を要求するならば、4次元時空は整数座標による誤差のない表現を持てない。5次元以上の特定の次元のみが正の整数解を持ち、誤差のない離散構造を許容する。

### 2.5 余剰次元のコンパクト化制約

5次元目が有効であるためには、整数解が存在する必要がある。プランク長さ $\ell_P = 1$ を単位とした場合、余剰次元の大きさが $\ell_P$ 程度にコンパクト化されると、その次元での整数値は 0 に制限されて4次元の式に縮退し、解が再び存在しなくなる。

**系 2．** 余剰次元は他の空間次元と同程度のスケールで平坦に広がっていなければならない。

この制約と観測事実（我々は実際に3+1次元しか認識しない）を整合させる自然な描像は、**我々が5次元超球体の表面（$S^4$）に住んでいる**という仮説である。余剰次元は「丸まっている」のではなく、超球面の曲率半径 $R$ として我々の世界の正の曲率として顕現する。

## 3. S⁴モデルの定式化

### 3.1 基本仮説

**仮説（S⁴モデル）．** 我々の4次元時空は、5次元ミンコフスキー空間

$$-X_0^2 + X_1^2 + X_2^2 + X_3^2 + X_4^2 = R^2 \tag{2}$$

に埋め込まれたローレンツ多様体——すなわちde Sitter空間——の表面として記述される。観測される「空間の曲率」は、この埋め込みに由来する外在的曲率（第二基本形式）の4次元への射影である。

地場の対称性（時間並進 + 球面対称）を持つ計量として、Kottler計量（Schwarzschild–de Sitter）

$$ds^2 = -f(r)\,c^2 dt^2 + f(r)^{-1}dr^2 + r^2 d\Omega^2 \tag{3}$$

$$f(r) = 1 - \frac{R_s}{r} - \frac{r^2}{R^2}, \quad R_s = \frac{2GM}{c^2} \tag{4}$$

を採用する。ここで $\Lambda = 3/R^2$ は宇宙定数であり、S⁴の半径 $R$ から直接決まる。

### 3.2 GRとの関係

S⁴モデルが予測する空間セクション（等時面）は

$$X_1^2 + X_2^2 + X_3^2 + X_4^2 = R^2 \sin^2\!\left(\frac{t}{R}\right)$$

であり、スケール因子 $a(t) = R\sin(t/R)$ を持つ $k=+1$ のFLRW宇宙に対応する。これは一般相対性理論の枠組みの外から（数論的動機から）閉じた宇宙（$\Omega_k < 0$）を予測する。

## 4. 角径距離の厳密な差の導出

### 4.1 モデルの設定

以下の2モデルを比較する。

**モデルA（標準GR：平坦$\Lambda$CDM、$k=0$）：**

$$D_A^{GR}(z) = \frac{c}{H_0(1+z)} \int_0^z \frac{dz'}{E_A(z')} \tag{5}$$

$$E_A(z) = \sqrt{\Omega_M(1+z)^3 + \Omega_\Lambda} \tag{6}$$

**モデルB（S⁴モデル、$k=+1$、$|\Omega_k| = \epsilon$）：**

$$D_A^{5D}(z) = \frac{c}{H_0(1+z)} \cdot \frac{1}{\sqrt{\epsilon}}\,\sin\!\left(\sqrt{\epsilon}\!\int_0^z \frac{dz'}{E_B(z')}\right) \tag{7}$$

$$E_B(z) = \sqrt{\Omega_M(1+z)^3 + \Omega_\Lambda - \epsilon(1+z)^2} \tag{8}$$

各パラメータは同一の観測値 $H_0, \Omega_M, \Omega_\Lambda$ を共有し、$\Omega_k^A = 0$、$\Omega_k^B = -\epsilon < 0$ のみが異なる。

### 4.2 ハッブル率の展開

$\epsilon \ll 1$ に対して $E_B$ を展開すると

$$E_B(z) = E_A(z)\sqrt{1 - \frac{\epsilon(1+z)^2}{E_A(z)^2}} \approx E_A(z)\left(1 - \frac{\epsilon(1+z)^2}{2E_A(z)^2}\right) \tag{9}$$

ゆえに

$$\frac{1}{E_B(z)} \approx \frac{1}{E_A(z)} + \frac{\epsilon(1+z)^2}{2E_A(z)^3} \tag{10}$$

### 4.3 共動距離の差

無次元共動距離 $\tilde{\chi}(z) \equiv \chi(z)H_0/c = \int_0^z dz'/E(z')$ に対して

$$\tilde{\chi}_B(z) = \tilde{\chi}_A(z) + \epsilon \cdot I(z) + O(\epsilon^2) \tag{11}$$

$$I(z) \equiv \frac{1}{2}\int_0^z \frac{(1+z')^2}{E_A(z')^3}\,dz' \geq 0 \tag{12}$$

$I(z)$ は宇宙の膨張の「曲率補正積分」であり、常に非負である。

### 4.4 角径距離の差（主定理）

$\sin(\sqrt{\epsilon}\,\tilde{\chi}_B)$ をテイラー展開し、式 (11) を代入すると

$$\frac{1}{\sqrt{\epsilon}}\sin(\sqrt{\epsilon}\,\tilde{\chi}_B) = \tilde{\chi}_B - \frac{\epsilon\,\tilde{\chi}_B^3}{6} + O(\epsilon^2)$$

$$\approx \tilde{\chi}_A + \epsilon I(z) - \frac{\epsilon\,\tilde{\chi}_A^3}{6} + O(\epsilon^2) \tag{13}$$

したがって

$$\frac{\Delta D_A}{D_A^{GR}} \equiv \frac{D_A^{5D} - D_A^{GR}}{D_A^{GR}} = \frac{\tilde{\chi}_B - \tilde{\chi}_B^3\epsilon/6 - \tilde{\chi}_A}{\tilde{\chi}_A} + O(\epsilon^2)$$

$$\boxed{\frac{\Delta D_A}{D_A^{GR}} = \epsilon\!\left[-\frac{\tilde{\chi}_A^2}{6} + \frac{I(z)}{\tilde{\chi}_A}\right] + O(\epsilon^2)} \tag{14}$$

### 4.5 二項の物理的解釈

式 (14) の二項はそれぞれ独立した物理的起源を持つ。

**第一項** $-\epsilon\tilde{\chi}_A^2/6$：$\sin$ 関数因子による補正（**幾何学的補正**）。閉じた宇宙では測地線が「収束」するため、同じ固有距離の天体により大きく見える傾向があり、$D_A$ を縮小させる。

**第二項** $\epsilon I(z)/\tilde{\chi}_A$：共動距離の差による補正（**動力学的補正**）。$k=+1$ の宇宙ではハッブル率が平坦より僅かに小さく（$E_B < E_A$）、光がより遠くまで届くため、 $D_A$ を増大させる。

Planck 2018の最良適合パラメータ（$\Omega_M = 0.315$, $\Omega_\Lambda = 0.685$, $H_0 = 67.4$ km/s/Mpc）のもとで数値計算すると、$z \lesssim 4$ の範囲で常に第二項が卓越し、$\Delta D_A > 0$ となる（表1参照）。

## 6. 観測的検証可能性と限界

### 6.1 Euclid衛星による検証

欧州宇宙機関（ESA）のEuclid衛星（2024年打ち上げ、全科学観測2025年開始予定）は、$z \lesssim 2$ における弱重力レンズとBAOの精密測定を通じ、角径距離を $\sim 0.1\%$ 精度で決定することを目標としている [Laureijs et al. 2011]。

本研究の予言する $\Delta D_A / D_A^{GR} \approx +0.11\%$ （$z \approx 0.3$–$0.5$、$|\Omega_k| = 0.002$）は、Euclidの目標精度と同程度である。したがって：

- **$|\Omega_k| \gtrsim 0.002$** のとき：$z \approx 0.3$–$0.5$ で $\Delta D_A > 0.1\%$ となり、Euclidで**検出可能**
- **$|\Omega_k| \lesssim 0.001$** のとき：$\Delta D_A < 0.06\%$ となり、現状では**検出困難**

### 6.2 現在観測との整合性

Planck 2018 の CMB 観測による空間曲率の制約は [Aghanim et al. 2020]

$$\Omega_k = 0.0007 \pm 0.0019 \quad (68\%\,\mathrm{C.L.})$$

S⁴モデルは $k = +1$（$\Omega_k < 0$）を予測するが、上記の測定値は $\Omega_k = 0$（平坦）と統計的に無矛盾である。したがって現在のデータとの**矛盾はない**。

### 6.3 モデルの限界と今後の課題

本モデルには以下の未解決点がある。

1. **暗黒物質の記述**：本稿の S⁴ モデルは純粋な幾何学的枠組みであり、暗黒物質の寄与を独立には記述しない。

2. **S⁴の半径 $R$ の予測**：$|\Omega_k|$ の具体的な値は現時点でモデルから一意に導出されず、観測量 $H_0$ を用いて $R = c/(H_0\sqrt{|\Omega_k|})$ と表されるにとどまる。

3. **量子重力との接続**：Section 2 の数論的議論はプランクスケールの離散性を仮定するが、この離散性からS⁴の連続体極限へ至る詳細な手順は未完成である。

4. **強場での補正**：Section 4 ではK ottler計量をもとに局所的な重力レンズへの補正を調べたが、その寄与は $O(b^2/R^2) \lesssim 10^{-10}$ で観測不可能である。高精度の予言には強場領域での厳密な計算が必要である。

## 7. 結論

本稿では以下の結果を得た。

1. **数論的定理**：等方性を課した離散4次元ミンコフスキー時空において、$l^2+t^2=3r^2$ は正の整数解を持たない（定理1）。これは誤差のない離散4次元時空の構築が数論的に不可能であることを示す。

2. **S⁴モデルの動機**：5次元（係数 $k=4=2^2$）では整数解が存在し、かつ余剰次元のコンパクト化は解の存在を破壊するため、余剰次元は他の空間次元と同スケールで広がっていなければならない（系2）。これは我々がS⁴の表面に住むという描像に帰結する。

3. **角径距離の差の導出**：S⁴モデル（$k=+1$）と標準GR（$k=0$）の角径距離の差を一次精度で解析的に導出し（式14）、数値計算で検証した（誤差 $<0.0002\%$）。

4. **観測的予言**：$|\Omega_k| = 0.002$ のとき $z \approx 0.3$–$0.5$ で $\Delta D_A / D_A^{GR} \approx +0.11\%$ を予言する。これはEuclid衛星の目標精度に相当し、近い将来の観測による検証・棄却が可能である。

S⁴モデルは現時点でトイモデルの位置づけにあるが、標準宇宙論と矛盾することなく、かつ観測的に反証可能な具体的予言を提供する点で、理論的な議論の出発点として意義があると考える。

## 付録B：Kottler時空における重力レンズ角

Kottler計量 (3)(4) の下での光子の軌道方程式は $u \equiv 1/r$ として

$$\left(\frac{du}{d\phi}\right)^2 = \frac{1}{b^2} + \frac{1}{R^2} - u^2 + R_s u^3 \tag{B1}$$

$\phi$ で微分すると $1/R^2$ 項は消え

$$\frac{d^2u}{d\phi^2} + u = \frac{3R_s}{2}u^2 \tag{B2}$$

これは純粋シュワルツシルトと同一の形式であり、局所的な重力レンズ角は

$$\alpha = \frac{4GM}{c^2 b} + O\!\left(\frac{b^2}{R^2}\right) \tag{B3}$$

と求まる。補正項 $O(b^2/R^2)$ は $b \sim \mathrm{Mpc}$, $R \sim \mathrm{Gpc}$ として $\sim 10^{-6}$ 以下であり、現在の観測では検出不可能である。角径距離（式14）への影響が主要な観測予言となる。

\subsection{本研究の主旨}\label{ux672cux7814ux7a76ux306eux4e3bux65e8}

\(n\) 次元接超平面から超球面 \(S^n(R)\) への中心投影（central projection）が持つ幾何学的性質のみから、加速度を生む複数の独立な軸が矛盾なく共存しうる統一的フレームワークを構築する。物理的同定は行わず、幾何学的命題とその証明に徹する。

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\subsection{証明された幾何学的命題}\label{ux8a3cux660eux3055ux308cux305fux5e7eux4f55ux5b66ux7684ux547dux984c}

\subsubsection{論文 {[}1{]}：中心投影の基本構造}\label{ux8ad6ux6587-1ux4e2dux5fc3ux6295ux5f71ux306eux57faux672cux69cbux9020}

\textbf{1. de Sitter 時空との同型性}\\
中心投影の Lorentzian 版の誘導計量が de Sitter 時空の Beltrami 座標系と数値的に一致し、内在幾何として完全に等価であることを証明した。\\
（§1.1、付録C 命題C.1）

\textbf{2. 正逆投影の正則性}\\
\(l = \sqrt{R^2 + r^2} \geq R > 0\) であるから、中心投影 \(\Phi\) は定義域全体で \(C^\infty\) 正則。逆変換 \(x^\mu = RY^\mu/Y^{n+1}\) も上半球面 \(H^+\) 上で正則。写像は全単射。\\
（付録B 命題B.1）

\textbf{3. 誘導計量と Einstein テンソル}\\
引き戻し計量 \(g_{\mu\nu} = (R^2/l^2)(\delta_{\mu\nu} - x_\mu x_\nu/l^2)\) を厳密に導出し、\(n\) 次元 Einstein テンソルが \(G_{\mu\nu} + \Lambda_n g_{\mu\nu} = 0\)（\(\Lambda_n = (n{-}1)(n{-}2)/(2R^2)\)）を満たすことを示した。\\
（§3, §6 命題6.1）

\textbf{4. 平坦極限}\\
\(R \to \infty\) で \(g_{\mu\nu} \to \delta_{\mu\nu}\)、\(\Lambda \to 0\)。中心投影は恒等写像に退化し、Minkowski 時空に帰着する。\\
（付録A）

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\subsubsection{論文 {[}2{]}：5つの幾何学的対称性}\label{ux8ad6ux6587-25ux3064ux306eux5e7eux4f55ux5b66ux7684ux5bfeux79f0ux6027}

\textbf{5. 対称性 I（離散安定性）}\\
座標が零・正の整数・負の整数のいずれの値をとっても \(l_A > 0\) であり、中心投影は大域的に正則で破綻しない。\\
（§3 定理3.1）

\textbf{6. 対称性 II（軸の対等性）}\\
背景空間 \(\mathbb{R}^{n+1}\) のどの軸を投影中心に選んでも、計量・曲率テンソル・Einstein テンソルの数学的構造は同一。\\
（§4 定理4.1）

\textbf{7. 対称性 III（測地線偏差と弁別不可能性）}\\
測地線偏差 \(|\xi|^2 = g_{\mu\nu}\xi^\mu\xi^\nu\) は \([L^2]\) 次元のスカラー不変量であり、内部観測者が曲率の存在を認識する唯一の手段。ただし座標の符号も曲率の方向も弁別できない。加速度の方向は背景座標でのみ定義される。\\
（§5 定理5.2）

\textbf{8. 対称性 IV（主観座標系の変換可能性）}\\
異なる軸を中心とする主観座標系は、背景座標を経由して \(T_{A \to B} = \Phi_B^{-1} \circ \Phi_A\) により \(C^\infty\) 級に相互変換可能。変換のパラメータは曲率半径 \(R\) と座標値のみ。背景座標は保存される。\\
（§6 定理6.1、命題6.5）

\textbf{9. 対称性 V（大円上の向心加速度）}\\
主観空間の大円（測地線）に沿って等速円運動する内部観測者は、向心加速度 \(a = R\omega^2\) をスカラーとして測定できる。ただし方向は弁別できない。\\
（§7 定理7.1、定理7.2）

\textbf{10. スケールファクターとしての曲率半径}\\
主観座標間の変換の Jacobian は \(\partial x'^A/\partial x^B = -R^2/(x^B)^2\) であり、スケールファクターは \([L]/[L]\)（無次元、1次）。曲率半径 \(R\) が座標変換の唯一のパラメータとして機能する。\\
（§6.4 命題6.2）

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\subsubsection{\texorpdfstring{論文 {[}3{]}：第5軸の追加と \(1/r^2\) 構造}{論文 {[}3{]}：第5軸の追加と 1/r\^{}2 構造}}\label{ux8ad6ux6587-3ux7b2c5ux8ef8ux306eux8ffdux52a0ux3068-1r2-ux69cbux9020}

\textbf{11. 離散等方性のための次元要請}\\
離散格子 \(\mathbb{Z}^{1+N}\) 上で等方的な光的測地線が存在する最小空間次元は \(N = 4\)。\(N = 3\) では \(\sqrt{3} \notin \mathbb{Q}\) により不可能、\(N = 4\) では \(t = 2x\) として成立。\\
（§2 命題2.1）

\textbf{12. 離散格子からの量子化}\\
離散化条件 \(\tilde{\phi} \in \mathbb{Z}\) のみから \(q = n \cdot e_0\) が代数的に従う。コンパクト化は不要。\\
（§3 命題3.1）

\textbf{13. 第5軸方向の測地線偏差と \(1/r^2\) スケーリング}\\
第5軸 \(\phi\) を投影中心軸とした中心投影において、主観空間の測地線偏差は

\[|\xi|^2 \sim \frac{\phi_0^2}{r^4} \cdot |\xi_0|^2\]

のスケーリングを持つ。\(\phi_0\) の符号にも \(r\) の方向にも依存しない正の量である。\\
（§5 命題5.1、注意5.2）

\subsection{フレームワークの構造}\label{ux30d5ux30ecux30fcux30e0ux30efux30fcux30afux306eux69cbux9020}

上記の幾何学的命題を組み合わせると、次の構造が得られる：

\begin{verbatim}
中心投影 Φ: Π_R → S^n(R)
│
├── 軸の対等性（対称性 II）
│    どの軸を投影中心に選んでも同一構造
│
├── 主観座標の変換可能性（対称性 IV）
│    異なる軸の主観座標は R をパラメータとして正逆変換可能
│
├── 離散安定性（対称性 I）
│    整数座標でも正則
│
├── 弁別不可能性（対称性 III, 定理5.2）
│    曲率方向・符号は内部から見えない
│
├── 加速度の測定可能性（対称性 V）
│    向心加速度 a = Rω² はスカラーとして測定可能
│
└── 第5軸の追加（論文 [3]）
     ├── 離散等方性の数論的要請 → N=4
     ├── 離散格子 → 量子化
     └── 測地線偏差 ∝ φ₀²/r⁴（1/r² 構造）
\end{verbatim}

このフレームワークは、加速度を生む独立な軸（曲率半径の異なる複数の軸）が共存しても幾何学的に破綻しない構造を持つ。各軸の主観座標は対称性IVにより相互変換可能であり、対称性IIにより全ての軸が対等である。

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\subsection{本研究が主張しないこと}\label{ux672cux7814ux7a76ux304cux4e3bux5f35ux3057ux306aux3044ux3053ux3068}

本研究は純粋に幾何学的な命題の証明に限定される。以下の物理的同定は行わない：

\begin{itemize}
\tightlist
\item
  第5軸 \(\phi\) が物理的な「電荷」次元と同一であること
\item
  \(|\xi|^2 \propto \phi_0^2/r^4\) が Coulomb の法則と同型であること
\item
  Maxwell 方程式が本枠組みから導出されること
\item
  電磁場と重力場の統一が達成されていること
\end{itemize}

これらの物理的同定は、本研究の幾何学的命題を超える追加の仮定を要する（論文 {[}3{]} §6）。

## 概要

前論文 [1] において、グノモン正写像により4次元ユークリッド空間を超球面 $S^4(R)$ に写像し、Einstein テンソルの幾何学的恒等式 $G_{\mu\nu} + \Lambda g_{\mu\nu} = 0$（$\Lambda = 3/R^2$）を導出した。しかし以下の問いは未解決であった：

1. なぜ時間は空間と異なる符号 $(-,+,+,+)$ を持つのか
2. 写像は離散的な座標系でも定義可能か
3. 自由パラメータはいくつ必要か

本稿は、これらの問いに対する幾何学的回答を与える。第一に、グノモン写像の $R$ 方向が内部観測者にとって原理的に不可観測であることを動機として、Lorentzian 符号を公理的に導入する（公理 3.1）。第二に、物理計量が2乗形式であるという事実が離散化の数論的必要条件であることを指摘し（§6.1）、唯一の結合定数 $C$（次元：長さ）の導入により写像が離散座標系で定義可能であることを示す。Euclidean 符号では写像の大域的正則性が保証され、Lorentzian 符号では宇宙論的ホライズンが自然に出現することを証明する（定理 6.1）。第三に、$C$ から導かれる物理量の整合性を確認する（§8）。

## 1. はじめに

### 1.1 前論文の到達点

前論文 [1] は、$n$ 次元ユークリッド空間から $n+1$ 次元超球面 $S^n(R)$ への中心射影（グノモン正写像）

$$\Phi: \Pi_R \to S^n(R), \quad Y^\mu = \frac{Rx^\mu}{l}, \quad Y^{n+1} = \frac{R^2}{l}, \quad l = \sqrt{R^2 + \sum_\mu (x^\mu)^2} \tag{1.1}$$

の $n = 4$ の場合に対し、引き戻し計量

$$g_{\mu\nu} = \frac{R^2}{l^2}\left(\delta_{\mu\nu} - \frac{x_\mu x_\nu}{l^2}\right) \tag{1.2}$$

を導出し、$S^4(R)$ が定曲率空間として

$$G_{\mu\nu} + \Lambda g_{\mu\nu} = 0, \qquad \Lambda = \frac{3}{R^2} \tag{1.3}$$

を満たすことを示した。さらに付録 D において符号パラメータ $\epsilon_\mu \in \{+1, -1\}$ を導入し、Euclidean/Lorentzian の統一的記述

$$g_{\mu\nu} = \frac{R^2}{l^2}\left(\epsilon_\mu \delta_{\mu\nu} - \frac{\epsilon_\mu \epsilon_\nu x_\mu x_\nu}{l^2}\right), \qquad l^2 = R^2 + \sum_\mu \epsilon_\mu (x^\mu)^2 \tag{1.4}$$

を与えた。

### 1.2 未解決の問い

前論文は $\epsilon_4 = -1$（Lorentzian 符号）を**選択肢**として示したが、なぜそれが物理的に実現されるかを説明しなかった。本稿では、この符号に対する幾何学的動機付けを与え、それを**公理として採用した場合の帰結**を体系的に示す。さらに、物理計量が2乗形式であるという普遍的事実が離散化の必要条件であることを指摘し、離散座標系の導入の動機を明確にする。

## 2. 一般次元での Einstein テンソル

前論文 [1] の結果を一般次元に拡張して整理する。

**命題 2.1**（一般次元の Einstein テンソル [2, 3]）  
$n$ 次元球面 $S^n(R)$ 上では：

$$R_{\mu\nu\rho\sigma} = \frac{1}{R^2}(g_{\mu\rho}g_{\nu\sigma} - g_{\mu\sigma}g_{\nu\rho}), \qquad R_{\mu\nu} = \frac{n-1}{R^2}g_{\mu\nu} \tag{2.1}$$

$$R_{\text{scalar}} = \frac{n(n-1)}{R^2}, \qquad G_{\mu\nu} = -\frac{(n-1)(n-2)}{2R^2}g_{\mu\nu} \tag{2.2}$$

$$\Lambda_n = \frac{(n-1)(n-2)}{2R^2} \tag{2.3}$$

$\blacksquare$

| 次元 $n$ | $\Lambda_n$ | Einstein テンソルの状態 |
| -------- | ----------- | ----------------------- |
| 1        | 0           | 自明解（矛盾なし）      |
| 2        | 0           | 恒等的に零（自明）      |
| 3        | $1/R^2$     | 非自明（最低次元）      |
| **4**    | **$3/R^2$** | **前論文の主結果**      |

**注意 2.1：** GR はいかなる次元でも矛盾を生じない。低次元では自明解になるだけであり、「GR が成立する」$\neq$「非自明な物理が存在する」。非自明な GR は $n \geq 3$ で成立し、$n = 4$ は特に物理的に際立つ。

## 3. Lorentzian 符号の幾何学的動機付けと公理的設定

### 3.1 $R$ 方向の構造的特異性

グノモン写像 (1.1) の5次元像を分析する。

$$\underbrace{Y^1, Y^2, Y^3, Y^4}_{\text{4次元座標}} = \frac{R x^\mu}{l}, \qquad \underbrace{Y^5}_{\text{$R$ 方向}} = \frac{R^2}{l}$$

| 性質                    | $Y^1, \ldots, Y^4$                | $Y^5$               |
| ----------------------- | --------------------------------- | ------------------- |
| 球面上の振る舞い        | $S^4(R)$ に沿って「丸まる」       | 垂直方向（平坦）    |
| $r \to \infty$ での極限 | $Y^\mu \to R \hat{x}^\mu$（飽和） | $Y^5 \to 0$（消失） |
| $r = 0$ での値          | $Y^\mu = 0$                       | $Y^5 = R$（最大値） |

**命題 3.0**（$R$ 方向と4次元座標の非対称性）  
$Y^5$ は $Y^1, \ldots, Y^4$ とは質的に異なる：4次元座標は $S^4(R)$ の内在的座標であるのに対し、$Y^5$ は埋め込み空間の法線方向であり、球面の内部に留まる観測者が持つ測定手段では原理的に測定できない。$\blacksquare$

### 3.2 不可観測次元の虚軸化

**公理 3.1**（Lorentzian 符号の公理的設定）

以下の二つの公理を採用する：

> **公理 A（認知制約）：** $S^n(R)$ の内在計量 $g_{\mu\nu}$（式 (1.2)）は埋め込み座標 $Y^5$ を含まない（Gauss の驚異定理）。したがって、内在計量のみを用いる観測者は、$Y^5$ 方向を直接観測する手段を持たない。

> **公理 B（不可観測次元の虚軸化）：** 公理 A により内在的に不可観測な $R$ 方向を、有効的な虚軸として表現する。

公理 A は Gauss の驚異定理から直接従う数学的帰結であり、仮定の余地はない。公理 B は物理的仮定であり、「不可観測な次元を虚軸で表現する」という特定の解釈を採用するものである。

以下、公理 A, B のもとでの帰結を述べる。

(i) 4次元座標のうち3つ $x^1, x^2, x^3$ を空間座標、残り1つ $x^4 = ct$ を時間座標と同定する（この同定は外部からの物理的入力である）。

(ii) 公理 B により、$R$ 方向の有効的な寄与を虚軸として表現する：

$$w_{\text{有効}} = i \cdot \frac{1}{R} \tag{3.1}$$

(iii) 前論文 [1] 付録 D の統一的記述（式 (1.4)）において、グノモン写像の構造は $Y^5 = R^2/l$、$l^2 = R^2 + \sum_\mu \epsilon_\mu (x^\mu)^2$ である。公理 B のもとで $R \to iR$ とすると、$l^2$ の定義式において $R^2 \to -R^2$ となる。しかし $Y^5$ が不可観測（公理 A）であるため、$l^2$ の全体符号は物理的に意味ある形で再定義される必要がある。$R^2$ を正に保つように $l^2$ を再定義すると、対応する補償として $x^4 = ct$ の項の符号が反転し、$\epsilon_4 = +1 \to -1$ となる。これは前論文付録 D の Remark D.3 で示した $\epsilon_4$ の選択に物理的動機を与えるものである。結果として：

$$w_{\text{有効}}^2 = -\frac{1}{R^2} \tag{3.2}$$

$$l^2 = R^2 + x^2 + y^2 + z^2 - c^2t^2 \tag{3.3}$$

**Lorentzian 符号 $(-,+,+,+)$ が、公理 A, B のもとで得られる。**

**注意 3.1**（本主張の位置づけ）  
公理 A は数学的に厳密（Gauss の驚異定理の直接的帰結）であるが、公理 B は物理的仮定である。「不可観測 → 虚軸」の同定は、他の解釈（例：不可観測次元が単に有効理論から消去される、ゲージ自由度として扱われる、Kaluza-Klein 的にコンパクト化される等）も原理的には可能である。本稿は公理 B を採用した場合の帰結を探求する。公理 B の正当化は、その帰結（Lorentzian 符号、ローレンツ群、宇宙論的ホライズン）が観測と整合することによる。

### 3.3 従来の Wick 回転との関係

| 従来の Wick 回転                    | 本稿の定式化                                      |
| ----------------------------------- | ------------------------------------------------- |
| $t \to it$ を数学的手続きとして導入 | $R$ 方向の不可観測性が $w = i/R$ を動機付ける（公理 B） |
| Lorentzian 符号は経験的事実         | Lorentzian 符号は公理 A, B の帰結             |
| なぜ符号が反転するかは未説明        | 不可観測な直交軸 → 虚軸 → 負符号          |

**本稿の立場は因果関係を逆転させる：** Wick 回転が「便利な手法」なのではなく、内部観測者にとって Lorentzian 世界が「認知される現実」であり、Euclidean 構造が「背後にある幾何学的実体」である。

## 4. ローレンツ群の幾何学的起源

### 4.1 対称群の鎖

$S^4(R)$ の等長群は符号選択に依存する。

**命題 4.1**（対称群の鎖 [3, 6]）

$$SO(5) \xrightarrow[\text{公理 3.1}]{\epsilon_4: +1 \to -1} SO(4,1) \xrightarrow[R \to \infty]{\text{平坦極限}} SO(3,1) \tag{4.1}$$

- $SO(5)$：Euclidean $S^4(R)$ の等長群（10個の生成子）
- $SO(4,1)$：de Sitter 群（Lorentzian $dS_4$ の等長群）
- $SO(3,1)$：ローレンツ群（Minkowski 時空の等長群）

$\blacksquare$

**意義：** ローレンツ変換は「光速不変の原理から導かれる」のではなく、「グノモン写像の de Sitter 対称性の平坦極限」として幾何学的に導出される。光速不変はこの幾何学的構造の帰結であり、より根本的な原理はグノモン写像の対称性である。

## 5. 内部観測者の認知限界

### 5.1 三角形内角の和の上限

**定理 5.1**（内角上限定理）  
$S^n(R)$ 上の測地三角形（2次元断面）の内角 $\alpha, \beta, \gamma$ について：

$$\alpha + \beta + \gamma = \pi + \frac{A}{R^2} \tag{5.1}$$

ここで $A$ は測地三角形の面積。最大面積（半球面）$A_{\max} = 2\pi R^2$ において：

$$(\alpha + \beta + \gamma)_{\max} = 3\pi = 540° \tag{5.2}$$

**この上限値 540° は $R$ に依存しない普遍的な幾何学的定数である。** $\blacksquare$

**証明：** 球面三角法の Gauss–Bonnet の定理 [2] より、Gauss 曲率 $K = 1/R^2$ の曲面上の測地三角形について $\alpha + \beta + \gamma - \pi = KA = A/R^2$。$A$ の最大値は半球面の面積 $2\pi R^2$ であるから、$(\alpha + \beta + \gamma)_{\max} = \pi + 2\pi = 3\pi$。この値は $R$ をキャンセルしている。$\square$

**注意 5.1**（$R$ の測定可能性について）  
上限値 540° が $R$ に依存しないことと、$R$ 自体が測定不能であることは異なる。内部観測者は、既知の面積 $A$ を持つ三角形の内角超過 $\Delta = A/R^2$ を測定することで $R$ の値を**決定できる**。正確には、$R$ は内在曲率 $K = 1/R^2$ を通じて Riemann テンソルから原理的に測定可能である。定理 5.1 が示しているのは「**上限値**が $R$ に依存しない」という幾何学的事実のみである。

### 5.2 共形スケーリングの構造

**命題 5.2**（共形因子の性質）  
グノモン写像は共形因子 $R/l$ を含む。ただし $l = \sqrt{R^2 + r^2}$ は位置に依存するため、これは一様なスケーリングではなく**位置依存的な共形変換**である。

$$x^\mu \to \frac{R}{l}x^\mu \tag{5.3}$$

原点（$r = 0$）では $R/l = 1$（変換なし）、$r \to \infty$ では $R/l \to 0$（圧縮）。$\blacksquare$

## 6. 離散座標と結合定数 $C$

### 6.1 動機：2乗計量と離散化の数論的関係

前論文 [1] は連続座標 $x^\mu \in \mathbb{R}$ のもとで議論した。本節では座標を離散化し、写像が整数格子上でも定義可能であることを示す。

離散化の導入に先立ち、以下の数論的事実を指摘する。

**観察 6.0**（2乗形式の数論的動機付け）  
整数格子 $\mathbb{Z}^n$ 上で、乗法的ノルム（$N(zw) = N(z)N(w)$）を持ちかつ整数に値を取る計量を定義しようとすると、ガウス整数環 $\mathbb{Z}[i]$ のノルム $N(a+bi) = a^2 + b^2$ が最小の構造となる。より一般に、Hurwitz の定理 (1898) [15] により、乗法的ノルムを持つ実数上の組成代数は次元 $1, 2, 4, 8$（実数、複素数、四元数、八元数）に限られる。$\mathbb{Z}[i]$ のノルム $a^2 + b^2$ はこの中で最小次元の構造であり、物理計量が2乗形式であることは離散格子上で乗法的に閉じた計量を定義するための自然な選択である。$\blacksquare$

**注意 6.0**（離散性の傍証）  
物理計量が2乗形式であること（$ds^2 = g_{\mu\nu}dx^\mu dx^\nu$, $l^2 = R^2 + \sum \epsilon_\mu(x^\mu)^2$ 等）は、私たちの時空が離散的であることの傍証となる。連続時空でもたしかに2乗計量は成立する（矛盾しない）が、離散化するためには必須である。含意の方向は一方向である：

$$\text{離散性} \Longrightarrow \text{2乗計量が必須}$$
$$\text{2乗計量} \not\Longrightarrow \text{離散性}（ただし傍証となる）$$

### 6.2 結合定数 $C$ の導入

**定義 6.0**（結合定数 $C$）  
次元「長さ」を持つ定数 $C > 0$ を導入し、曲率 $w = C/R$ を定義する。以下の離散化は、$w$ を非負整数に制限するという**追加仮定**のもとでの帰結を調べるものである：

$$w = \frac{C}{R} \in \mathbb{Z}_{\geq 0}, \qquad R = \frac{C}{w} \tag{6.1}$$

無次元座標を $\tilde{x}^\mu = x^\mu / C$ として、$\tilde{x}^\mu \in \mathbb{Z}$ に制限する。

### 6.3 離散グノモン写像

**定義 6.1**（離散グノモン写像）  
無次元量を用いて：

$$\tilde{L}^2 = 1 + w^2 \tilde{\sigma}^2, \qquad \tilde{\sigma}^2 = \sum_\mu \epsilon_\mu (\tilde{x}^\mu)^2 \tag{6.2}$$

$$\tilde{Y}^\mu = \frac{\tilde{x}^\mu}{\tilde{L}}, \qquad \tilde{Y}^5 = \frac{1}{w\tilde{L}} \tag{6.3}$$

ここで $w, \tilde{x}^\mu \in \mathbb{Z}$、$\epsilon_\mu = (+1,+1,+1,-1)$。

### 6.4 写像の定義域と正則性

**定理 6.1**（符号依存の正則性）  
離散グノモン写像 (6.2)–(6.3) において：

**(i) Euclidean 符号**（$\epsilon_\mu = (+1,+1,+1,+1)$）では：

$$\tilde{\sigma}^2 = \tilde{x}^2 + \tilde{y}^2 + \tilde{z}^2 + \tilde{t}^2 \geq 0 \quad \Longrightarrow \quad \tilde{L}^2 = 1 + w^2\tilde{\sigma}^2 \geq 1 > 0 \tag{6.4}$$

分母 $\tilde{L}$ は**いかなる格子点においても零にならず**、写像は大域的に正則である。

**(ii) Lorentzian 符号**（$\epsilon_\mu = (+1,+1,+1,-1)$）では：

$$\tilde{\sigma}^2 = \tilde{x}^2 + \tilde{y}^2 + \tilde{z}^2 - \tilde{t}^2 \tag{6.5}$$

時間的領域（$\tilde{t}^2 > \tilde{x}^2 + \tilde{y}^2 + \tilde{z}^2$）では $\tilde{\sigma}^2 < 0$ となり得るため、$\tilde{L}^2 = 1 + w^2\tilde{\sigma}^2 < 1$ となる。特に $w^2|\tilde{\sigma}^2| > 1$ のとき $\tilde{L}^2 < 0$ となり、写像は定義域外となる。

**写像は $\tilde{L}^2 > 0$ の領域でのみ定義される。** $\blacksquare$

**証明：** (i) Euclidean の場合：$\tilde{\sigma}^2 \geq 0$（各項は整数の2乗の和に限られる）であるから $w^2\tilde{\sigma}^2 \geq 0$、したがって $\tilde{L}^2 \geq 1$。(ii) Lorentzian の場合：$\tilde{\sigma}^2 = \tilde{x}^2 + \tilde{y}^2 + \tilde{z}^2 - \tilde{t}^2$ は負になり得る。$\tilde{L}^2 = 0$ の境界は：

$$1 + w^2(\tilde{x}^2 + \tilde{y}^2 + \tilde{z}^2 - \tilde{t}^2) = 0 \quad \Longleftrightarrow \quad c^2t^2 = R^2 + x^2 + y^2 + z^2 \tag{6.6}$$

これは de Sitter 時空における**宇宙論的ホライズン**に対応する [6]。$\square$

**注意 6.1**（ホライズンの幾何学的意味）  
式 (6.6) のホライズンは追加仮定なしにグノモン写像の定義域の境界として自然に出現する。これは Guo ら [6] が Beltrami-de Sitter 座標系において確認した既知の構造と一致する。Beltrami 座標は de Sitter 時空の**静的パッチ**（$l^2 > 0$ の領域）のみを被覆し、ホライズンの存在はグノモン写像（= Beltrami 座標）の本質的な性質である。

**注意 6.2**（$\tilde{L}^2 = 0$ の離散格子解とディオファントス方程式）  
離散 Lorentzian 版において、$\tilde{L}^2 = 0$（ホライズン上の格子点）の存在は自明ではない。$w = 1$ かつ等方的な格子点 $\tilde{x} = \tilde{y} = \tilde{z} = s$ のとき、条件は Pell 方程式 $\tilde{t}^2 - 3s^2 = 1$ に帰着し、整数解が存在する（最小解 $(s, \tilde{t}) = (1, 2)$）。これは離散格子上にもホライズンが出現することを示す。

一方、著者の先行研究 [14] は、離散 Minkowski 格子 $\mathbb{Z}^{1+N}$ 上の等方的測地線方程式 $l^2 + t^2 = Nx^2$（$l, t, x \in \mathbb{Z}_{>0}$）の解の存在条件を完全に決定した。Fermat の二平方和定理および無限降下法により：

> **定理（[14] 定理 3.1）：** $l^2 + t^2 = Nx^2$ が非自明な正整数解を持つための必要十分条件は、$N$ の素因数分解において $p \equiv 3 \pmod{4}$ の素数の指数がすべて偶数であること。

これは本稿のグノモン離散化と直接的に関連する。本稿の枠組みは任意の $n$ 次元球面 $S^n(R)$ に拡張可能であり（命題 2.1）、時間軸が1次元であると仮定する場合、空間次元 $N$ に対して等方的な離散測地線の存在は $N$ の素因数構造による数論的制約を受ける。$N = 1$ から $N = 11$ までの完全な分類を以下に示す：

| 空間次元 $N$ | 時空次元 $D = N+1$ | $N$ の素因数 | 等方解の存在 | 備考 |
|---|---|---|---|---|
| 1 | 2 | $1$ | あり ✅ | Pythagorean 三つ組 |
| 2 | 3 | $2$ | あり ✅ | |
| **3** | **4** | $3$ ($3 \equiv 3 \pmod{4}$, 奇数乗) | **なし ❌** | **我々の宇宙（空間のみ）** |
| 4 | 5 | $2^2$ | あり ✅ | Kaluza-Klein 拡張 |
| 5 | 6 | $5$ ($5 \equiv 1 \pmod{4}$) | あり ✅ | |
| 6 | 7 | $2 \times 3$ ($3$: 奇数乗) | なし ❌ | |
| 7 | 8 | $7$ ($7 \equiv 3 \pmod{4}$, 奇数乗) | なし ❌ | |
| 8 | 9 | $2^3$ | あり ✅ | |
| 9 | 10 | $3^2$ (偶数乗) | あり ✅ | 超弦理論の次元 |
| 10 | 11 | $2 \times 5$ | あり ✅ | M理論の次元 |
| 11 | 12 | $11$ ($11 \equiv 3 \pmod{4}$, 奇数乗) | なし ❌ | |

また、$N = 25$（$D = 26$, $5^2$）では等方解が存在し、これはボソン弦理論の次元と一致する。

$N = 3$ が「算術的アノマリー」を持つこと、および $N = 4, 9, 10, 25$ が存在理論物理学で知られる特定の次元と一致することは、本枠組みの任意次元拡張における重要な数論的制約である。どの追加次元がどの物理的力に対応するかは本稿の範囲外であり、今後の課題である。

**系 6.2**（連続極限との形式的整合）  
$C \to 0$（格子間隔ゼロの極限）で $w = C/R \to 0$（固定 $R$ に対して）かつ $\tilde{x}^\mu = x^\mu/C \to \infty$（固定 $x^\mu$ に対して）。このとき $\tilde{L}^2 = 1 + (C/R)^2 \sum \epsilon_\mu (x^\mu/C)^2 = 1 + \sum \epsilon_\mu (x^\mu)^2/R^2 = l^2/R^2$。したがって $\tilde{Y}^\mu = x^\mu/(l/R) = Rx^\mu/l = Y^\mu$ であり、連続版 (1.1) に**形式的に**一致する。ただし、$w \in \mathbb{Z}$ から連続体への収束の数学的な厳密さ（格子の稠密化による連続体の復元）は、今後の課題とする。$\square$

### 6.5 $C$ の物理的同定

**仮説 6.3**（$C$ と Planck 長さの同定）

$$C \sim \ell_P = \sqrt{\frac{\hbar G}{c^3}} \approx 1.616 \times 10^{-35}\,\text{m} \tag{6.5}$$

この仮説のもとで以下が量子化される：

| 物理量   | 量子化の形           | $C = \ell_P$ での値                    |
| -------- | -------------------- | -------------------------------------- |
| 最小長さ | $\Delta x = C$       | $\ell_P \approx 1.6 \times 10^{-35}$ m |
| 最小時間 | $\Delta t = C/c$     | $t_P \approx 5.4 \times 10^{-44}$ s    |
| 宇宙定数 | $\Lambda = 3w^2/C^2$ | 離散値のみ                             |

## 7. 特異点回避と宇宙論的ホライズン

### 7.1 Euclidean 版の正則性（大域的）

**定理 7.1**（前論文 [1] 命題 C.1 の再掲）  
Euclidean 符号（$\epsilon_\mu = (+1,+1,+1,+1)$）において、固定 $R > 0$ に対し、$\Phi: \mathbb{R}^4 \to S^4(R)$ は $\mathbb{R}^4$ 全域で $C^\infty$ 級正則である。分母 $l = \sqrt{R^2 + r^2} \geq R > 0$ であるから、$l$ は常に正である。$\blacksquare$

### 7.2 Lorentzian 版の正則性（静的パッチ内）

**定理 7.2**（Lorentzian 版の正則性）  
Lorentzian 符号（$\epsilon_\mu = (+1,+1,+1,-1)$）において、グノモン写像は $l^2 = R^2 + x^2 + y^2 + z^2 - c^2t^2 > 0$ を満たす領域（静的パッチ）で $C^\infty$ 級正則である。境界 $l^2 = 0$、すなわち $c^2t^2 = R^2 + x^2 + y^2 + z^2$ が**宇宙論的ホライズン**を定義する（定理 6.1 (ii)）。$\blacksquare$

**注意 7.1**（離散版の正則性）  
離散 Euclidean 版では、定理 6.1 (i) より $\tilde{L} \geq 1 > 0$ が全格子点で成立する。離散 Lorentzian 版では、$\tilde{L}^2 > 0$ を満たす格子点でのみ正則である。

### 7.3 $R \to 0$ の特異点回避

**命題 7.3**  
Euclidean 版において、$R \to 0$（$r > 0$ 固定）では：

$$Y^5 = \frac{R^2}{\sqrt{R^2 + r^2}} \to 0, \qquad Y^\mu = \frac{Rx^\mu}{\sqrt{R^2 + r^2}} \to 0 \tag{7.1}$$

写像像は原点に縮退するが**発散しない**。すなわち、曲率半径がゼロに近づく極限においても、写像は数学的に定義されたままである。$\blacksquare$

### 7.4 情報保存

**定理 7.4**（全単射・情報保存）  

**(i) Euclidean 版：** $R > 0$ の固定に対し、グノモン写像 $\Phi: \mathbb{R}^4 \to S^4(R)$ は開半球面への $C^\infty$ 級全単射である。逆写像 $\Phi^{-1}$ が存在し、情報は原理的に保存される。

**(ii) Lorentzian 版：** $\Phi$ は $l^2 > 0$ の領域（静的パッチ）から de Sitter 時空の対応する領域への全単射である。ホライズン（$l^2 = 0$）を越えた領域は別の座標パッチで被覆する必要がある。

$\blacksquare$

## 8. 整合性の確認

### 8.1 $C = \ell_P$ の仮定に基づく物理量の整合性

仮説 6.3（$C = \ell_P$）を**仮定として**採用し、そこから導かれる物理量が既知の値と矛盾しないかを確認する。これは理論の検証ではなく整合性の確認であり、$C$ の独立な決定は§10 の今後の課題である。

**検証 1：宇宙定数 $\Lambda$ の現在値**

観測値 $\Lambda_{\text{obs}} \approx 1.1 \times 10^{-52}\,\text{m}^{-2}$ [7] と $\Lambda = 3/R^2$ から：

$$R_{\text{current}} = \sqrt{\frac{3}{\Lambda_{\text{obs}}}} \approx 1.65 \times 10^{26}\,\text{m} \approx 5.3\,\text{Gpc} \tag{8.1}$$

これは観測可能な宇宙の大きさ $R_H \approx 4.4 \times 10^{26}$ m のオーダーと整合する。

**検証 2：離散曲率パラメータ $w$**

$$w = \frac{C}{R_{\text{current}}} = \frac{1.6 \times 10^{-35}}{1.65 \times 10^{26}} \approx 10^{-61} \tag{8.2}$$

$w \ll 1$ であり、現在の宇宙は $w = 0$（Minkowski 平坦時空）に極めて近い離散格子点に対応する。

**検証 3：$w$ の整数性の条件**

$w \in \mathbb{Z}$ が要求されるが、$w \approx 10^{-61} \ll 1$ である。本モデルの離散的な $w$ では、$w = 0$（$\Lambda = 0$、完全平坦）と $w = 1$（$\Lambda = 3/C^2 \sim 10^{122} \Lambda_{\text{obs}}$、Planck 密度）の間に中間状態が存在しない。観測される微小な $\Lambda > 0$ は、$w$ の離散ステップでは再現できず、これは宇宙定数問題の離散版である。この問題の解決には、$w$ の離散性の修正（例：$w$ が有理数値を取り得る、または追加次元の寄与を含む）が必要であり、これは§10 の今後の課題に含まれる。

### 8.2 Planck スケールとの整合

| 物理量               | $C = \ell_P$ からの値                      | 既知の値                           | 整合性           |
| -------------------- | ------------------------------------------ | ---------------------------------- | ---------------- |
| Planck 長さ          | $C = 1.616 \times 10^{-35}$ m              | $\ell_P = 1.616 \times 10^{-35}$ m | ✅（定義）        |
| Planck 時間          | $C/c = 5.39 \times 10^{-44}$ s             | $t_P = 5.39 \times 10^{-44}$ s     | ✅                |
| $R_{\text{current}}$ | $\sqrt{3/\Lambda} = 1.65 \times 10^{26}$ m | $R_H \sim 4.4 \times 10^{26}$ m    | ✅（同オーダー）  |
| 初期 $w$             | $w_{\text{初期}} \sim C/C = 1$             | —                                  | 整合（最大曲率） |

### 8.3 本節の限界と注意

**注意 8.1**（循環論法の回避について）  
$C = \ell_P$ と仮定した上で Planck 量が出現すること自体は、仮定の直接的帰結であり独立な検証ではない。本節で非自明なのは、$R_{\text{current}}$ が宇宙の観測可能半径 $R_H$ と**独立に同オーダーとなる**こと（検証 1）のみである。$C$ の真の検証には、複数の独立な観測量から $C$ を決定し、それらが一致することを示す必要がある。これは§10 の博士課程の課題に含まれる。

## 9. 幾何学的全単射性とその情報論的含意

### 9.1 Euclidean 版での幾何学的特異点の不在

**命題 9.1**  
Euclidean 符号（$\epsilon_\mu = (+1,+1,+1,+1)$）において、グノモン写像 $\Phi$ は $C^\infty$ 級の全単射であるから（定理 7.1, 7.4 (i)）、$R$ をいかに小さくしても：

1. 幾何学的特異点は生じない
2. 情報は原理的に保存される
3. 逆写像 $\Phi^{-1}$ が存在し、情報は復元可能

$\blacksquare$

### 9.2 Lorentzian 版での制限

Lorentzian 符号では、情報保存は静的パッチ（$l^2 > 0$）内に限られる（定理 7.4 (ii)）。ホライズン（$l^2 = 0$）を越えた領域は Beltrami 座標では被覆されず、別の座標系（例：de Sitter 大域座標）が必要である。この制限がブラックホール物理にどう影響するかは今後の課題である。

### 9.3 離散版での情報保存

離散 Euclidean 座標系（§6）において、$C > 0$ かつ定理 6.1 (i) により $\tilde{L} \geq 1 > 0$ が全格子点で保証される。有限格子上の全単射は情報の保存を自明に保証する。

### 9.4 位置づけ

本節の結果は純粋幾何学の命題であり、量子重力・ホーキング輻射等の物理的議論とは独立である。物理的なブラックホール情報パラドックスへの適用には、グノモン枠組みでの量子論の定式化が必要であり、これは§10 の今後の課題に含まれる。

## 10. 博士課程への展望

本稿は、グノモン正写像と結合定数 $C$ による Lorentzian 時空の幾何学的起源を、電磁力を含まない純粋な4次元時空の枠組みで完結させた。以下の課題は、本稿の範囲を超えるものとして、博士課程以降の研究に委ねる。

### 10.1 任意次元への拡張と離散数論的制約

本稿のグノモン枠組みは $n$ 次元球面 $S^n(R)$ に対して任意の $n$ で定義できる（命題 2.1）。時間軸が1次元であると仮定した場合、注意 6.2 で示した離散数論的制約（Fermat の二平方和定理 [14]）は、等方的な離散測地線が存在する空間次元 $N$ を強く制約する。特に $N = 3$（4次元時空）が算術的に特異であり、$N = 4, 9, 10, 25$ が存在理論物理学の特定の次元と一致することの意味は、今後の重要な研究課題である。

注目すべきは、$N = 3$（4次元時空）では等方的な離散測地線が存在しないのに対し、1次元を追加した $N = 4$（5次元時空）で初めて解が存在するという事実である。この数論的制約は、離散時空が等方性を保つために**5次元目が必然的に要請される**ことを示唆する。Kaluza-Klein 理論 [8, 9] では、コンパクトな第5次元 $\phi$ を電荷（クーロン）の次元と同定し、その上の $U(1)$ ゲージ対称性から Maxwell 方程式を導出する。本稿の数論的制約と合わせると、離散等方性の要請 → 5次元の必要性 → Kaluza-Klein 的解釈 → 電磁力の創発、という論理的鎖が成立する可能性がある。この接続の厳密化は今後の論文の主題とする。前論文 [1] 付録 E に予備的な見通しを記した。

### 10.2 微細構造定数 $\alpha$ の幾何学的起源

$S^4 \hookrightarrow S^5$ の埋め込みと Wyler 公式 (1971) [10] との接続。予備的な数値計算では $\alpha_{\text{Wyler}} = 1/137.036$ が $0.0001\%$ の精度で再現される。グノモン模型の等長群 $SO(5,1)$ が Wick 回転により $SO(4,2) \cong SU(2,2)/\mathbb{Z}_2$ に変換されることは確認済みであるが、Wyler 公式の完全な導出は Bergman 核・Hermite 対称空間の表現論を必要とし、残る課題である。

### 10.3 強い力と弱い力

- **強い力：** 閉じ込めポテンシャル $V \sim \kappa r$ の幾何学的記述。点源ではなく弦源（フラックスチューブ）が必要であり、§6 の離散格子拡張における「弦型ソース」の導入が候補
- **弱い力：** フレーバー変換（$u \to d + W^+$）は座標の変化ではなく状態遷移であり、本モデルの座標系には記述する自由度がない。本質的に新しい拡張が必要

### 10.4 量子論の幾何学的基盤

離散座標系（§6）は格子場理論・ループ量子重力と構造的に類似する。経路積分・演算子形式との接続、および波動関数の幾何学的解釈の定式化。

### 10.5 粒子質量スペクトルの導出

$w_n = C/R_n \in \mathbb{Z}$ の制約から、既知の素粒子質量が離散スペクトルとして再現されるかの検証。

### 10.6 宇宙論的観測値との定量的整合

$R(t)$ のダイナミクス方程式の導出、Friedmann 方程式との詳細な対応、インフレーション・ダークエネルギーの定量的説明。

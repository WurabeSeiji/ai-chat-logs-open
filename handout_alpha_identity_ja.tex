{\small
\textbf{主結果：自己整合代数恒等式が $\alpha$ を \textbf{8.7 ppb} で予言}\enspace
4次元単位球の二つの初等的幾何学量 — $N(1) = 137$（半径3の4次元球への単位立方体完全包含数、直接計数）と $V_4(1) = \pi^2/2$（4次元単位球体積、標準公式）— が満たす自己整合方程式
\[ \alpha^{-1} = N(1) + V_4(1)\,\alpha = 137 + \tfrac{\pi^2}{2}\alpha \quad\Longleftrightarrow\quad \tfrac{\pi^2}{2}\alpha^2 + 137\alpha - 1 = 0 \]
の正根 $\alpha = 7.29735194\times 10^{-3}$ は CODATA 2018 観測値 $7.29735257\times 10^{-3}$ と相対誤差 $\mathbf{8.7 \times 10^{-8}}$（\textbf{8.7 ppb}、Eddington 整数当てのズレ $2.6\times 10^{-4}$ の約 1/3000）で一致。\\[2pt]
\textbf{Eddington 罠との決定的差異}\enspace
(1) 整数のみで当てる単一の数式ではなく、$\alpha$ 自身を含む\textbf{自己整合方程式}。(2) 整数 137 と補正係数 $\pi^2/2$ の\textbf{両方が独立な4次元幾何学から直接派生}（外部から借りない）。(3) 摂動展開 $1 = 137\alpha + V_4(1)\alpha^2$ が QFT の繰り込み構造と類比的。(4) 物理機構候補が独立な思考実験から提示済み。\\[2pt]
\textbf{留保}\enspace
本論文は $\alpha$ の第一原理導出を主張しない。提示するのは観察された数値構造であり、それが幾何学的必然か数値的偶然かは未解決問題として残す。}
\end{tcolorbox}

\textbf{$N(1) = 137$ の直接計数}\enspace
半径 $R=3$ の4次元球内に整数中心の単位立方体が完全包含される条件 $\sum_i (|c_i|+1/2)^2 \le 9$ を中心の絶対値パターン $(s_1,\ldots,s_4)$ で列挙：

{\footnotesize
\begin{tabular}{@{}lr@{}}
\toprule
パターン & 立方体数 \\
\midrule

\smallskip
$(2,1,1,0)$ 以降は不等式 $\sum s_i^2 + \sum s_i \le 8$ を満たさず除外。完全に手計算可能。

\smallskip
\textbf{$V_4(1) = \pi^2/2$ の標準公式}\enspace
$V_n(R) = \pi^{n/2}/\Gamma(n/2+1) \cdot R^n$ で $n=4, R=1$：$V_4(1) = \pi^2/\Gamma(3) = \pi^2/2 \approx 4.9348$。Planck 単位の4+1次元 BH エントロピー $S_{BH}(1) = \pi^2/2$ と一致（参考）。

\smallskip
\textbf{摂動展開構造}\enspace
等価形 $1 = 137\alpha + (\pi^2/2)\alpha^2$ は $\alpha$ の冪展開：tree-level $137\alpha$ と一階補正 $V_4(1)\alpha^2$ が単位に等しい。QFT の繰り込み構造と類比的。

\smallskip
\textbf{inside / outside 分解}\enspace
半径3の系は $V_4(3) = 81\pi^2/2 \approx 399.72$。うち 137 個の立方体が占める内部 34\%、境界の隙間が \textbf{66\%}。境界が支配的。
\begin{itemize}[topsep=1pt]
\item $137\alpha$：inside 137 個の幾何学的接点
\item $V_4(1)\alpha^2$：outside 境界自由度の自己エネルギー補正
\end{itemize}

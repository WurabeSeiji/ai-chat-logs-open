## 出発点：c=1, G=1 における加速度の次元

標準 SI 単位：
$$[a] = [L T^{-2}]$$

**c = 1（$[T] = [L]$）**：
$$[a] = [L][L^{-2}] = [L^{-1}]$$

**G = 1（$[M] = [L]$）**：
$$[L] = [T] = [M]$$

すべて同じ次元。

結論：c=1, G=1 において
$$\boxed{[a] = [k] = [\omega] = [L^{-1}] = [M^{-1}]}$$

加速度・波数・振動数はすべて同じ次元。長さ・時間・質量・エネルギー・運動量はすべて同じ次元。両者は**逆数関係**。

## 気づき 8（核心的修正）：c=1, G=1 では k = 1/m、Compton 関係ではない

これまで AI（Claude）はしばしば $k = m/\hbar$（Compton 関係）を持ち込んでいたが、**これは c=1, ℏ=1 の解釈であって c=1, G=1 ではない**。

### 厳密な c=1, G=1 次元解析

定義：
- $c = 1$ → $[T] = [L]$
- $G = 1$ → $[M] = [L]$

つまり $[L] = [T] = [M]$、すべて同じ次元。

ℏ は採用しない（ℏ=1 はメモリで明示的に禁止）。純幾何学的に：
$$\boxed{k = \frac{1}{m} \quad (\text{c=1, G=1, ℏ なし})}$$

**重い粒子 → 小さい波数値**

### レプトン世代の検証

| 粒子 | 質量 m | 波数 k = 1/m（任意単位） |
|---|---|---|
| 電子 e⁻ | 0.511 MeV | $k_x \propto 1.96$（最大） |
| ミューオン μ⁻ | 105.7 MeV | $k_y \propto 0.00946$（中間） |
| タウ τ⁻ | 1777 MeV | $k_z \propto 0.000563$（最小） |

比率：
$$k_x : k_y : k_z \approx 3477 : 16.8 : 1$$

**v_z（タウの空間波数）が最小、v_x（電子の空間波数）が最大**。

### 既存 [質量構造] の修正必要事項

メモリ既存：
> 第3世代が z 軸を使う → k_z · k_t が最大 → 最重量

**これは Compton 解釈に基づく誤り**。

正しくは：
- 第3世代（τ, t, b）が z 軸を使う → **k_z · k_t が最小**（次元 [M⁻²] の値も 1/m² で小さい）→ 最重量
- 第1世代（電子、u, d）が x 軸を使う → **k_x · k_t が最大** → 最軽量

符号付き面積：
$$M(i,j) = k_i \cdot k_t \propto \frac{1}{m^2}$$

**小さい M ↔ 重い粒子**。

### 物理的描像の反転

| 直感（誤り） | 純幾何（正しい） |
|---|---|
| 重い粒子は「強く振動」して大きい波数 | 重い粒子は「ゆっくり振動」して小さい波数 |

c=1, G=1 では **質量 = Schwarzschild 半径**として読むのが自然。重い粒子はより大きな曲率半径を持つ「大きな塊」で、ゆっくり位相回転する。

## 気づき 9：離散運動方程式の制約構造

加速度・波数の次元解析が確定したので、**離散世界での運動方程式**を厳密に検討する。

### 出発点

連続式：
$$x = \frac{1}{2} a (\delta t)^2$$

c=1, G=1, $a = 1/R = 1/r^2$（R は平方整数）として：
$$x = \frac{(\delta t)^2}{2 r^2}$$

### 離散整数性の要請

すべての量（x, δt, r）は int256 整数。$x = 1$ で δt = 1 とすると：
$$1 = \frac{1}{2r^2} \quad \Rightarrow \quad 2r^2 = 1$$

これは整数解を持たない。**δt = 1 では破綻**。

### 2r² は決して完全平方数にならない

$2r^2 = m^2$ なら $m/r = \sqrt{2}$（無理数）→ 矛盾。**$\sqrt{2}$ の無理性が離散性を強制する**。

### 三つの解決策

#### A. x = 1 を禁止、最小単位を x = 2 とする

$\delta t = 2rj, x = 2j^2$ の形で整数解：

| j | δt | x |
|---|---|---|
| 1 | 2r | **2** |
| 2 | 4r | 8 |
| 3 | 6r | 18 |

最小空間ステップは **2、決して 1 ではない**。

物理的解釈：
- フレームワーク自然単位 = **「半プランク長」**
- 最小物理ステップ（1 プランク長）= **2 フレームワーク単位**
- 連続式の 1/2 因子はこの **半単位構造**に由来

#### B. Pell 方程式による特権的 R 値

離散和 $x = a \cdot n(n-1)/2$ で $x = 1$ を要求すると：
$$n(n-1) = 2r^2$$

これは Pell 方程式 $m^2 - 8r^2 = 1$ と等価。解：

| r | R = r² | n |
|---|---|---|
| 1 | 1 | 2 |
| 6 | 36 | 9 |
| 35 | 1225 | 50 |
| 204 | 41616 | 289 |
| 1189 | 1413721 | 1682 |

**特定の R 値（Pell 数列）でのみ x = 1 が許される**。これは **量子化された安定状態**を示唆するかもしれない。

#### C. 連続式の 1/2 を取り除く（離散専用式）

$x = a \cdot \delta t^2$（1/2 なし）なら $\delta t = r$ で $x = 1$ が成立。ただし連続極限と整合しない。

### 採用する解釈：選択肢 A

選択肢 A が最もシンプルで、人為的な数学操作を必要としない。

**基本関係式**：
$$\boxed{(\delta t)^2 = 2 R \cdot x}$$

これがフレームワーク離散運動方程式の核心。

### 多段ステップでも整合

$\Delta v = a \cdot \delta t = 2j/r$ も整数（適切な j 選択で）。

| r | 最小 j | δt = 2rj | x = 2j² | Δv = 2j/r |
|---|---|---|---|---|
| 1 | 1 | 2 | 2 | 2 |
| 2 | 1 | 4 | 2 | 1 |
| 3 | 3 | 18 | 18 | 2 |
| 4 | 2 | 16 | 8 | 1 |

**すべて整数で閉じる**。多段ステップ・複数粒子のシミュレーション可能。

## 気づき 10：R → 0 と特異点問題の幾何学的解決

R → 0 の極限で何が起こるか、慎重に分析する。

### 540° 漸近極限

連続系で球面三角形：
$$\text{角度和} = \pi + \frac{A}{R^2}$$

R → 0 で固定主観空間三角形（サイズ L）の場合、各内角は π = 180° に漸近、**角度和は厳密に 540° = 3π に収束**。

ただし **有限 ρ では 540° を厳密に超えることはない**：
- $\theta = \arctan(\rho/R)$ で R → 0 では θ → π/2（赤道）
- 球面三角形が **半球被覆** する境界状態
- これが 540° の幾何学的意味

### 540° 三角形 = 半球の表面

3 次元での具体的な絵：
- 3 頂点が赤道上（等間隔または不等間隔）
- 3 辺が **赤道に沿った大円弧**
- **内部が上半球の表面全体**
- 各頂点の内角 180°（辺が一直線で繋がる、退化）

つまり：
$$\boxed{\text{540° 三角形} = \text{半球の表面そのもの}}$$

中心投影との対応：
- 主観空間（4D）↔ S^4 の上半球（4D 表面）
- 540° 三角形 ↔ 半球全体の超表面

**主観空間とは「ある半球の内部空間」** という幾何学的描像。

### 720° 上限の妥当性

D1 の SpinPhase は最大 720° まで定義済（Spin 1/2 の二重被覆）：
- 540°（半球被覆）<< 720°（フェルミオン全周回転）
- **180° の余裕**で安全に表現可能
- 離散系でも巻き付きなしで処理可能

### 4 次元構造は破綻しない

R = 0 で起こるのは：
- 三角内角和が **540° に到達**（漸近極限）
- 三角形が **退化**（赤道に潰れる）
- 計量が共形的に縮退（しかし共形構造は保持）
- **4 次元構造は論理的に保持**
- 720° 位相空間は完全に正則

つまり R = 0 は **「破綻」ではなく「最大限に湾曲した状態」**。

### 局所と大域の分離

R → 0 の主観空間で：

| 性質 | 状態 |
|---|---|
| 局所（一点）の 3 軸直交性 | **90° 直交を保持** |
| 微小三角形 | 局所ユークリッド近似が成立 |
| 有限三角形 | **すべて退化、角度和 540°** |
| 軸の大域的独立性 | **失われる**（少し歩くと軸が収斂） |
| 共形構造 | 保持 |
| 位相空間（720°） | 正則 |

**「局所には正常、大域には極限」** の状態。

### ビッグバン特異点の幾何学的解決

これが特異点問題の幾何学的解決：

| 標準 GR | 本フレームワーク |
|---|---|
| R = 0 で曲率発散 | R = 0 でも 540° 漸近のみ、発散しない |
| 特異点で物理が破綻 | 退化状態（情報保持、整然） |

ビッグバン = 「**形がない、色（位相）だけある**」最大対称状態 → 何らかの遷移で R が成長 → 大域構造が「立ち上がる」。

各粒子は **4 次元整数格子（xyzt メッシュ）** 上を独立に移動：

- 位置は常に整数座標
- 移動ピッチは 2 の倍数（$x = 2j^2$ から）
- 各粒子は自分のリズム（δt_i）で動く
- **同期は不要、メッシュ整合だけが条件**

### 観測可能・不可能の区別

| 観測量 | 内容 |
|---|---|
| 粒子の位置 (x, y, z, t) | 整数メッシュ上の点（外部観測者に見える） |
| 移動ピッチ | 2 の倍数（離散制約） |

**観測できないもの**：
- 粒子の固有時間 τ_i
- 粒子の δt_i
- これらは **粒子の内部状態**で、外部観測者には不可視

つまり：
$$\boxed{\text{固有時間は粒子の私的情報、観測されるのは位置のみ}}$$

### Lamport 的因果順序

メモリ既存「**Lamport 的な論理的因果順序の性質は維持される（Δt が因果を定義する）**」と完全整合。

```
粒子 A:  ●──●──●──●──●──●  (δt_A = 2)
粒子 B:    ●─────●─────●─    (δt_B = 4)
粒子 C:  ●─●─●─●─●─●─●─●─●  (δt_C = 1)
```

各粒子が自分のティックで進む。**「同時刻」は存在しない**が、**「因果関係」は明確**（メッシュの t 軸で順序付けられる）。

### 相互作用 = メッシュ点の共有

粒子 A と B が相互作用するのは：
$$\text{粒子 A の位置} = \text{粒子 B の位置} \quad (\text{4D メッシュ点が一致})$$

両者の固有時間は異なる値だが、**メッシュ点 (x, y, z, t) は同一**。

これは特殊相対論と整合：
- 2 つの時計が交差点で違う時刻を示す（時間の遅れ）
- どちらも正しい（観測者依存）
- イベント自体は不変（時空の点）

### 半プランク単位とプランク単位の関係

「ピッチ 2 の倍数」の意味：

- **計算精度**：フレームワーク単位（半プランク長）で行う
- **物理的単位**：観測される位置はプランク長の倍数（2 フレームワーク単位）
- 連続式の 1/2 因子はこの **半単位 vs 整単位の構造**

開始位置を 0 とすると、位置の進行：
- 0, 2, 8, 18, 32, 50, ...（差分：2, 6, 10, 14, 18, ... すべて偶数）

**位置は常に 2 の倍数**で、メッシュ整合性が自動的に保たれる。

### 動力学のシミュレーション構造

```
For each particle:
    while simulating:
        compute next position based on (δt_i, σ_i, R_i)
        check: is new position on the 4D integer mesh?
        check: is move pitch a multiple of 2?
        if multiple particles at same mesh point:
            trigger interaction event
        update particle state
```

**離散イベント駆動シミュレーション**（DES）として実装可能。大域時計は不要。

### 重力時間遅れの幾何学的起源

| 粒子 | R | δt | 物理的描像 |
|---|---|---|---|
| 重い粒子（トップ） | 大 | 大（ゆっくり） | 「ゆっくり生きている」 |
| 軽い粒子（電子） | 小 | 小（速く） | 「速く生きている」 |
| 光子 | 特殊 | ? | 固有時 = 0 |

重い粒子の近くで時間がゆっくり進む = 重力時間遅れ。**等価原理が自動的に成立**。

### 残る本質的問題

1. **「世界線の交差判定」**：2 粒子が同じメッシュ点に到達する条件
2. **相互作用ルール**：メッシュ点が一致した時に何が起こるか（[W11] で定義）
3. **光子の δt**：固有時ゼロをどう扱うか

これらは「同期問題」ではなく「**相互作用ルールの問題**」。動力学の基本構造は同期不要で完成している。

## 統合：気づき 8〜11 の動力学的含意

| # | 気づき | 動力学への含意 |
|---|---|---|
| 8 | k = 1/m（純幾何） | 重い粒子は小さい波数、ゆっくり位相回転 |
| 9 | δt² = 2R·x、最小 x = 2 | 半プランク単位の自然単位系、整数演算で閉じる |
| 10 | R → 0 で 540° 漸近、4D 構造保持 | 特異点問題の幾何学的解決、720° 範囲内で正則 |

これで **動力学の実装の基本構造**が明確化：

- 各粒子は独自の δt で動く
- 位置は 4D メッシュ上の整数点
- 移動ピッチは半プランク単位の偶数倍
- 相互作用はメッシュ点の共有として定義
- 大域時計不要、Lamport 的因果順序で動作

これは **離散物理学の自然な定式化**として、量子場理論・一般相対論を統合するシミュレーション基盤になりうる。

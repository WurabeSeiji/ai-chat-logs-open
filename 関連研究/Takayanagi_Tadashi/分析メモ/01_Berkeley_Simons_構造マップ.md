## 5セクション分類

### 1. モチベーション（なぜこの研究が必要か）

**該当スライド**: #4

#### 提起される問題
- ブラックホールのエントロピー $S_{BH} = A_{BH}/(4G_N)$ という公式がある
- 量子スピン系のエンタングルメント・エントロピー $S_A$ にも面積則がある
- 両者とも「面積に比例」という共通の性質を持つ
- **両者は本当に同じものなのか？** （スライド中の `?` マーク）

#### 私の理解
このスライドは「重力時空の情報量と量子物質の情報量の対応はあるのか？」という問いを設定している。聴衆（量子情報・計算複雑性の研究者）に対し、「あなたたちの専門である量子もつれが、重力理論と関係があるんですよ」という導入になっている。

---

### 2. 基礎知識の復習

**該当スライド**: #5, #6, #7, #8, #9, #10, #11, #12

#### 各スライドが復習する内容
- **#5 EEの定義**: $S_A = -\mathrm{Tr}_A[\rho_A \log \rho_A]$（von Neumann entropy）。簡約密度行列 $\rho_A = \mathrm{Tr}_B|\Psi\rangle\langle\Psi|$ を経由する。
- **#6 EEの操作的意味**: LOCC（局所操作と古典通信）で N 個のBell対に蒸留できる純粋状態の数として理解できる。$S_A = \lim_{M\to\infty} N/M$。
- **#7 重力ホログラフィー**: 't Hooft (1993), Susskind (1994)。「重力の自由度は体積でなく面積に比例する」という標語。
- **#8 AdS/CFT**: Maldacena 1997。$D+1$ 次元 AdS 重力 = $D$ 次元 CFT。
- **#9 Ryu-Takayanagi 公式（Ver.1）**: 静的時空で $S_A = \min_{\Gamma_A \approx A}[A(\Gamma_A)/(4G_N)]$。極小曲面の面積で EE を計算する。
- **#10 HEE の振舞い**: 面積則発散項、奇数次元では普遍量 $F$、偶数次元では中心電荷（共形アノマリー）に一致する対数項が現れる。
- **#11 強劣加法性**: $S_{AB} + S_{BC} \ge S_{ABC} + S_B$、$S_{AB} + S_{BC} \ge S_A + S_C$。情報理論的な性質が幾何学から幾何学的に証明できる。
- **#12 HRT 公式（Ver.2）**: ローレンツ的時間依存時空への拡張。$S_A(t) = \min_{\Gamma_A}\mathrm{Ext}_{\Gamma_A}[A(\Gamma_A)/(4G_N)]$。極値化（極小ではない）。

#### 私の理解：このセクションの構造的特徴
**Ver.1 → Ver.2 という「バージョン進化」の見せ方**が重要だと感じた。次の中心的主張で「**Ver.3** ホログラフィックエントロピー公式は何か？」と問われ、それが Pseudo Entropy だ、という展開になる。スライドの章立てが「数式の連続的な拡張」として組まれている。

### 3. 中心的主張（Pseudoentropy が新しい量である）

**該当スライド**: #13, #16, #17, #18, #19, #20, #21

#### 主張の論理構造

**【ステップ1】量子もつれが時空を作る**（#13）
- ホログラフィック EE 公式から、Planck長 $l_P \approx 1.6 \times 10^{-35}$ m の面積ごとに qubit のBell対が1個存在することになる
- 1cm² で約 $10^{65}$ qubits（！）という具体的な数字
- スライドのキャッチコピー: "Spacetime may emerge from entangled Qubits! → Tensor Network (TN) realizes this idea!"

**【ステップ2】テンソルネットワークが具体例**（#14, #15）
- MERA, HaPPY model, Random TN, 経路積分最適化など、複数の TN モデルで AdS の幾何学的構造が実現できる
- 経路積分最適化では「計算コスト最小化が重力に至る」という結論

**【ステップ3】「Ver.3 公式」の問い**（#16, #17）
- AdS の時間スライスは Euclidean (非ユニタリー) → Path Integral
- Comment: 「AdS/CFT における創発する空間は **非ユニタリーゲート** に関係しているように見える」
- 「**ユークリッド時間依存的漸近 AdS 空間における極小面積 = CFT における何の量子情報量か？**」
- **答え: Pseudo Entropy！**

**【ステップ4】Pseudo Entropy の定義**（#18）
- 2つの量子状態 $|\psi\rangle$ と $|\varphi\rangle$ から**遷移行列** $\tau^{\psi|\varphi} = |\psi\rangle\langle\varphi| / \langle\varphi|\psi\rangle$ を作る
- 簡約遷移行列 $\tau_A^{\psi|\varphi} = \mathrm{Tr}_B[\tau^{\psi|\varphi}]$
- Pseudo Entropy: $S(\tau_A^{\psi|\varphi}) = -\mathrm{Tr}[\tau_A^{\psi|\varphi} \log \tau_A^{\psi|\varphi}]$
- 通常のエンタングルメント・エントロピーの**初期状態 = 終状態という制約を外した一般化**

**【ステップ5】基本性質**（#19）
- 一般に**複素数値**（$\tau_A$ がエルミートでないため）
- $|\psi\rangle = |\varphi\rangle$ なら通常の Renyi entropy に戻る
- $S(\tau_A) = S(\tau_B)$（"SA = SB"、$A$ と $B$ で値が等しい性質）

**【ステップ6】量子情報的意味づけ**（#20）
- Pseudo Entropy = "modular operator" の弱値（Aharonov-Albert-Vaidman 1988）として解釈できる
- 量子論で**ポストセレクション (post-selection)** を考えるときに遷移行列が現れる
- 弱値は一般に複素数値

**【ステップ7】ホログラフィック公式 Ver.3**（#21）
- $S(\tau_A^{\psi|\varphi}) = \min_{\Gamma_A}[A(\Gamma_A)/(4G_N)]$
- Euclidean 時間依存 AdS で、極小曲面の面積を $4G_N$ で割る公式が**そのまま使える**
- 性質: $\rho_A$ が純粋状態なら 0、$\psi$ または $\varphi$ がエンタングルしてないなら 0、$S_A = S_B$

#### 私の理解：中心的主張のポイント
- 通常の EE（Ver.1, Ver.2）は「**1つの状態の中での部分系のもつれ**」を測る
- Pseudo Entropy（Ver.3）は「**初期状態と終状態が異なるとき、間にある状態のもつれ**」を測る
- ホログラフィー側の幾何公式は同じ形（極小面積/4GN）で書ける
- ただし**遷移行列はエルミートでないので、結果は複素数値になりうる**

### 4. 具体例・計算例

**該当スライド**: #22, #23, #24, #25, #26, #27, #28

#### スライド毎の具体例
- **#22 PE = エンタングルメント蒸留**: 同時対角化可能な2状態の場合、PE は中間状態の Bell 対の数として解釈できる
- **#23 SVD entropy**: PE を改良し**実数で非負**にした量。$S_{SVD}(\tau_A) = -\mathrm{Tr}[|\tau_A|\log|\tau_A|]$（$|\tau_A| \equiv \sqrt{\tau_A^\dagger \tau_A}$）。Parzygnat-Taki-Wei-TT 2023。
- **#24, #25 量子相転移**: 異なる相にある2状態の間の PE を取ると、$\Delta S = \mathrm{Re}[S(\tau^{1|2})] - (S(\rho^1) + S(\rho^2))/2$ が**正**になる（同相内なら負）。横磁場イジング模型で数値検証。
- **#26 Heuristic Interpretation**: 異なる量子相の界面に gapless phase（CFT！）が現れ、それが PE を増強する。トポロジカル PE という概念にもつながる（Nishioka-Taki-TT 2021）。
- **#27 Entanglement Phase Transition**: 測定誘起相転移をホログラフィックモデルで記述。End-of-the-world brane（境界ブレーン）と非ユニタリーダイナミクスを組み合わせる。
- **#28 Double Wick Rotation**: $\tau = it$ という二重 Wick 回転で時間発展を導入。3つの相（BTZ phase、Poincaré AdS phase、Thermal AdS phase）が現れる。

#### 私の理解：具体例の選び方
講演者は**「PE が単なる数学的構成ではなく、実際の物理現象を捉える」**ことを示すため、量子相転移、測定誘起相転移、トポロジカル相という現実の凝縮系物理の文脈で計算例を提示している。聴衆である量子情報研究者が興味を持ちやすい題材を選んでいると感じた。

### 5. 未解決問題（今後の方向性）

**該当スライド**: #29, #30

#### #29 結論で挙げられた到達点
- PE は entanglement entropy の一般化
- PE は初期状態と終状態の両方に依存
- PE は一般に複素数値
- 対角化可能な状態で、PE は中間状態のもつれの量を測る
- AdS/CFT で PE = 極小曲面の面積 → **PE からの空間の創発**
- PE は量子相の差を検出できる → **新しい量子秩序パラメータ**

#### #30 Future directions（未解決問題）
1. **PE の複素数値の量子情報的意味は何か？**
   - 「重力公式が EE は QI で有用と示唆するなら、PE も同様かもしれない」
   - "pseudo pseudo entanglement" という新概念の可能性？（暫定的な造語）
2. **量子重力（特に de Sitter 空間を含む）への含意は？**
   - **PE の虚部 = ホログラフィーにおける時間の創発？**
   - 引用: Doi-Harper-Mollabashi-Taki-TT, PRL 130 (2023) 031601

#### 私の理解：未解決問題の構造
1番目（複素数値の意味）は「数学的形式は分かったが、何を測っているか分からない」という根本問題。
2番目（時間の創発）は「もしこれが正しいなら、量子重力の根本機構を握る」という予想であり、まだ予想段階。

「**重力理論と量子情報のつながりを使って、量子重力の核心に迫る**」――これがこの研究プログラム全体の旗印だと、私には読めた。

## 学習者としての所感（W8等への言及はしない）

- 数式の量は多いが、論理の骨格は意外とシンプル：「EE → HEE → PE」という3段階。
- Pseudo Entropy という言葉は最初「擬似的なエントロピー」と読んでしまうが、「**遷移行列のエントロピー**（pseudo = transition matrix のエルミート性が壊れた状態の意味）」と理解した方が誤解が少ない。
- 「複素数値」は数学的には自然な拡張だが、それが**物理的に何を意味するか**は本講演でも未解決として残されている。これは私のような素人にも追える「未解決問題」だと感じた。

**要旨**

本論文は、プランク長に等しい直径を持つ4D空間球に基づく幾何学の観点から、物理的宇宙における対称性を検討する。これらの球を通じて、粒子の基本的性質とそれらが生成する場との間の関連が提案され、質量、エネルギー、運動量および電荷の性質が、空間の回転と曲率の発現として現れることが強調される。

また、量子力学と相対性理論の文脈における、これらの対称性の含意が検討され、より深い幾何学的枠組みのなかで概念を統一する可能性が考慮される。これらのアイデアの探究は、質量、エネルギー、および粒子間の基本的相互作用の本質について、新しい視点を提供し得るものである。

**2.1 はじめに**

対称性は物理学において根本的な役割を果たしており、基本的相互作用の振る舞いを支配する保存則を確立している。ネーターの定理は、各対称性に保存則が対応することを確立しており、これにより時間と空間を通じた物理法則の不変性を説明する整合的な理論的枠組みの発展が可能となった。

古典物理学においては、エネルギー保存、線形運動量保存および角運動量保存は、それぞれ時間変換、空間変換および回転変換に対する方程式の不変性から生じる。量子力学と場の理論においては、この概念はレプトン数、バリオン数およびCPT対称性（電荷・パリティ・時間）のような新しい対称性と保存則に拡張される。しかし、特定の条件下では、これらの法則の一部が部分的に破れることがある：

- **エネルギーの一時的な破れ**：非常に短い時間スケールにおいて、不確定性原理はエネルギー保存の一時的な破れを許容する。
- **パリティ（P）の破れ**：量子力学における一部の弱い相互作用はパリティ対称性を破る。これは、特定の粒子がその空間座標が反転された場合に同じ振る舞いをしないことを意味する。
- **CP対称性の破れ**（電荷共役とパリティ）：特定の過程において、電荷反転とパリティの組み合わせ（CP）は保存されない。
- **CPT対称性の破れ**（電荷共役・パリティ・時間）：電荷反転・パリティ・時間の組み合わせ（CPT）は保存されると期待されるが、標準模型を超える理論においては、非常に小さな破れの可能性が考慮されている。

本章では、静止および運動しているエネルギー要素間に存在する対称性を扱う。静止したエネルギーの各要素（質量、空間または曲率、時間または振動数、および電荷）は、それ自体がエネルギー要素（加速度、エネルギー、運動量、および電場）である場を生成し、静止エネルギー要素とそれが生成する場の積が力を生み出す。

- 曲率 ─ エネルギー
- 振動数 ─ 運動量
- 加速度 ─ 質量
- ？ ─ 電荷

しかしながら、疑問が生じる：他の場合とは異なり、電荷は静止状態でも運動状態でも、特定のエネルギー要素に直接関連しているようには見えない。これが本論文で取り扱う中心的な問題を提起する：なぜ電場と磁場には直接関連するエネルギー要素がないように見えるのか？

**2.2 質量─加速度の対称性**

ニュートンの理論では、質量は慣性質量と重力質量という二つの異なる概念で提示される。慣性質量は、物体の運動変化に対する慣性または抵抗を表す基本的な物理量である。それは粒子の固有の性質である。慣性質量は、物体の物質の量として定義される。同じ体積を占める物体に同じ力を加えると異なる加速度が生じるため、それらを区別するために慣性質量の概念が作られた。一方、重力質量は、重力によって他の物体を引き付ける性質である。重力質量はニュートンの万有引力の法則から定義される。同じ体積を占める物体でも、重さが異なるものがあることが観測され、それらを区別するために重力質量の概念が作られた。結局のところ、我々が質量と呼ぶものを測定する二つの異なる方法があるということである。

両方の概念は物体の同じ性質を記述するが、異なる文脈から生じる。慣性質量はニュートンの第二法則を通じて測定される：

F⃗ = mᵢa u⃗ᵣ    (2.1)

ここで mᵢ は物体の慣性質量、F は加えた力、a は結果として生じる加速度である。一方、重力質量は物体間の万有引力の法則から定義される。第一の物体の重力質量を m、第二の物体の重力質量を mₘ とすると、それらの間の重力は次のようになる：

F⃗_G = G (mmₘ/r²) u⃗ᵣ    (2.2)

ここで G は万有引力定数、r は物体間の距離である。この力をニュートンの第二法則の式と等置すると：

a = (mₘ/mᵢ)(G m/r²)    (2.3)

実験的に mᵢ と mₘ が等しいことが確認される。ニュートンは根本的な問いを提示した：概念的には異なる現象を記述するのに、なぜ mᵢ と mₘ は必然的に等しくなければならないのか？

**一般相対性理論と質量─加速度の対称性**

一般相対性理論は等価原理に基づいており、慣性質量と重力質量は区別不可能であると仮定する。これは、加速度によって生じる慣性力が重力に類似していることを意味する。両者が加速度に関連する場によって生成されることから、質量と加速度の間の根本的な関連が確立される。

万有引力の法則は、Aᵢⱼ = G の場合の式(1.41)から得ることができる。質量が作る重力場は次のように表される：

g⃗ = G(m/r²)u⃗ᵣ    (2.4)

その単位は加速度である。

**慣性場と慣性質量**

慣性質量は、力が加わった物体の加速度によって生成される質量場として解釈できる。加速度（運動中の振動数）をエネルギー要素として考えると、その相互作用は次のように記述できる：

s⃗ᵢ = v⃗/t → λᵢ = λₐ = c²/a    (2.5)

ここで λᵢ = λₐ は運動に関連した波長である。式(1.42)を考慮すると：

Aᵢⱼ = G(ℏ²/c²)AᵢAⱼ = G(ℏ²/c²)(1/c²)(1/c²)    (2.6)

(1.43)に代入すると：

F⃗ₐ = (Gℏ²/c⁶)(a⃗×a⃗'/r²)    (2.7)

場は単位加速度（a' = 1）に対して与えられるため：

S⃗ₐ = (Gℏ²/c⁶)(a⃗/r²) = (ℏ/c³)(lₚ²/r²)a⃗    (2.8)

ここで lₚ はプランク長であり、単位加速度に関連する場を表す。プランク条件では：

S⃗ₐₚ = (ℏ/c³)a⃗ₚ    (2.9)

単位加速度によって生成される二次場（Sₐ）は質量の単位を持つ。したがって、加速度 a を持つ質量 m は、加速度による質量場を生成し、その値は：

m̃ᵢ = ℏ/(λₘc) = (ℏ/c³)(lₚ²/r²)a⃗    (2.10)

慣性質量は、力を加えた際にあらゆる物体が経験する加速度によって生成される質量場または慣性場によるものである。二つの同一の物体があると仮定する。mₘ 上に重力場を生成する慣性質量を mᵢ と呼ぶ。万有引力の法則を適用し、(2.10)を考慮すると：

F⃗ = G(mᵢmₘ/r²) = G(ℏ/c³)(lₚ²/r²)(mₘ/r²)a⃗ = (lₚ⁴/r⁴)mₘa⃗    (2.11)

極限では r=lₚ（物体が接触している場合）であるから：

F⃗ = mₘa⃗    (2.12)

したがって、慣性は物体の加速度による重力現象と見なすことができる。プランク条件では、質量場は空間の力学と粒子の性質を結びつける根本的な現象となる。Masreliezによれば、加速度の結果としてミンコフスキーの計量係数が変化すれば、慣性は重力効果としてモデル化できる。

「一般相対性理論によれば、慣性質量と重力質量は等価である。したがって、慣性加速度は、重力加速度と同様に、一般相対性理論の線要素における変化する計量係数と関連付けることができる。この可能性を調査する。宇宙論的な参照枠が存在するならば、ミンコフスキーの線要素は速度に依存するスケール因子によって修正でき、すべての加速運動は時空の測地線上の運動となる。これにより、慣性を重力型の現象としてモデル化できる可能性がある。」（Masreliez, 2006）

アインシュタインは、時空すなわち「新しいエーテル」が慣性の基準となると確信していた（Kostro, 2000）。

いわゆる慣性力は、非慣性系の観測者にとっての見かけの力または虚の力である。慣性力は、力を加えて加速する物体に対して質量 m の物体が経験する重力、すなわち質量 m' の物体の加速・減速によって生成される質量場で与えられ、加速度または減速度と同じ方向で反対の向きを持つ。実際に、式(2.10)を重力の式に代入すると：

F_G = G(mm'/r²) = G(m/r²)(ℏ/c³)(lₚ²/r²)a = (rₚ/r)⁴ ma    (2.13)

二つの物体が接触しているとき、r=rₚ であるから：

F⃗ᵢ = ma⃗    (2.14)

したがって、これは虚の力ではなく、むしろ実在の力であり、質量の存在によって生成される重力場と、加速度の作用による質量場または慣性場との間に存在する対称性に起因し、力は両方の場の積である。結局のところ、加速度は慣性場（慣性質量）を生成し、一方で質量は重力場を生成する。

m⃗ ↔ a⃗    (2.15)

したがって、「一様な重力場と一様に加速された運動を区別することは不可能」である。なぜなら、各々が他方を生成するからである。これは、アインシュタインが特殊相対性理論を定式化する際に拠り所とした等価原理と呼ばれる原理の一つである。

ニュートンの第二法則は、前章で見た重力の一般方程式（式(1.33)）から、質量と加速度の間の力として得ることができる。静止質量の波長は λₐ=h/mc であり、加速度は λⱼ=vc/a と表せるので、(1.33)に代入すると：

F_{m-a} = G(ℏ²/c²)(mc/ℏ)(a/vc)(1/r²) = G(ℏ/c³)(ma/r²)    (2.16)

重力質量 m の物体に力を加えると、それは光速で伝達される。一方、二つの物体が接触しているとき、r = λₚ = √(Gℏ/c³) であり、式(2.16)に代入すると：

F = ma    (2.17)

**物理的解釈**

重力場と慣性場の等価性は、一様な重力場と一様に加速された運動を区別することが不可能であることを意味する。この原理は一般相対性理論の定式化の基礎にあり、質量という性質が時空と加速度の相互作用から生じることを示唆している。

結論として、質量と加速度の間の対称性は、古典的概念と相対論的概念を統一し、時空の力学がいかにして物質の基本的性質を生み出すかについて整合的な見方を提供する。

**結論**

質量が物質の固有の性質であるという従来の解釈とは異なり、このアプローチは質量が時空の加速の結果であることを示唆している。これは重力と慣性が同一の幾何学的原理の発現であるという考えを補強し、基本的な力の統一の可能性への道を開く。

**2.3 運動量─振動数の対称性**

運動中のエネルギー要素は今や運動量（p）であり、質量と速度の積(1.28)として定義される。数学的には次のように表される：

p = s⃗ᵢ = mv⃗ → λᵢ = λₘᵥ = ℏ/sₘ    (2.18)

ここで Sᵢ は運動量（運動中の質量）、m は質量、v は物体の速度、λᵢ = λₘᵥ は運動に関連した波長である。この運動量は、粒子の運動に関連する動的エネルギーを特徴づける。

**運動量に関連する力**

運動量 p と p' を持つ二つの粒子間の相互作用は、重力に類似した力として記述でき、Aᵢⱼ=G/c² の場合の式(1.43)で与えられる。

F⃗ₘᵥ = (G/c²)(p⃗×p⃗'/r²)    (2.19)

ここで G は万有引力定数、c は光速、r は粒子間の距離である。この関係は、運動量が距離に依存する関連場を生成することを示唆している。

**運動量に関連する場**

単位要素（p'=1）によって生成される場は、次の式で記述される：

υ⃗ = (G/c²)(p⃗/r²)    (2.20)

ここで υ は振動数の単位（s⁻¹）を持ち、プランク条件で r=lₚ（プランク長）の場合、場は簡略化された形をとる：

p = mₚc  かつ  Gmₚ = lₚc² → Gp = lₚc³    (2.21)

式(2.20)に代入すると：

υ = 1/tₚ    (2.22)

ここで tₚ はプランク時間、υ は運動量によって生成される重力磁気場または振動数場である。したがって、力は次のようになる：

F⃗ = mv⃗ × υ⃗ = mv⃗ × (G/c²)(p⃗/r²)    (2.23)

**振動数に関連する場**

一方、振動数の逆数(1.27)を静止エネルギー要素として考えると、運動量場を導出できる。式(1.41)と(1.42)から、Aᵢ=Aⱼ=1/c として：

F⃗ₜ = G(ℏ²/c⁴)(1/tt')(1/r²)u⃗ₜ    (2.25)

したがって、単位要素（t'=1）に対する運動量場は次のようになる：

P⃗ = G(ℏ²/c⁴)(1/t)(1/r²)u⃗ₜ    (2.26)

これは確かに運動量の単位を持つ。この結果は根本的な対称性を確立する：運動中の粒子の運動量は振動数場を生成し、一方で振動数は運動量場を生成する。プランク条件では：

Pₚ = ℏ/(ctₚ) = mₚc    (2.27)

したがって、運動中の質量は振動数の単位（s⁻¹）を持つ場を生成し、時間の逆数（振動数）はエネルギー要素として考えた場合、運動量の単位を持つ場を生成する。これは、線形運動量によって生成される場の振動数を区別することは不可能であり、逆もまた同様であることを示す。これは等価原理を運動量と振動数に適用したものに他ならない。

mv⃗ ↔ 1/t    (2.28)

したがって、力を両方の場またはエネルギー要素の積として表すことができる。

F⃗ = mv⃗/t    (2.29)

**結論**

物体にエネルギーを加えると、その速度が増加し、したがって運動量が増加する。これは次に、物体を構成する粒子の回転周期（内部振動数）とそれに関連する波長の増加を引き起こし、運動量と振動数の間の直接的な関係を示す。この原理は、粒子の運動を場と振動数の観点からの記述と動的に結びつける。一方、この内部振動数の増加は物体のサイズを減少させ、その波長が減少し、結果として物体の質量が増加する。

光子の回転（ω）は運動量（p）を生成し、これにより光子は常に光速で移動する。なぜなら、運動量は回転に比例するからである（p = ωℏ/c）。

**2.4 曲率─エネルギーの対称性**

空間の曲率は、特性長の逆数として理解され、エネルギー要素として解釈できる（式(1.26)）。数学的には、次のように表される：

eᵢ = 1/λ → λᵢ = λ    (2.30)

ここで eᵢ はエネルギー要素または曲率であり、λ バーは当該空間に関連する波長である。このアプローチは、空間の曲率がエネルギー場と直接的な関係を持つことを示唆している。

**曲率に関連するエネルギー場**

曲率によって生成されるエネルギー場は、Aᵢ=Aⱼ=1 の場合の式(1.41)と(1.42)から得られる：

F⃗ᵢⱼ = G(ℏ²/c²)(1/λλ')(1/r²)u⃗ᵣ    (2.31)

場は次のようになる：

E⃗ = G(ℏ²/c²)(1/λ)(1/r²)u⃗ᵣ    (2.32)

これはエネルギーの単位を持ち、プランク条件ではプランクエネルギーが得られる：

Eₚ = ℏc/λₚ    (2.33)

したがって、空間の曲率はエネルギーと呼ばれる場を生成する。上の方程式は曲率によるエネルギー場の値を与える。

式(1.41)と(1.42)から、Aᵢ=Aⱼ=1/ℏc として：

F⃗_E = (G/c⁴)(EE'/r²)u⃗ᵣ    (2.34)

これは任意の二つのエネルギー形態間の力を与える。場は次のようになる：

ρ⃗ = (G/c⁴)(E/r²)u⃗ᵣ    (2.35)

これはメートルの負の一乗（m⁻¹）の単位を持ち、プランク条件では：

ρₚ = 1/λₚ    (2.36)

したがって、エネルギーは空間の曲率を生成する。式(2.35)は、エネルギー E から距離 r における空間の曲率を与える。エネルギーは質量だけでなく電荷にも起因し得る。

E⃗ ↔ 1/r⃗    (2.37)

これにより、力を両方の要素または場の積、すなわちエネルギーと曲率の積として表すことができる：

F⃗ = E/r⃗ = ρ⃗E    (2.38)

**物理的解釈**

アインシュタインの一般相対性理論は、時空がいかにして物質によって曲げられるか、また時空の曲率がいかにして重力を生み出すかを記述している。式(2.32)と(2.35)の解釈はやや異なる。「エネルギーは時空の曲率に起因し、逆にエネルギーは時空を曲げる」。ただし、これは粒子または粒子の集合体が占める時空についてである。この定式化において、曲率はエネルギーの結果であるだけでなく、その発現でもある。この対称性は、空間と粒子の基本的性質が相互に関連しているという考えを補強する。

等価原理を適用すれば、エネルギーを空間の曲率によって生成される場と区別することは不可能であるということになる。

**結論**

曲率とエネルギーの間の対称性は、両方の概念が深く絡み合っていることを示している。空間の曲率はエネルギーの一形態として解釈でき、一方でエネルギーは空間の幾何学を変える力を持つ。この統一原理は、より根本的な視点から重力的および動的現象を探究するための概念的基盤を提供する。

**2.5 電荷─電場の対称性**

電場は、すべての電荷または帯電した物体に力が作用する空間の領域である。電荷は物質の固有の性質である。電荷は正または負であり得るため、他の帯電粒子に対する力は引力または斥力となる。ある点における電場の強度 E は、その点に置かれた単位電荷に作用する力として定義される。電場は、電気的性質を持つ物体やシステム間の相互作用を記述する。この場は、電荷の存在によって生成されるエネルギー要素と考えることができる。数学的に、電場中の電荷に作用する力は次のように表される：

F⃗ = qE⃗    (2.39)

ここで F は電荷に作用する力、q は電荷の大きさ、E は電場の強度である。

**電荷に関連する場**

以前の対称性と同様に、場の定義そのものにより、エネルギー要素と場の積が力を生じる。一方、電場をエネルギー要素として考えることもでき、それが今度はクーロン単位で測定される場を生成する。

違いは、今や空間や時間との直接的な関係がないということであり、それが静止状態であれ運動状態であれ同様である。これは、電荷と呼ぶ物質の固有の性質が、何らかの形で時空と関連しているはずであることを示唆している。なぜなら、電荷はエネルギーのもう一つの形態だからである。

電荷の波長は式(1.25)で与えられる：

λ_q = √(G/K)(ℏ/qc)    (2.40)

そして係数 Aᵢ は式(1.36)により次のようになる：

A_q = √(K/G)(c/ℏ)    (2.41)

(1.42)に代入すると：

A_{qq'} = K    (2.42)

最後に、この値を力の一般方程式(20.41)に導入すると、クーロンの法則が得られる：

F⃗_{qq'} = K(qq'/r²)u⃗ᵣ    (2.43)

これにより電場は（q'=1 として）：

E⃗ = K(q/r²)u⃗ᵣ    (2.44)

**電荷と場の間の対称関係**

電場は電荷によって生成されるだけでなく、エネルギー要素としても考えることができ、それが今度は場を生成する。これは、自然界で観察される他の対称性と同様の、電荷と場の間の双方向的な関係を示唆する。したがって、その波長は式(2.44)から得られる：

λ_Ē = K(q/Ēr)    (2.45)

力の一般方程式(20.41)に代入すると：

F⃗ = G(ℏ²/c²)(Ēr/Kq)(Ē'r/Kq')(1/r²)    (2.46)

場は次のようになる：

Q⃗ = G(ℏ²/c²)(E⃗/Kq)(1/Kq')(1/r²)    (2.47)

プランク条件では：

Q⃗ₚ = G(ℏ²/c²)(1/KQₚ)(1/KQₚ)(1/lₚ²) = (G/K)(ℏ²/c²)(1/Qₚ²)(1/lₚ²) = ℏc/(KQₚ)    (2.48)

その単位はクーロンであり、プランク電荷の定義と一致する。電荷と電場の間の対称性は新しいものをもたらさない。なぜなら、電荷はクーロンで表され、原理的には時空との関係が存在しないように見えるからである。興味深いのは、重力場、線形運動量場、エネルギー場の場合のように、電場をメートルと秒の単位で表すことができるかどうかである。1アンペアの電流（I=1 A）の場合、クーロンは秒と一致する。これは以前に見たような変数変換に他ならない。

**結論**

電荷と電場の間の対称性は、粒子の基本的性質がそれらが生成する場との相互作用から生じるという考えを補強する。この関連性は、より根本的な原理の観点から電磁的相互作用をより深く理解するための基盤を提供する。

**2.6 対称性のまとめ**

エネルギー要素とそれらが生成する場の間の対称性の分析により、基本的相互作用の記述における統一的枠組みを確立することができる。以下にこれらの関係をまとめる：

1. **曲率─エネルギー：**
   - エネルギー要素：長さの逆数（1/λ）。
   - 生成される場：エネルギー（E=ℏc/λ）。
   - 力：エネルギーと曲率の積。

2. **振動数─運動量：**
   - エネルギー要素：振動数（1/t）。
   - 生成される場：運動量（P=ℏ/(tc)）。
   - 力：運動量と振動数の積。

3. **加速度─質量：**
   - エネルギー要素：加速度（a）。
   - 生成される場：質量（Sₐ=ℏc/(a·λₚ²)）。
   - 力：質量と加速度の積。

4. **電荷─電場：**
   - エネルギー要素：電荷（q）。
   - 生成される場：電場（E=(1/(4πε₀))(q/r²)）。
   - 力：電荷と電場の積。

表2.2 エネルギー要素と生成される場：

| エネルギー要素 | | 単位 | 場 | 単位 | 力 |
|---|---|---|---|---|---|
| 曲率 | 1/λ | m⁻¹ | E（エネルギー） | J | E/λ |
| 振動数 | 1/t | s⁻¹ | P（運動量） | kg m s⁻¹ | mv/t |
| 加速度 | a | m s⁻² | M（質量） | kg | ma = mg |
| 電荷 | q | C | E（電場） | V m⁻¹ | qE |
| 運動中の電荷 | qv | C m s⁻¹ | B（磁場） | V m⁻² s | qvB |

各エネルギー要素は前のものに光速を掛けて得られる一方、場は前のものを光速で割って得られることに注目されたい。ただし電荷については、他とは独立に現れるように見える。

**一般的解釈**

各エネルギー要素は関連する場を生成し、要素とその場の相互作用が力を生成する。この対称的構造は物理法則の記述を簡素化し、異なる基本的物理量の振る舞いを整合的な体系の下に統一する。

対称性は優雅な概念的枠組みを提供するだけでなく、質量、電荷、エネルギーなどの粒子の性質が、時空のより深い力学の発現であることを示唆している。

運動中の電荷と磁場の間の対称性も新しいものをもたらさない。なぜなら、電荷は時空とは独立にクーロンで表されるからである。

一方、力をエネルギー要素とそれが生成する場の積として表すことにより、量子力学に現れるような異なる保存則の存在は不要となる。このようにして、重力とクーロン力を同じ式で表すことができる。

「したがって、質量とエネルギーは本質的に同じものであり、同じものの異なる表現に過ぎない。物体の質量は定数ではない。そのエネルギーの変化に応じて変化する。」（Felker, 2004）──アルベルト・アインシュタイン

質量とエネルギーが本質的に同じものであるだけでなく、質量も電荷も空間も時間もエネルギーの異なる発現であり、すべてが相互に交換可能である。例えば、電子と陽電子の衝突において、質量を生み出す電子の回転は光子の回転に変わる。これはエネルギーの交換（mc² = ℏω）ではなく、エネルギーの表現形態の変化である。空間における回転は変化しないが、電荷を生み出す第四次元の回転はプランク回転に達するまで増加する。しかし、我々の視点からは、回転は反対方向であるため互いに打ち消し合い、これが電荷保存則を生み出す。

**2.7 エネルギー要素：速度×加速度**

回転する球では、球の各点は加速度と速度を受けるが、その両者の積も受ける。したがって、速度と加速度の積も新しいエネルギー要素となり得ると考えられる。数学的定式化は次のように始まる：

eᵢ = v · a = c³/λ    (2.49)

ここで eᵢ はエネルギー要素、v は速度、a は加速度、λ バーはシステムに関連する特性長である。この関係から次が得られる：

λᵢ = v · a = c³/eᵢ    (2.50)

**速度×加速度の積に関連する力**

この関係を重力の一般方程式（式(20.33)）に代入すると：

Fᵢⱼ = (Gℏ²/c²)(eᵢ/v³λⱼ)(1/r²)    (2.51)

ここで F は力、G は万有引力定数、r は考慮する点間の距離である。単位要素に対する関連場は次のように定義される：

Q = (Gℏ²/c²)(1/v³λⱼ)(1/r²)    (2.52)

この場の単位は kg m⁻¹ s である。基本的な値に達するプランク条件では：

Qₚ = (ℏ/c²)(1/λₚ) = mₚ/c    (2.53)

ここで mₚ はプランク質量、Qₚ はプランク電荷、c は光速である。

式(3.37)から次が得られる：

mₚ/c = √(μ₀/4G)Qₚ    (2.54)

質量の単位としてのキログラムが任意であるのと同様に、電荷の単位としてのクーロンも任意であることを考慮すると、重力場と磁場の定数は、速度で割った静止質量を電荷に変換する。したがって、電荷は速度と加速度の積によって生成される場である。この場に、任意に選ばれた単位に適合させるため、重力定数と磁気定数を代数的に掛ける必要がある。

**電荷との関係**

これらの考慮から、速度×加速度の積によって生成される場は、電荷と関連付けることができる。質量の単位を普遍定数を用いて電荷の単位に変換すると、電荷は質量を速度で割ったもの（q = m/v）として表すことができる。

これは、電荷が本質的に速度と加速度の積によって生成される場であることを示している。重力定数と磁気定数は、これらの量を従来の測定単位に合わせて調整する。

**関連場の性質**

万有引力定数により kg を m³ s⁻² に変換でき（式(8.11)）、これにより電荷は m² s⁻¹ で与えられることになる。これらの単位は速度×加速度によって生成される場に対応する。一方、電場の単位（m² s⁻³）は、エネルギー要素としての速度×加速度の単位に対応する。

さらに、この相互作用によって生成される場は、球の回転方向に依存する性質を持つ。回転が時計回りであれば、関連する電荷は正であり、反時計回りであれば負となる。この振る舞いにより、粒子の性質をプランクスケールの4D球の回転によって生成される場として解釈することが可能となる。

**2.8 ヒッグス粒子は粒子に質量を与えない**

前節では、粒子の質量はヒッグス粒子ではなく、4D空間球の回転に関連する現象に起源を持つことを提起した。以下に、この仮説を支持する論拠を詳述する。

**ヒッグス粒子の限界**

素粒子物理学の標準模型の文脈において、ヒッグス粒子はヒッグス機構と関連付けられている。ヒッグス機構は、粒子がヒッグス場との相互作用によってどのように質量を獲得するかを説明する。しかし、この機構には以下の限界がある：

1. **ヒッグス場への依存性：** 粒子の質量はヒッグス場との相互作用から生じるが、この説明は慣性質量と重力質量の根本的な本質や、実験的に観測されるそれらの等価性の理由に取り組んでいない。

2. **時空の幾何学との非互換性：** ヒッグス機構は、質量の起源としての時空の曲率や空間の力学の役割を考慮しない。これは、質量が本質的に時空の曲率に関連している一般相対性理論に基づく理論と対照的である。

3. **ニュートリノ質量の問題：** ヒッグス粒子は多くの基本粒子の質量を説明できるが、ニュートリノの質量を直接説明することはできない。これは、追加の、あるいは異なる機構が必要である可能性を示唆している。

4. **基本粒子の質量の問題：** 粒子の質量は、その慣性または静止エネルギーを測定することにより実験的に決定される。ヒッグス粒子は粒子の質量の数値を提供するのではなく、粒子がヒッグス場との相互作用を通じてどのように質量を獲得するかを説明する。

**4D球の回転に基づく代替的説明**

本論文で提示される仮説は、粒子の質量がプランク長に対応する直径を持つ4D空間球の回転に起源を持つことを提案する。この回転は引力をもたらす質量場を生成する。以下に、この説明の要点をまとめる：

1. **回転と質量場：**
   - 4D球が回転すると、その角速度と半径の積に比例する遠心加速度が生じる。
   - 三次元の視点からは、この加速度は球の中心に向かう重力場として発現する。

2. **4Dにおける回転の対称性：**
   - 四次元空間には、空間軸を含む回転（「空間的」回転）と、第四次元を含む回転の二種類の回転が存在する。
   - 第四次元の回転が静止質量を生み出し、空間的回転が運動量として知覚される場を生成する。

3. **誘起される重力場：**
   - 4D球の回転によって生成される場は重力を担い、ヒッグス場のような外部の機構を必要とせずに、慣性質量と重力質量の概念を統一する。

**4Dにおける加速度と力**

自転する、あるいは任意の軸の周りを回転する空間球があると仮定する。その回転の結果として、いわゆる遠心力を生じる質量場が生成される。したがって、球の各点に回転軸に垂直で外向きの力が現れる。

F_cf = -ma    (2.55)

ここで m は加速度によって生成される質量場である。

**4Dの構造と回転**

出発点となる仮説は、空間がプランク長を直径とする4次元（4D）空間球で構成されており、そのうちの一部が自転しているということである。4Dでは、任意の二つの軸が他の二つの軸の周りを回転することを思い出そう。したがって、u軸の周りに3つの可能な回転がある。すなわち、xy軸、xz軸、zy軸がそれぞれu軸およびz軸、y軸、x軸の周りを回転する。もう3つの可能な組み合わせがあり、u軸と別の軸（x、y、またはz）がそれぞれyz、xz、xyの周りを回転する。

したがって、4D球は二つの回転を持つことができる。空間における回転（ωₑ）は、任意の二つの軸がu軸および別の軸の周りを回転するものである。第四次元の回転（ωᵤ）は、u軸と別の軸が回転するものである。二つの回転により、球の任意の点における力線は中心に向かう。

第四次元（u）を他の三つ（x, y, z）から数学的に区別するには、虚数単位 i を含める必要がある。したがって、4D空間における合計加速度は次のように表される：

a = d²ui/dt² = d²(i lₚ/2)/dt² = i(dvᵤ/dt) = i(d(i ωₑ lₚ/2)/dt) = i²ωᵤ²lₚ = -ωᵤ²lₚ    (2.56)

ここで ωᵤ と ωₑ はそれぞれ第四次元と三次元空間の角速度であり、lₚ はプランク長である。静止時には ωᵤ=ωₑ/2 であり、遠心力は次の形をとる：

F_cf = -m(-ωᵤ²lₚ) = mωᵤ²lₚ    (2.57)

この力は中心に向かっており、したがって引力である。我々の空間的視点からは、第四次元は虚数（ui）である。空間中の物体の加速度は加速度と反対方向の力を生成する。しかし、空間そのものの加速度（ωᵤ² · lₚ）は加速度と同じ方向の力を生成し、結果として質量と呼ばれる場を生み出す。一方、観測される力は回転する空間の動的性質に関連している。

**等価原理との関連**

慣性質量と重力質量の等価性は、この仮説から自然に導かれる。なぜなら、両者は同じ現象、すなわち時空の加速度と4D球の幾何学的性質の間の相互作用から生じるからである。

**結論**

ヒッグス粒子は標準模型の重要な構成要素であるが、質量の起源におけるその役割は再評価される可能性がある。4D球の回転に基づく仮説は、質量を量子的時空の根本的な力学と結びつける代替的説明を提供し、古典物理学と量子物理学の概念をより整合的に統一する可能性がある。

表2.3 ヒッグス粒子との基本的な相違点：

| 質量の起源 | ヒッグス場との相互作用 | 4D球の回転と時空の曲率 |
| 幾何学との関係 | 時空の幾何学を考慮しない | 質量は4D空間の曲率と加速度に結びつく |

## 概要

現代物理学では、空間次元は物理的条件から導かれると言える。カルツァ＝クライン理論やD-ブレーンがその典型例である。しかし、そのような条件によるだけでなく、物理的条件なしに既知の事実からの洞察によって空間次元について考えるべきである。本発表では、空間の次元性を根本から再考する。

---

## 1. 物理的条件の例

- 標準模型には、実験結果によって値が決定される19個の数値定数が必要である。これが、標準模型を超えた統一理論を構築しようとする素粒子物理学者の動機の一つである。
- ヘンドリック・ローレンツとジョージ・フィッツジェラルドは、マイケルソン＝モーリーの実験結果を説明するため、空間が「収縮」するという奇妙な考えを導入した――ローレンツ＝フィッツジェラルド収縮である。
- 天動説を支持するために惑星軌道に周転円が与えられた。16世紀にはコペルニクスの説よりも天動説の方が高精度であった。
- アイザック・ニュートン以前の天文学者たちは、惑星の軌道は天球上にあると信じていた。
- 3次元球面または3次元双曲面に基づく宇宙モデル。

### 束縛条件の概念

テーブルの上を転がるボール（図1）について考えよう。これはボールが跳ねなければ平面上の運動のみとみなすことができる。つまり2次元空間における運動と言える。しかし一方で、「これは2次元空間における運動である」と本当に言えるだろうか？ この考え方は解析力学の「束縛条件」の概念に基づいている。ボールの重力がテーブルからの垂直抗力と釣り合っているため、垂直方向の自由度を無視している。したがって、これは偽の2次元運動である。

地球上の我々についても同様に考えることができる。すなわち、我々は閉じた2次元空間または球面に住んでいることになる。さらに、宇宙論において我々がトポロジー的に「3次元」球面すなわち3次元球面にいる宇宙にいるという考えはどうか？ これもまた「束縛条件」の概念に基づいていることは間違いない。そうだとすれば（図2）、我々の銀河が宇宙内で運動する際に与えられる遠心力は「向心力」と完全に釣り合っていることになる。

これは、テーブルの上のボールのように、我々の銀河に「束縛条件」を与え、それゆえに4次元目の自由度を知覚できないということなのか？ 特に、この「向心力」とは何か？ それは重力、電磁力、弱い力、強い力に続く第5の力（クインテッセンス）を示唆しているのか？ さもなければ（図3）、我々の銀河は「4次元空間」に放り出されてしまう。あるいは天球のように銀河を支える何か特別なものがあるのか？

同様に、地球の自転の遠心力と重力の釣り合いが保たれなければ、我々は「宇宙空間」に放り出されてしまう。

**したがって、導入する物理的条件は少ない方が好ましいことは確かである。**

## 2. 具体的洞察

物理的条件が与えられれば、歴史的に見て、間違った考えでさえ正当化されてきた。特に高次元空間については、未知の力を仮定しなければならなかった。カルツァ＝クライン理論も余剰次元空間のコンパクト化に未知の力を必要とし、それは上述の3次元多様体宇宙モデルと同様で、微視的な天球のようなものである。したがって、空間次元とは何かを知ることが重要である。まず、この問題をより深く掘り下げよう。

### 事実

- **事実1.** 我々は、この宇宙で永久に2次元空間や平面内を移動したり閉じ込められたりしている粒子を検出したことがない。これは我々の3次元宇宙におけるD2-ブレーンについても同様であろう。
- **事実2.** 例えば、我々が5次元空間に住んでいるが変数は3つ（x, y, z）しかないとしよう。だから余剰次元空間が見えないのだ、と。しかしこの考えは間違いである。これは事実1から明らかである。
- **事実3.** 我々は地球上の2次元球面上に住んでいる。しかし、我々のうち誰が2次元空間に住んでいると信じているだろうか？

上記の事実を公理として考えよう。

### 公理1

**低次元空間は高次元空間の部分空間ではない。それらは互いに独立（disjoint）である。**

それらは互いに異なる世界である。不思議の国のアリスのような状況を想像しよう。彼女は我々の3次元世界から出発する。そこでは点は「理論的に」x, y, zの3つの変数、つまり3自由度を持つ。彼女は3次元空間の中で1次元のトンネルに移される。トンネルの中で彼女は1自由度しか持たないのだろうか？

アリスがトンネルから通常の3次元世界に戻るとき、彼女が持っている変数や自由度はどうなるのか？

答えは、アリスは1次元空間にいても3つの変数x, y, zを持っているということである。たとえ10次元空間のようなはるかに高次元の世界に行ったとしても、彼女はx, y, zの3つの変数または座標しか持たない。ただし、それは (x, y, z, 0, 0, 0, 0, 0, 0, 0) を意味するのではない。そして我々の世界に戻ったとき、1次元空間に旅した前と全く同じである。これらの問題を考察しよう。

### 低次元空間から高次元空間への点の写像

方程式 x = a は、1次元空間では点、2次元空間では直線、3次元空間では平面である（図5）。方程式は常に同じだが、グラフは異なる次元空間間で異なる。統一的な視点から議論すべきである。

### 定義1

- 1次元空間の点 x = a が任意の高次元空間に写像される場合、その点のy座標やz座標（変数）は決して与えられない。

例えば、1次元空間の x = a が2次元空間に写像されると、座標は x = a のまま保たれ、(x, y) = (a, 0) にはならない。

1次元空間の x = a が3次元空間に写像されると、座標は x = a のまま保たれ、(x, y, z) = (a, 0, 0) にはならない。

### 高次元への射影の実際

連立方程式系についてグラフの観点から再考すべきである。例えば、f(x,y,z) = x² + y² + z² - 1 = 0 と g(x,y) = x - y = 0 の連立方程式を考える。これは h₁(x,z) = 2x² + z² - 1 = 0 と h₂(y,z) = 2y² + z² - 1 = 0 という2つの方程式を同時に生成するが、これらの楕円の方程式は球 f(x,y,z) と直線 g(x,y) の交線の解ではないように見え、意味をなさないように思える。しかし、g(x,y) を3次元空間に射影された平面と考えれば、h₁(x,z) と h₂(y,z) はそれぞれ x-z平面と y-z平面に射影された方程式であることが分かる（図7）。

交線の曲線は (x,y,z) = (h₁(x,z), h₂(y,z), z) として、平面が3次元空間に射影されたものとして示される。

∴ (x,y,z) = (±√((1-z²)/2), ±√((1-z²)/2), z), -1 ≤ z ≤ 1

**注意.** 量子論の観点からは、波動関数の崩壊を示唆するかもしれない。

### 高次元空間から低次元空間への点の移動

低次元空間から高次元空間への点の写像と同様に、高次元空間から低次元空間への点の写像を定義したい。

### 定義2

- 点が低次元空間に写像される場合、元の座標の数は決して変わらない。

言い換えれば、点が高次元空間から任意の低次元空間に写像される場合、**自由度または座標の数は決して減少しない。**

### 例1（同次座標変換）

変数の数は3つ：(x, y, z)。座標をスケーリングまたは写像 f: (x, y, z) ↦ (x/z, y/z) とその逆 φ: (x/z, y/z) ↦ (x, y, z) によって変換する。すると方程式

x² + y² = z² は (x/z)² + (y/z)² = X² + Y² = 1 に変わる。

3次元空間での x² + y² = z² と「2次元」空間での (x/z)² + (y/z)² = X² + Y² = 1 の関係は、双曲面 x² + y² = z² が円 X² + Y² = 1 に射影されることである。

**しかし、**(x/z)² + (y/z)² = 1 は「2次元」空間において変数の数を「3」に保っている：2つの真の変数 x と y、そして円の半径を制御するパラメータ z である。

### 例2（極座標による射影系列）

B³ = (r₃, α, β) の座標が B² = (r₃cosα, β) = (r₂, β) に射影され、B¹ では r₁ = r₃cosαcosβ = r₂cosβ となる。

射（f ∘ g = E₃₂E₂₁ = E₃₁）を持つ可換図式。

### 例3（準同型の基本定理）

線形代数の準同型に関する基本定理を利用する。V と W をベクトル空間、f を V から W への線形写像とすると、

V/Ker(f) ≅ Im(f)  ... (2.1)

f: R³ ∋ (x, y, z)ᵀ ↦ x + y + z ∈ R は準同型である。

x + y + z は R 中に3自由度または3変数を持つので、不変量は3である。

**注意.** (1,0,-1)ᵀ と (0,1,-1)ᵀ は (1,1,1)ᵀ に直交する。この直交性は本議論において重要である。

### 例4（微分による次元縮小）

次元縮小の文脈で、多項式方程式の微分を考えることができる。例えば、∂f(x,y)/∂x∂y = z_xy も3次元から1次元への写像と見なせる。逆として積分を考えるのは自然である。

### 注意

異なる次元空間間で任意の点を移す際、任意の点の元の変数の数は同じままである。

### 公理2

**元のn次元空間における任意の点の変数の数は、その点が他の任意の次元空間に移っても同じままである。これはトポロジーの不変量に相当する。**

## 3. 行列による記述

異なる次元間で点を射影するための特別な行列演算子を導入する。この行列は従来のものとは異なる。演算に一時変数を必要とする特別な演算子を含む。これは、この演算の前後で点の変数の数が異なるためである。

例えば、1次元空間の点 A₁ を2次元空間 (A₂) に写像する演算子 E₁₂ で操作する場合、方程式は A₂ = E₁₂A₁ であり、

∴ (x, y)ᵀ = diag(1, D)(a, T)ᵀ = (a, DT)ᵀ  ... (3.1)

ここで D は次元を上げる行列要素、T は演算後に2次元空間に対応するための一時変数。DT は y の全実数を一度に表す。

1次元から3次元：(x, y, z)ᵀ = diag(1, D, D)(a, T, T)ᵀ = (a, DT, DT)ᵀ  ... (3.2)

3次元から2次元：(x, y, z)ᵀ = diag(1, 1, D⁻¹)(a, b, c)ᵀ = (a, b, D⁻¹c)ᵀ  ... (3.3)

∴ E₃₂E₂₃ = (E₂₃)⁻¹E₂₃ = E₃₃ ≡ 1  ... (3.5)

### 一般的な次元単位行列 E_lm

l < m の場合：E_lm = diag(1,...,1, D,...,D)（1がl個、Dがm-l個）
m < l の場合：E_lm = (E_ml)⁻¹ = diag(1,...,1, D⁻¹,...,D⁻¹)（1がm個、D⁻¹がl-m個）

∴ E_jk E_kj = E_jj ≡ 1 ≡ E_kj E_jk = E_kk  ... (3.6)

E₀ₙ = diag(D,...,D)（n個）、Eₙ₀ = (E₀ₙ)⁻¹ = diag(D⁻¹,...,D⁻¹)（n個）

## 4. グルーポイド

変数の数が不変であることは、物理学では「対称性」、数学では「不変量」と見なすことができる。

**命題.** *前章の一連の部分関数的演算における行列は、以下のグルーポイド作用をなす：*

i. E_lm E_mn = E_ln（自己同型） ... (4.1)
ii. (E_kl E_lm) E_mn = E_kl (E_lm E_mn)（結合法則） ... (4.2)
iii. E_jk E_kj = E_jj ≡ 1 ≡ E_kj E_jk = E_kk（逆元） ... (4.3)
iv. E_kl E_lm E_ml = E_kl（右単位元） ... (4.5)
    E_lk E_kl E_lm = E_lm（左単位元） ... (4.6)
v. (E_lm E_mn)⁻¹ = (E_mn)⁻¹ (E_lm)⁻¹ ... (4.7)
vi. E_jj ≡ 1（スカラー値と等価な単位元） ... (4.8)

(i) と (vi) はグルーポイドに特有。

## 6. 有限単純グルーポイド

### 変換グルーポイドのモデリング

整数直線上の点の移動を ₘTₙ と表記。全元素 G = {ₘTₙ | m, n ∈ Z}。部分関数 ₗTₙ = ₗTₘ ₘTₙ。

### 定理6

I. ᵢTⱼ ⱼTₖ = ᵢTₖ ... (6.1)
II. (ᵢTⱼ ⱼTₖ)ₖTₗ = ᵢTⱼ(ⱼTₖ ₖTₗ)（結合法則） ... (6.2)
III. (ᵢTⱼ)⁻¹ = ⱼTᵢ ... (6.3)
IV. 右単位元・左単位元 ... (6.4), (6.5)
V. ᵢTⱼ ⱼTᵢ = ᵢTᵢ ≡ 1 ... (6.6)

### 部分グルーポイド

H = {₂ₘ₋₁T₂ₙ₋₁ | m, n ∈ Z}（奇数→奇数）
H' = {₂ₘT₂ₙ | m, n ∈ Z}（偶数→偶数）

**命題7.1.** HとH'はGの正規部分グルーポイドではない。
**命題7.2.** 極大固有部分グルーポイドも正規ではない。

### 参考情報

ₗTₘ が有限単純「群」であることが証明された。これは「コンウェイの悪夢」（有限単純群の分類における新たな群の存在可能性）に関連する。

## 7. 不変量と対称性 ―― ネーターの定理に向けて

- **不変量:** 数学的操作の下で変化しない物理系。対称性とも呼ばれる。
- **ネーターの定理:** 保存力を持つ物理系の作用の各微分対称性に対して、対応する保存則が存在する。離散対称性についても成立。

### グルーポイドにおける不変量

- **不変量は自由度の保存である。点がどこに射影されても、その自由度は保存される。**
- **高次元物理学がグルーポイドで記述されるならば、未知の物理的保存則を見出せるかもしれない。**

## 8. 集合論的アプローチ

### 非経験的な次元性の思考

次元性について経験的に考えるべきではない。各次元空間 Uₙ は他とは独立。相互作用も和集合もない。互いに独立（disjoint）である。

∴ U₀ ∩ U₁ ∩ U₃ ∩ ⋯ ∩ Uₙ = (U₀ ∪ U₁ ∪ U₃ ∪ ⋯ ∪ Uₙ)ᶜ = ∅

Uₙ の任意の点は写像前後ともに元のn自由度を保つ。自由度の不変性は不変量であり対称性でもある。

カントールやペアノ曲線の考えは、任意の点が持つ「自由度」を見落としているため間違いである。

## 結論

次元性の議論を通じて幾何学そのものを再構築する。3つの古典的幾何学を再構築：

1. **ユークリッド幾何学** — 任意の点は直交座標では直線や平面に、極座標では円や球面になりうる。『原論』の第一〜第三の公準と矛盾。
2. **座標幾何学（デカルト／フェルマー）** — 任意の点は元の自由度を保つ。デカルトの点は座標のラベルに過ぎず、ハエが持つ自由度を見落としていた。
3. **エルランゲン・プログラム（クライン）** — 異なる次元空間間の写像はグルーポイドで操作。エルランゲン・プログラムは同次元内の写像のみ（E_mm = 1 = I）。

古典的次元性と本理論の違いは天動説と地動説の違いに似ている。カルツァ＝クライン理論は3次元中心的であり、余剰次元のコンパクト化を要求する。

異なる次元空間のいずれも他の部分空間ではない。互いに独立である。

## 未解決問題

トポロジーの様々な概念を再定義する必要がある。引き戻しの再定義、S¹のR³やS³への写像の再解釈など。自由度をトポロジーの不変量として導入するだけなので、複雑な問題にはならないだろう。
